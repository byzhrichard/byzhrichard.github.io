📖概念

Agent: 一个能够感知环境、进行思考决策，并主动执行任务的智能体

Agent 通常指 AI Agent，也就是“不只是回答问题，而是能自己拆任务、调用工具、执行步骤、根据结果继续调整”的系统

普通大模型 vs Agent:

  • 普通大模型更像：你问一句 → 它答一句
  • Agent 更像：你给一个目标 → 它自己规划 → 调工具 → 执行 → 检查结果 → 继续修正

⚙️核心组成

看见情况 → 想办法 → 去做事 → 看结果 → 再调整:

目标 Goal
 ↓
感知 Observation
 ↓
思考 Reasoning / Planning
 ↓
行动 Action
 ↓
反馈 Feedback
 ↓
继续调整

1. 对话型 Agent

主要用于聊天、问答、客服

智能客服
学习助手
问答机器人

2. 工具型 Agent

可以调用工具完成任务

代码调试 Agent
数据分析 Agent
文档整理 Agent
邮件处理 Agent

3. 任务型 Agent

负责完成更复杂的连续任务

自动写报告
自动测试代码
自动规划旅行
自动做竞品分析

4. 多 Agent 系统

多个 Agent 分工合作 (虚拟团队)

一个 Agent 负责写代码
一个 Agent 负责测试
一个 Agent 负责审查
一个 Agent 负责写文档

⚙️工具调用

Agent 往往不是只靠大模型本身完成任务，而是会调用工具 skills

搜索网页 读取文件 运行 Python 操作数据库 调用 API 发送邮件 生成图片 修改代码

⚙代码Agent / 编程智能体

1. Claude Code

能在终端、IDE、Slack、网页中工作的 agentic coding tool，可以理解代码库、编辑文件、运行命令、协助调试和处理 Git 工作流

2. OpenCode

开源的 AI coding agent，官网介绍它可以在终端、IDE 或桌面端帮助你写代码。它强调开源、LSP 支持、多会话、分享会话，以及可以连接 GitHub Copilot、ChatGPT Plus/Pro 等账号

3. Codex

OpenAI 系的代码 Agent

4. Cursor

AI IDE + Agent 模式
