[DL从入门到入土] 拓展 - 正则化

📢个人导航

知乎：https://www.zhihu.com/people/byzh_rc

CSDN：https://blog.csdn.net/qq_54636039

注：本文仅对所述内容做了框架性引导，具体细节可查询其余相关资料or源码

参考文章：各方资料

@[TOC]

分类

标准化 Standardization: 把数据变成均值为 0、方差为 1 的分布

归一化 Normalization: 把数据缩放到某个范围，或者让数据满足某种统一尺度

正则化 Regularization: 防止模型过拟合，让模型泛化能力更好

参数级正则化:

  • L1 Regularization -> 在深度学习中使用较少，但理论经典 (Lasso)
  • L2 Regularization -> 常用 (Weight Decay)

随机失活类:

  • Dropout
  • DropConnect
  • Stochastic Depth (Resnet)

归一化类: (Normalization)

  • LRN
  • BatchNorm
  • LayerNorm
  • GroupNorm
  • InstanceNorm
  • Weight Normalization
  • Spectral Normalization

数据层正则化:

  • Data Augmentation (RandomCrop, Flip, Rotation, Noise)
  • Mixup
  • CutMix

训练策略类正则化:

  • Early Stopping
  • Label Smoothing

Dropout

核心思想: 训练时，随机让一部分神经元失效；测试时，使用完整网络

好处是: 模型不能过度依赖某几个神经元，被迫学习更加鲁棒、分散的特征

普通训练： 某几个神经元特别强，模型老依赖它们

Dropout： 每次随机拿掉一些神经元 模型：坏了，不能只靠大哥了，大家都得学点东西

优点:

  • 抑制过拟合
  • 能减少神经元之间的过度依赖

缺点:

  • 训练速度可能变慢
  • 很多时候不如数据增强、BN、权重衰减常用

LRN

Local Response Normalization: 局部响应归一化

在 AlexNet 里比较有名

核心思想: 对相邻通道之间的激活值做归一化，让强响应抑制附近的弱响应

有点像生物视觉里的“侧抑制”机制: “我很强，你们旁边几个通道都低调点”

但是现在 LRN 基本不常用了 BatchNorm 后来效果更好，也更稳定 所以 LRN 在现代网络里几乎被 BatchNorm 替代了

优点:

  • 增强局部通道之间的竞争

缺点:

  • 过时 (效果通常不如 BatchNorm)
  • 计算额外开销较大

BatchNorm

Batch Normalization: 批归一化

核心思想: 对一个 mini-batch 内的数据做归一化，让每一层的输入分布更加稳定

把特征变成类似：均值接近0, 方差接近1

计算当前 batch 的均值和方差：

$$ \mu_B = \frac{1}{m}\sum_{i=1}^{m}x_i $$

然后做归一化：

$$ \hat{x} = \frac{x-\mu_B}{\sqrt{\sigma_B^2+\epsilon}} $$

最后再做一个可学习的缩放和平移：

$$ y = \gamma \hat{x} + \beta $$

γ 和 $\beta$ 是可学习参数 因为如果强行把所有特征都变成均值 0、方差 1，可能会限制模型表达能力

-> BatchNorm先帮model标准化一下，但model想变回什么分布，也可以自己学

让特征分布更稳定:

Conv：我提取了一堆特征，但数值有点乱
BN：我帮你把每个通道的数值拉回正常范围!!!
ReLU：那我就不会一半死掉、一半爆炸了

优点:

  • 加速收敛, 训练更稳定, 允许使用更大学习率
  • 有一定正则化效果

缺点:

  • 小 batch 效果差
  • 训练和测试行为不同
  • 分布变化时 running mean / var 可能出问题

LayerNorm

Layer Normalization: 层归一化 LayerNorm是对单个样本内部归一化

Transformer 里基本都用 LayerNorm，而不是 BatchNorm 因为Transformer 经常处理变长序列、小 batch任务

Pre-LN:

x = x + attention(LayerNorm(x))
x = x + ffn(LayerNorm(x))

Post-LN:

x = LayerNorm(x + attention(x))
x = LayerNorm(x + ffn(x))

很多大模型更偏向 Pre-LN，因为 Pre-LN 训练深层网络更稳定

优点:

  • 不依赖 batch size
  • 训练和测试行为一致
  • 非常适合序列模型

缺点:

  • 对空间结构利用不如 BN / GN
  • 正则化效果较弱

GroupNorm

Group Normalization: 组归一化

目的: 解决 BatchNorm 对 batch size 依赖太强的问题

GroupNorm把通道分成若干组，在每一组内部做归一化

假设输入张量C=64, GN的num_groups=4

则每组有64/4=16个通道, GN会对组内的特征归一化

优点:

  • 不依赖 batch size
  • 训练和测试行为一致
  • 适合检测、分割、Diffusion、U-Net

缺点:

  • 需要手动选择 group 数
  • 大 batch CNN 中不一定优于 BatchNorm
  • 分组不合适会影响效果
