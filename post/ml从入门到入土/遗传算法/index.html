🌱背景

GA: Genetic Algorithm: 遗传算法

在自然界中，生物通过：

  • 选择
  • 交叉
  • 变异
  • 适者生存

不断进化，最终让更适应环境的个体保留下来

遗传算法借鉴了这个思想：

  • 把问题的一个解看成一个“个体”，
  • 把很多个解组成一个“种群”，
  • 让这些解不断进化，逐渐找到更优解

1. 为什么需要遗传算法

在机器学习、运筹优化、路径规划、参数搜索等问题中，经常会遇到这类情况：

  • 目标函数很复杂
  • 变量很多
  • 约束条件很多
  • 很难直接求导
  • 很难用传统数学方法求最优解
  • 搜索空间巨大，暴力枚举不现实

2. 核心思想

遗传算法不是从一个解开始优化，而是从一群解开始搜索

这一群解叫做：种群 Population 种群里的每一个解叫做：个体 Individual

每个个体有自己的编码，类似生物的基因

遗传算法会评价每个个体的好坏：适应度函数 Fitness Function -> 适应度越高，说明这个解越好 -> 让更优秀的个体更容易被保留下来，并通过交叉和变异生成新的个体

⚙️架构(公式) 

  graph TD
A[初始化种群] --> B[计算适应度]
B --> C{是否满足终止条件}
C -->|是| D[输出最优解]
C -->|否| E[选择操作]
E --> F[交叉操作]
F --> G[变异操作]
G --> H[生成新种群]
H --> B
概念 含义
个体 Individual 一个候选解
种群 Population 多个候选解组成的集合
染色体 Chromosome 个体的编码形式
基因 Gene 染色体中的一个位置
适应度 Fitness 衡量解的好坏
选择 Selection 挑选优秀个体
交叉 Crossover 两个个体交换部分基因
变异 Mutation 随机改变部分基因
迭代代数 Generation 进化轮数

1. 个体编码

编码: 把一个实际问题的解表示成算法可以处理的形式

  1. 二进制编码:
    • 1 表示选择
    • 0 表示不选择
    • e.g. [1, 0, 1, 1, 0]
  2. 整数编码:
    • 排列、路径、顺序类问题
    • e.g. [3, 1, 4, 2, 5]
  3. 实数编码:
    • 连续参数优化问题
    • e.g. [0.23, -1.52, 3.77]

2. 适应度函数

适应度函数用于评价一个个体的好坏

假设我们要求最大化目标函数：

$$ \max f(x) $$

那么可以直接令适应度为：

$$ Fitness(x) = f(x) $$

3. 选择 Selection

选择操作的作用是：让优秀个体更容易被保留下来

  1. 轮盘赌选择: 适应度越高，被选中的概率越大
  2. 锦标赛选择: 每次随机挑 n 个个体，谁最好谁留下 (重复这个过程，直到选够)
  3. 精英保留策略: 每一代中最好的几个个体，直接复制到下一代

4. 交叉 Crossover

两个父代个体交换部分基因，生成新的子代个体

4.1 交叉 - 二进制

  1. 单点交叉:

    父代1: 111 | 000
父代2: 000 | 111
-> 
子代1: 111111
子代2: 000000

  2. 两点交叉: 交换中间片段

    父代1: 11 | 000 | 11
父代2: 00 | 111 | 00
->
子代1: 11 | 111 | 11
子代2: 00 | 000 | 00

  3. 均匀交叉: 每个基因位置都随机决定

    父代1: [1, 1, 1, 1]
父代2: [0, 0, 0, 0]
->
子代:  [1, 0, 1, 0]

4.1 交叉 - 实数

  1. 算术交叉: 子代是父代的线性组合

    父代1: x₁
父代2: x₂
α ∈ [0, 1] (随机权重)
->
子代1: α·x₁ + (1-α)·x₂
子代2: α·x₂ + (1-α)·x₁

5. 变异 Mutation

如果没有变异，种群可能很快变得非常相似，陷入局部最优

5.1 变异 - 二进制

  1. 随机把某个位置翻转

    变异前: 10110
变异后: 10010

5.2 变异 - 实数

  1. 对某个实数基因加入随机扰动 $x_i' = x_i + \Delta$

    变异前: [1.2, 3.5, 2.7]
变异后: [1.2, 3.8, 2.7]

6. 终止条件

常见终止条件包括：

  • 达到最大迭代次数
  • 最优适应度达到要求
  • 连续多代没有明显提升
  • 运行时间达到上限

👍优点/创新点

1. 不依赖梯度

不需要目标函数可导

遗传算法只需要知道：这个解好不好？

-> 只需要适应度函数

2. 全局搜索能力较强

遗传算法不是只维护一个解，而是维护一群解

多个个体可以同时在搜索空间中探索不同区域

不能保证一定找到全局最优，但通常能找到不错的近似解

3. 适合组合优化问题

很多组合优化问题搜索空间极大。

比如一个简单的排列问题，如果有 $n$ 个元素，可能的排列数是 $n!$

-> 当 $n$ 很大时，暴力搜索基本不可行

遗传算法可以通过选择、交叉、变异在巨大搜索空间中进行启发式搜索

4. 算法框架通用

只要能定义：

  • 个体如何编码
  • 如何评价个体
  • 如何选择
  • 如何交叉
  • 如何变异

就可以用遗传算法求解很多问题

👎缺点

1. 参数较多

参数会明显影响算法效果:

  • 种群大小
  • 迭代次数
  • 交叉概率
  • 变异概率
  • 精英个体数量
  • 选择方式

比如：

  • 种群太小，容易早熟
  • 种群太大，计算成本高
  • 变异率太低，容易陷入局部最优
  • 变异率太高，搜索又太随机

2. 计算成本较高

每一代都需要计算整个种群的适应度

如果适应度函数本身很复杂，比如需要跑仿真、训练模型、计算路径代价，那么遗传算法的计算量会比较大

3. 容易早熟收敛

早熟收敛: 如果种群多样性下降太快，所有个体很快变得相似，算法可能陷入局部最优

解决方法包括：

  • 增大种群规模
  • 提高变异概率
  • 使用精英保留但不要保留太多
  • 引入随机新个体
  • 使用多种群遗传算法

💻模型代码

目标函数为：

$$ f(x) = x \sin(10\pi x) + 2 $$

搜索范围：

$$ x \in [-1, 2] $$

求解函数最大值问题

########################################
# 最大化目标函数:
# - 目标函数: f(x)=xsin(10πx)+2
# - 搜索访问: x∈[−1,2]
########################################

import numpy as np
import matplotlib.pyplot as plt


class GeneticAlgorithm:
    def __init__(
        self,
        objective_func,
        bounds,
        pop_size=50,
        max_generations=100,
        crossover_rate=0.8,
        mutation_rate=0.1,
        elite_size=2,
        mutation_scale=0.1,
        seed=None
    ):
        """
        objective_func: 目标函数，输入一个个体，输出适应度
        bounds: 每个变量的取值范围，例如 [(-1, 2)]
        pop_size: 种群规模
        max_generations: 最大迭代次数
        crossover_rate: 交叉概率
        mutation_rate: 变异概率
        elite_size: 精英保留数量
        mutation_scale: 变异扰动强度
        seed: 随机种子
        """
        self.objective_func = objective_func
        self.bounds = np.array(bounds)
        self.pop_size = pop_size
        self.max_generations = max_generations
        self.crossover_rate = crossover_rate
        self.mutation_rate = mutation_rate
        self.elite_size = elite_size
        self.mutation_scale = mutation_scale

        self.dim = len(bounds) # 优化变量的个数(问题的维度)
        self.rng = np.random.default_rng(seed) # 随机数生成器对象

        self.population = self._init_population() # 存储当前种群
        self.best_history = [] # 记录每一代最优个体适应度值

    def _init_population(self):
        """
        初始化种群
        """
        lower = self.bounds[:, 0]
        upper = self.bounds[:, 1]

        population = self.rng.uniform( # 均匀分布
            low=lower,
            high=upper,
            size=(self.pop_size, self.dim)
        )

        return population

    def _evaluate(self):
        """
        计算种群中每个个体的适应度(fitness)
        """
        fitness = np.array([
            self.objective_func(individual)
            for individual in self.population
        ])

        return fitness

    def _tournament_selection(self, fitness, tournament_size=3):
        """
        锦标赛选择 (每次随机挑 n 个个体，谁最好谁留下)
        """
        selected = []

        for _ in range(self.pop_size):
            candidates = self.rng.choice(
                self.pop_size,
                size=tournament_size,
                replace=False # 不重复抽取(同一轮锦标赛中不会选到同一个体两次)
            )

            best_candidate = candidates[np.argmax(fitness[candidates])]
            selected.append(self.population[best_candidate])

        return np.array(selected)

    def _crossover(self, parent1, parent2):
        """
        实数编码交叉：算术交叉
        """
        # (1 - self.crossover_rate) % 的概率 不交叉
        if self.rng.random() > self.crossover_rate:
            return parent1.copy(), parent2.copy()

        # self.crossover_rate % 的概率 交叉
        alpha = self.rng.random(self.dim)

        child1 = alpha * parent1 + (1 - alpha) * parent2
        child2 = alpha * parent2 + (1 - alpha) * parent1

        return child1, child2

    def _mutate(self, individual):
        """
        实数变异：对部分基因加入高斯噪声
        """
        for i in range(self.dim):
            if self.rng.random() < self.mutation_rate:
                individual[i] += self.rng.normal(0, self.mutation_scale)

        # 防止越界
        individual = np.clip(
            individual,
            self.bounds[:, 0],
            self.bounds[:, 1]
        )

        return individual

    def run(self):
        """
        执行遗传算法
        """
        for generation in range(self.max_generations):
            fitness = self._evaluate() # 计算self.population的适应度

            # 记录当前最优
            best_idx = np.argmax(fitness)
            best_individual = self.population[best_idx].copy() # 最优个体
            best_fitness = fitness[best_idx]
            self.best_history.append(best_fitness)

            print(
                f"Generation {generation + 1:03d} | "
                f"Best Fitness = {best_fitness:.6f} | "
                f"Best Individual = {best_individual}"
            )

            # 精英保留
            elite_indices = np.argsort(fitness)[-self.elite_size:]
            elites = self.population[elite_indices].copy() # 精英

            # 选择 (锦标赛)
            selected_population = self._tournament_selection(fitness)

            # 交叉 + 变异
            new_population = []

            for i in range(0, self.pop_size - self.elite_size, 2):
                parent1 = selected_population[i]
                parent2 = selected_population[(i + 1) % self.pop_size]

                # 交叉
                child1, child2 = self._crossover(parent1, parent2)

                # 变异
                child1 = self._mutate(child1)
                child2 = self._mutate(child2)

                new_population.append(child1)

                if len(new_population) < self.pop_size - self.elite_size:
                    new_population.append(child2)

            # 合并新种群和精英个体
            self.population = np.vstack([
                np.array(new_population),
                elites
            ])

        # 最终结果
        fitness = self._evaluate()
        best_idx = np.argmax(fitness)

        return self.population[best_idx], fitness[best_idx]


def objective(x):
    """
    目标函数:
    f(x) = x * sin(10πx) + 2
    """
    return x[0] * np.sin(10 * np.pi * x[0]) + 2


if __name__ == "__main__":
    ga = GeneticAlgorithm(
        objective_func=objective,
        bounds=[(-1, 2)],
        pop_size=50,
        max_generations=100,
        crossover_rate=0.8,
        mutation_rate=0.1,
        elite_size=2,
        mutation_scale=0.1,
        seed=42
    )

    best_x, best_y = ga.run()

    print("\nFinal Result:")
    print("Best x:", best_x)
    print("Best y:", best_y)

    # 绘制收敛曲线
    plt.figure(figsize=(8, 5))
    plt.plot(ga.best_history)
    plt.xlabel("Generation")
    plt.ylabel("Best Fitness")
    plt.title("Genetic Algorithm Convergence")
    plt.grid(True)
    plt.show()

💻项目实例 

########################################
# 0-1背包问题:
# - 给定若干物品，每个物品有重量和价值
# - 在背包容量限制下，选择一部分物品
# - 目标: 最大化所选物品的总价值
########################################

import numpy as np


class KnapsackGA:
    def __init__(
        self,
        weights,
        values,
        capacity,
        pop_size=100,
        max_generations=200,
        crossover_rate=0.8,
        mutation_rate=0.02,
        elite_size=2,
        seed=None
    ):
        """
        weights: 物品重量列表
        values: 物品价值列表
        capacity: 背包最大容量
        pop_size: 种群规模
        max_generations: 最大迭代次数
        crossover_rate: 交叉概率
        mutation_rate: 变异概率
        elite_size: 精英保留数量
        seed: 随机种子
        """
        self.weights = np.array(weights) # 物品重量
        self.values = np.array(values) # 物品价值
        self.capacity = capacity # 背包容量

        self.pop_size = pop_size
        self.max_generations = max_generations
        self.crossover_rate = crossover_rate
        self.mutation_rate = mutation_rate
        self.elite_size = elite_size

        self.num_items = len(weights) # 物品数量
        self.rng = np.random.default_rng(seed) # 随机数生成器对象

        self.population = self._init_population() # 存储当前种群
        self.best_history = [] # 记录每一代最优适应度值

    def _init_population(self):
        """
        初始化种群

        二进制编码:
        1 表示选择该物品
        0 表示不选择该物品
        """
        return self.rng.integers(
            low=0,
            high=2,
            size=(self.pop_size, self.num_items)
        )

    def _fitness(self, individual):
        """
        计算单个个体的适应度

        对于背包问题:
        - 如果总重量超过背包容量，则适应度为 0
        - 如果没有超过容量，则适应度为总价值
        """
        total_weight = np.sum(individual * self.weights) # 当前方案总重量
        total_value = np.sum(individual * self.values) # 当前方案总价值

        # 超出背包容量，说明该方案不可行
        if total_weight > self.capacity:
            return 0

        return total_value

    def _evaluate(self):
        """
        计算种群中每个个体的适应度(fitness)
        """
        fitness = np.array([
            self._fitness(individual)
            for individual in self.population
        ])

        return fitness

    def _tournament_selection(self, fitness, tournament_size=3):
        """
        锦标赛选择

        每次随机挑选 tournament_size 个个体
        从中选择适应度最高的个体进入下一代候选种群
        """
        selected = []

        for _ in range(self.pop_size):
            candidates = self.rng.choice(
                self.pop_size,
                size=tournament_size,
                replace=False # 不重复抽取
            )

            best_candidate = candidates[np.argmax(fitness[candidates])]
            selected.append(self.population[best_candidate])

        return np.array(selected)

    def _crossover(self, parent1, parent2):
        """
        单点交叉

        随机选择一个交叉点
        将两个父代在该位置之后的基因片段进行交换
        """
        # (1 - self.crossover_rate) % 的概率 不交叉
        if self.rng.random() > self.crossover_rate:
            return parent1.copy(), parent2.copy()

        # self.crossover_rate % 的概率 交叉
        point = self.rng.integers(1, self.num_items)

        child1 = np.concatenate([
            parent1[:point],
            parent2[point:]
        ])

        child2 = np.concatenate([
            parent2[:point],
            parent1[point:]
        ])

        return child1, child2

    def _mutate(self, individual):
        """
        二进制变异

        对每一个基因位:
        - 0 -> 1
        - 1 -> 0
        """
        for i in range(self.num_items):
            if self.rng.random() < self.mutation_rate:
                individual[i] = 1 - individual[i]

        return individual

    def run(self):
        """
        执行遗传算法
        """
        for generation in range(self.max_generations):
            fitness = self._evaluate() # 计算 self.population 的适应度

            # 记录当前最优
            best_idx = np.argmax(fitness)
            best_fitness = fitness[best_idx]
            best_individual = self.population[best_idx].copy() # 最优个体

            self.best_history.append(best_fitness)

            print(
                f"Generation {generation + 1:03d} | "
                f"Best Value = {best_fitness} | "
                f"Best Individual = {best_individual}"
            )

            # 精英保留
            elite_indices = np.argsort(fitness)[-self.elite_size:]
            elites = self.population[elite_indices].copy() # 精英个体

            # 选择 (锦标赛)
            selected_population = self._tournament_selection(fitness)

            # 交叉 + 变异
            new_population = []

            for i in range(0, self.pop_size - self.elite_size, 2):
                parent1 = selected_population[i]
                parent2 = selected_population[(i + 1) % self.pop_size]

                # 交叉
                child1, child2 = self._crossover(parent1, parent2)

                # 变异
                child1 = self._mutate(child1)
                child2 = self._mutate(child2)

                new_population.append(child1)

                if len(new_population) < self.pop_size - self.elite_size:
                    new_population.append(child2)

            # 合并新种群和精英个体
            self.population = np.vstack([
                np.array(new_population),
                elites
            ])

        # 最终结果
        fitness = self._evaluate()
        best_idx = np.argmax(fitness)

        return self.population[best_idx], fitness[best_idx]


if __name__ == "__main__":
    weights = [2, 3, 4, 5, 9, 7, 6, 1, 8, 4]
    values = [3, 4, 8, 8, 10, 6, 7, 2, 9, 5]
    capacity = 20

    ga = KnapsackGA(
        weights=weights,
        values=values,
        capacity=capacity,
        pop_size=80,
        max_generations=100,
        crossover_rate=0.8,
        mutation_rate=0.03,
        elite_size=2,
        seed=42
    )

    best_solution, best_value = ga.run()

    total_weight = np.sum(best_solution * np.array(weights)) # 最优方案总重量
    selected_items = np.where(best_solution == 1)[0] # 被选中的物品编号

    print("\nFinal Result:")
    print("Best Solution:", best_solution)
    print("Selected Items:", selected_items)
    print("Total Weight:", total_weight)
    print("Total Value:", best_value)
