参考资料

Box & Jenkins：时间序列分析经典框架

背景

很多序列除了“惯性”（AR）之外，还有另一种朴素现象：系统会被随机冲击不断扰动，而我们观测到的值像是“过去冲击的叠加回响”

设备噪声、市场短期扰动、传感器瞬态抖动、某些不可解释的短时误差

MA 模型用最简单的线性方式表达这种“冲击记忆”：

  • 当前值 = 常数 + 当前及过去 q 步噪声的线性组合
  • 常作为 AR 的互补基线，并与 AR 组合成 ARMA/ARIMA

架构(公式)

1) MA(q) 定义

设序列为 $\{x_t\}$，MA(q)：

$$ x_t = \mu + \varepsilon_t + \sum_{i=1}^{q}\theta_i \color{purple}{\varepsilon_{t-i}} $$
  • $\mu$：常数项（可对应均值）
  • $\theta_i$：滑动平均系数（冲击/噪声的“回响权重”）
  • $q$：阶数（用多少个滞后噪声）
  • $\varepsilon_t$：白噪声（通常假设 $\varepsilon_t \sim \mathcal{N}(0,\sigma^2)$）

直观：当前 = 均值 + 当前噪声 + 过去 q 个噪声的加权和

2) 关键区别：MA 不是“用过去的 x 当特征”

AR：$x_t$ 依赖过去的 x（可直接观测） MA：$x_t$ 依赖过去的 噪声 $\varepsilon$（不可直接观测）

所以 MA 拟合更像“在估计隐变量噪声 + 参数”，通常用 MLE 或专门算法（而不是简单 OLS）

3) 可逆性（Invertibility，非常重要）

MA(q) 常要求可逆，否则参数不唯一、估计不稳定

  • MA(1)：$x_t = \mu + \varepsilon_t + \theta \varepsilon_{t-1}$ 可逆条件：$|\theta| < 1$
  • MA(q)：更一般是特征多项式的根在单位圆外

记忆法：AR 讲“平稳”，MA 讲“可逆”，本质都在防止模型“爆/不唯一”

4) 阶数 q 怎么选（常用经验）

  • 看 ACF：MA(q) 的 ACF 往往在 $q$ 阶“截尾”
  • 或用信息准则：AIC/BIC（越小越好）

5) MA 与“滑动窗口均值”不是一回事

很多人把 “moving average 平滑(rolling mean)” 当成 MA 模型：

  • rolling mean：对 观测值 x 做窗口平均（数据处理/平滑技巧）
  • MA(q) 模型：对 噪声 ε 做线性组合（统计生成模型）

rolling mean 可以看成一个滤波器，但不等价于 MA(q) 的概率模型

创新点

  1. 极简但强的假设：当前值由近期随机冲击叠加而成
  2. 更擅长刻画：短期误差/扰动的相关结构（“噪声带记忆”）
  3. 与 AR 互补：==AR 记忆在 x，MA 记忆在 ε== → 组合成 ARMA 很自然
  4. 可解释：$\theta_i$ 表示“第 i 个冲击对现在的影响强度”

代码实现 - numpy

说明：MA(q) 的严格拟合通常用 MLE（statsmodels 更合适）

numpy 部分这里重点给：

  1. MA(q) 序列模拟（可控冲击回响）
  2. rolling mean 平滑（对比“不是一个东西”）
import numpy as np
import matplotlib.pyplot as plt

plt.rcParams["font.sans-serif"] = ["SimHei"]
plt.rcParams["axes.unicode_minus"] = False


# =========================
# 1) MA(q) 模拟
# =========================
def simulate_maq(n: int = 400, theta=(0.6, -0.2), mu=0.0, noise_std=1.0, burn_in=200, seed=0):
    rng = np.random.default_rng(seed)
    theta = np.asarray(theta, dtype=float)
    q = len(theta)

    eps = rng.normal(0.0, noise_std, size=n + burn_in + q)
    x = np.zeros(n + burn_in + q, dtype=float)

    for t in range(q, n + burn_in + q):
        past_eps = eps[t-q:t][::-1]          # 长度一定是 q：eps_{t-1}...eps_{t-q}
        x[t] = mu + eps[t] + float(theta @ past_eps)

    return x[burn_in + q:]



# =========================
# 2) rolling mean 平滑 (滑动窗口均值)（不是 MA 模型）
# =========================
def rolling_mean(x: np.ndarray, window: int = 10):
    """
    对观测值做窗口均值平滑（滤波/预处理），与 MA(q) 模型不同
    """
    x = np.asarray(x, dtype=float)
    if window <= 0:
        raise ValueError("window 必须 > 0")
    if len(x) < window:
        raise ValueError("序列长度必须 >= window")

    w = np.ones(window, dtype=float) / window
    y = np.convolve(x, w, mode="valid")
    # 为了对齐长度，前面 pad NaN
    pad = np.full(window - 1, np.nan, dtype=float)
    return np.concatenate([pad, y])


# =========================
# 3) Demo：MA序列 + 平滑对比
# =========================
def demo():
    x = simulate_maq(
        n=500,
        theta=(0.75, -0.35),   # “冲击回响”
        mu=0.2,
        noise_std=0.8,
        burn_in=300,
        seed=42
    )

    rm = rolling_mean(x, window=15)

    t = np.arange(len(x))
    plt.figure(figsize=(10, 5))
    plt.plot(t, x, label="MA(q) 生成序列")
    plt.plot(t, rm, label="rolling mean 平滑(窗口=15)")
    plt.title("MA(q) 模拟 vs rolling mean 平滑（注意：rolling mean 不是 MA 模型）")
    plt.xlabel("时间 t")
    plt.ylabel("$x_t$")
    plt.legend()
    plt.tight_layout()
    plt.show()


if __name__ == "__main__":
    demo()

import numpy as np
import matplotlib.pyplot as plt

plt.rcParams["font.sans-serif"] = ["SimHei"]
plt.rcParams["axes.unicode_minus"] = False

from statsmodels.tsa.arima.model import ARIMA


# =========================
# 1) MA(q) 模拟（同上）
# =========================
def simulate_maq(n: int = 400, theta=(0.6, -0.2), mu=0.0, noise_std=1.0, burn_in=200, seed=0):
    rng = np.random.default_rng(seed)
    theta = np.asarray(theta, dtype=float)
    q = len(theta)

    eps = rng.normal(0.0, noise_std, size=n + burn_in + q)
    x = np.zeros(n + burn_in + q, dtype=float)

    for t in range(q, n + burn_in + q):
        ma_part = 0.0
        for i in range(1, q + 1):
            ma_part += theta[i - 1] * eps[t - i]   # eps[t-1], eps[t-2], ...
        x[t] = mu + eps[t] + ma_part

    return x[burn_in + q:]



# =========================
# 2) 用 BIC 选 MA 阶数 + 拟合（ARIMA的 order=(p,d,q)，这里 p=0,d=0）
# =========================
def fit_ma_by_bic(train: np.ndarray, q_max: int = 20):
    """
    返回：
      best_res: ARIMAResults
      best_q  : 最优阶数
      bic_list: 各阶 BIC
    """
    train = np.asarray(train, dtype=float)

    best_res = None
    best_q = None
    bic_list = []

    for q in range(0, q_max + 1):
        # MA(0) = 白噪声 around mean
        res = ARIMA(train, order=(0, 0, q), trend="c").fit()
        bic_list.append(res.bic)
        if best_res is None or res.bic < best_res.bic:
            best_res = res
            best_q = q

    return best_res, best_q, np.array(bic_list, dtype=float)


# =========================
# 3) 预测：one-step rolling（可选严格版）与 multi-step
# =========================
def one_step_rolling_forecast(train: np.ndarray, test: np.ndarray, q: int):
    """
    单步滚动：每一步用真实值更新历史，并重新拟合（严格但慢）
    """
    train = np.asarray(train, float)
    test = np.asarray(test, float)

    history = list(train)
    preds = []

    for xt in test:
        tmp_res = ARIMA(np.array(history), order=(0, 0, q), trend="c").fit()
        yhat = float(tmp_res.forecast(steps=1)[0])
        preds.append(yhat)
        history.append(float(xt))

    return np.array(preds, dtype=float)


def multi_step_forecast(res, steps: int):
    """
    多步预测：一次拟合，直接 forecast steps 步
    """
    pred = res.forecast(steps=steps)
    return np.asarray(pred, dtype=float)


# =========================
# 4) Demo：拟合 + 子图可视化
# =========================
def demo():
    x = simulate_maq(
        n=500,
        theta=(0.75, -0.35),
        mu=0.2,
        noise_std=0.8,
        burn_in=300,
        seed=42
    )

    n_train = 400
    train = x[:n_train]
    test = x[n_train:]

    res, q, bic_list = fit_ma_by_bic(train, q_max=20)
    print(f"[BIC选阶] best q={q}, BIC={res.bic:.2f}")
    print("[参数] ", res.params)  # 包含截距与MA系数等

    pred_roll = one_step_rolling_forecast(train, test, q=q)
    pred_multi = multi_step_forecast(res, steps=len(test))

    mse_roll = float(np.mean((pred_roll - test) ** 2))
    mse_multi = float(np.mean((pred_multi - test) ** 2))
    print(f"[Test] one-step rolling MSE={mse_roll:.4f} | multi-step forecast MSE={mse_multi:.4f}")

    t_train = np.arange(len(train))
    t_test = np.arange(len(train), len(train) + len(test))

    fig, axes = plt.subplots(2, 1, figsize=(10, 8), sharex=True)

    ax = axes[0]
    ax.plot(t_train, train, label="train")
    ax.plot(t_test, test, label="test (true)")
    ax.plot(t_test, pred_roll, label="MA单步滚动预测(每步重拟合)")
    ax.axvline(len(train) - 1)
    ax.set_title(f"MA基线(选择q={q}) - 单步滚动预测 | MSE={mse_roll:.3f}")
    ax.set_ylabel("$x_t$")
    ax.legend()

    ax = axes[1]
    ax.plot(t_train, train, label="train")
    ax.plot(t_test, test, label="test (true)")
    ax.plot(t_test, pred_multi, label="MA多步预测(一次拟合)")
    ax.axvline(len(train) - 1)
    ax.set_title(f"MA基线(选择q={q}) - 多步预测 | MSE={mse_multi:.3f}")
    ax.set_xlabel("时间 t")
    ax.set_ylabel("$x_t$")
    ax.legend()

    plt.tight_layout()
    plt.show()


if __name__ == "__main__":
    demo()
