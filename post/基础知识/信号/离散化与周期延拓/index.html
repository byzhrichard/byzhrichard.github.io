ж—¶еҹҹе‘Ёжңҹе»¶жӢ“пјҢйў‘еҹҹзҰ»ж•ЈеҢ–

еҒҮи®ҫеҺҹжқҘжңүдёҖдёӘиҝһз»ӯдҝЎеҸ·пјҡ

$$ x(t) $$

еҰӮжһңжҲ‘д»¬жҠҠе®ғеңЁж—¶еҹҹдёӯеҸҚеӨҚеӨҚеҲ¶пјҢеҸҳжҲҗе‘ЁжңҹдҝЎеҸ·пјҡ

$$ x_p(t) = x(t) \text{ е‘ЁжңҹйҮҚеӨҚ} $$

йӮЈд№Ҳе®ғзҡ„йў‘и°ұе°ұдёҚжҳҜиҝһз»ӯзҡ„дёҖж•ҙзүҮдәҶпјҢиҖҢжҳҜеҸҳжҲҗдёҖж №дёҖж №зҡ„и°ұзәҝ

жҜ”еҰӮеӮ…йҮҢеҸ¶зә§ж•°:

е‘ЁжңҹдҝЎеҸ·еҸҜд»ҘеҶҷжҲҗпјҡ

$$ > x_p(t)=\sum_{k=-\infty}^{\infty} c_k e^{j k \omega_0 t} > $$

е®ғзҡ„йў‘зҺҮеҸӘеҮәзҺ°еңЁпјҡ

$$ > \omega = k\omega_0 > $$

ж—¶еҹҹе‘ЁжңҹжҳҜ $T_0$ зҡ„иҜқ, йӮЈд№Ҳйў‘еҹҹй—ҙйҡ”жҳҜ $1/T_0$

ж—¶еҹҹзҰ»ж•ЈеҢ–пјҢйў‘еҹҹе‘ЁжңҹеҢ–

еҺҹжқҘжңүдёҖдёӘиҝһз»ӯж—¶й—ҙдҝЎеҸ·пјҡ

$$ x(t) $$

еҰӮжһңжҲ‘д»¬еҜ№е®ғйҮҮж ·пјҡ(ж—¶еҹҹд»Һиҝһз»ӯеҸҳжҲҗдәҶзҰ»ж•ЈзӮ№)

$$ x[n] = x(nT_s) $$

йӮЈд№Ҳйў‘и°ұдјҡе‘ЁжңҹйҮҚеӨҚ -> иҝһз»ӯдҝЎеҸ·зҡ„йў‘и°ұ $X(f)$ дјҡеңЁйў‘еҹҹдёӯжҢүз…§йҮҮж ·йў‘зҺҮ $f_s$ дёҖд»ҪдёҖд»ҪеӨҚеҲ¶

иҝҷд№ҹжҳҜдёәд»Җд№ҲйҮҮж ·еҗҺдјҡеҮәзҺ°ж··еҸ 

еҰӮжһңйҮҮж ·йў‘зҺҮеӨӘдҪҺпјҢйў‘еҹҹдёӯйҮҚеӨҚзҡ„йў‘и°ұдјҡдә’зӣёйҮҚеҸ 

жүҖд»ҘжүҚйңҖиҰҒеҘҲеҘҺж–Ҝзү№йҮҮж ·е®ҡзҗҶпјҡ$f_s > 2 f_{\max}$

ж—¶еҹҹйҮҮж ·й—ҙйҡ”жҳҜ $T_0$ зҡ„иҜқ, йў‘еҹҹе‘ЁжңҹжҳҜ $1/T_0$

д»Јз Ғ 

import numpy as np
import matplotlib.pyplot as plt

# дёӯж–Ү
plt.rcParams['font.sans-serif'] = ['SimHei']  # и®ҫзҪ®дёӯж–Үеӯ—дҪ“
plt.rcParams['axes.unicode_minus'] = False    # и§ЈеҶіиҙҹеҸ·жҳҫзӨәй—®йўҳ

# =========================
# еҸӮж•°и®ҫзҪ®
# =========================
T = 1.0          # еҚ•дёӘзҹ©еҪўи„үеҶІе®ҪеәҰ
T0 = 3.0         # е‘Ёжңҹе»¶жӢ“еҗҺзҡ„е‘Ёжңҹ
t_min, t_max = -8, 8
f_min, f_max = -5, 5

# =========================
# еҺҹе§Ӣж—¶еҹҹдҝЎеҸ·пјҡеҚ•дёӘзҹ©еҪўи„үеҶІ
# x(t) = 1, |t| <= T/2
# =========================
t = np.linspace(t_min, t_max, 5000)
x = np.where(np.abs(t) <= T / 2, 1.0, 0.0)

# =========================
# еҺҹе§Ӣйў‘еҹҹпјҡCTFT
# X(f) = T * sinc(fT)
# np.sinc(x) = sin(pi x) / (pi x)
# =========================
f = np.linspace(f_min, f_max, 5000)
X = T * np.sinc(f * T)

# =========================
# е‘Ёжңҹе»¶жӢ“еҗҺзҡ„ж—¶еҹҹдҝЎеҸ·
# жҠҠзҹ©еҪўи„үеҶІжҜҸйҡ” T0 еӨҚеҲ¶дёҖж¬Ў
# =========================
xp = np.zeros_like(t)
for k in range(-10, 11):
    xp += np.where(np.abs(t - k * T0) <= T / 2, 1.0, 0.0)

# =========================
# е‘ЁжңҹдҝЎеҸ·зҡ„йў‘еҹҹпјҡеӮ…йҮҢеҸ¶зә§ж•°и°ұзәҝ
# йў‘зҺҮеҸӘеҮәзҺ°еңЁ f = k / T0
# зі»ж•°еӨ§е°ҸдёҺеҺҹйў‘и°ұеңЁиҝҷдәӣзӮ№зҡ„йҮҮж ·жңүе…і
# c_k = 1 / T0 * X(k / T0)
# =========================
k = np.arange(-20, 21)
fk = k / T0
ck = (1 / T0) * T * np.sinc(fk * T)

# =========================
# з”»еӣҫ
# =========================
fig, axes = plt.subplots(2, 2, figsize=(12, 7))

# дёҠе·ҰпјҡеҺҹе§Ӣж—¶еҹҹ
axes[0, 0].plot(t, x)
axes[0, 0].set_title("еҺҹжқҘпјҡж—¶еҹҹ - еҚ•дёӘзҹ©еҪўи„үеҶІ")
axes[0, 0].set_xlabel("t")
axes[0, 0].set_ylabel("x(t)")
axes[0, 0].grid(True)

# дёҠеҸіпјҡеҺҹе§Ӣйў‘еҹҹ
axes[0, 1].plot(f, X)
axes[0, 1].set_title("еҺҹжқҘпјҡйў‘еҹҹ - иҝһз»ӯ sinc йў‘и°ұ")
axes[0, 1].set_xlabel("f")
axes[0, 1].set_ylabel("X(f)")
axes[0, 1].grid(True)

# дёӢе·Ұпјҡе‘Ёжңҹе»¶жӢ“еҗҺзҡ„ж—¶еҹҹ
axes[1, 0].plot(t, xp)
axes[1, 0].set_title("зҺ°еңЁпјҡж—¶еҹҹ - е‘Ёжңҹе»¶жӢ“")
axes[1, 0].set_xlabel("t")
axes[1, 0].set_ylabel("x_p(t)")
axes[1, 0].grid(True)

# дёӢеҸіпјҡе‘Ёжңҹе»¶жӢ“еҗҺзҡ„йў‘еҹҹ
axes[1, 1].stem(fk, ck)
axes[1, 1].set_title("зҺ°еңЁпјҡйў‘еҹҹ - зҰ»ж•ЈеҢ–")
axes[1, 1].set_xlabel("f")
axes[1, 1].set_ylabel("c_k")
axes[1, 1].grid(True)

plt.tight_layout()
plt.savefig("awa1.png", dpi=300)

import numpy as np
import matplotlib.pyplot as plt

# дёӯж–Ү
plt.rcParams['font.sans-serif'] = ['SimHei']  # и®ҫзҪ®дёӯж–Үеӯ—дҪ“
plt.rcParams['axes.unicode_minus'] = False    # и§ЈеҶіиҙҹеҸ·жҳҫзӨәй—®йўҳ

# =========================
# еҸӮж•°и®ҫзҪ®
# =========================
sigma = 0.25      # й«ҳж–ҜдҝЎеҸ·е®ҪеәҰ
fs = 4.0          # йҮҮж ·йў‘зҺҮ
Ts = 1 / fs       # йҮҮж ·й—ҙйҡ”

t_min, t_max = -3, 3
f_min, f_max = -12, 12

# =========================
# еҺҹе§Ӣиҝһз»ӯж—¶еҹҹдҝЎеҸ·пјҡй«ҳж–ҜдҝЎеҸ·
# x(t) = exp(-t^2 / (2 sigma^2))
# =========================
t = np.linspace(t_min, t_max, 5000)
x = np.exp(-t ** 2 / (2 * sigma ** 2))

# =========================
# еҺҹе§Ӣиҝһз»ӯйў‘еҹҹ
# й«ҳж–ҜдҝЎеҸ·зҡ„еӮ…йҮҢеҸ¶еҸҳжҚўд»Қз„¶жҳҜй«ҳж–ҜеҪўзҠ¶
# иҝҷйҮҢдё»иҰҒдёәдәҶжј”зӨәеҪўзҠ¶пјҢжүҖд»ҘеҝҪз•Ҙеёёж•°еӣ еӯҗ
# =========================
f = np.linspace(f_min, f_max, 5000)
X = np.exp(-2 * (np.pi ** 2) * (sigma ** 2) * (f ** 2))

# =========================
# ж—¶еҹҹзҰ»ж•ЈеҢ–пјҡйҮҮж ·
# x[n] = x(nTs)
# =========================
n = np.arange(np.ceil(t_min / Ts), np.floor(t_max / Ts) + 1)
tn = n * Ts
xn = np.exp(-tn ** 2 / (2 * sigma ** 2))

# =========================
# йҮҮж ·еҗҺзҡ„йў‘еҹҹпјҡйў‘и°ұе‘ЁжңҹеҢ–
# йҮҮж ·еҗҺйў‘и°ұдјҡд»Ҙ fs дёәе‘ЁжңҹйҮҚеӨҚ
# X_s(f) = sum X(f - kfs)
# =========================
Xs = np.zeros_like(f)
for k in range(-5, 6):
    Xs += np.exp(-2 * (np.pi ** 2) * (sigma ** 2) * ((f - k * fs) ** 2))

# =========================
# з”»еӣҫ
# =========================
fig, axes = plt.subplots(2, 2, figsize=(12, 7))

# дёҠе·ҰпјҡеҺҹе§Ӣиҝһз»ӯж—¶еҹҹ
axes[0, 0].plot(t, x)
axes[0, 0].set_title("еҺҹжқҘпјҡж—¶еҹҹ - иҝһз»ӯдҝЎеҸ·")
axes[0, 0].set_xlabel("t")
axes[0, 0].set_ylabel("x(t)")
axes[0, 0].grid(True)

# дёҠеҸіпјҡеҺҹе§Ӣиҝһз»ӯйў‘еҹҹ
axes[0, 1].plot(f, X)
axes[0, 1].set_title("еҺҹжқҘпјҡйў‘еҹҹ - иҝһз»ӯйқһе‘Ёжңҹ")
axes[0, 1].set_xlabel("f")
axes[0, 1].set_ylabel("X(f)")
axes[0, 1].grid(True)

# дёӢе·ҰпјҡйҮҮж ·еҗҺзҡ„ж—¶еҹҹ
axes[1, 0].stem(tn, xn)
axes[1, 0].set_title("зҺ°еңЁпјҡж—¶еҹҹ - зҰ»ж•ЈеҢ–")
axes[1, 0].set_xlabel("t")
axes[1, 0].set_ylabel("x[n]")
axes[1, 0].grid(True)

# дёӢеҸіпјҡйҮҮж ·еҗҺзҡ„йў‘еҹҹ
axes[1, 1].plot(f, Xs)
axes[1, 1].set_title("зҺ°еңЁпјҡйў‘еҹҹ - е‘ЁжңҹеҢ–")
axes[1, 1].set_xlabel("f")
axes[1, 1].set_ylabel("X_s(f)")
axes[1, 1].grid(True)

plt.tight_layout()
plt.savefig("awa2.png", dpi=300)
