短时傅里叶变换 - STFT

DFT 只能告诉你整段信号里有哪些频率 STFT 进一步告诉你：这些频率分别在什么时候出现 (比如一个信号前 1 秒是 5 Hz，后 1 秒是 20 Hz)

核心思想: 把信号切成一小段一小段，每一段分别做 DFT

原始信号：
|------------------------------------------------|

切成小窗口：
|------|
   |------|
      |------|
         |------|

最后就得到 时间 × 频率 的二维图，也就是频谱图 / spectrogram

连续时间 STFT 定义为：

$$ X(\tau, f)=\int_{-\infty}^{+\infty}x(t)w(t-\tau)e^{-j2\pi ft}dt $$

离散时间 STFT 定义为：

$$ X[m,k]=\sum_{n=0}^{N-1}x[n+mH]w[n]e^{-j\frac{2\pi}{N}kn} $$
  • 多了窗口编号 $m$
  • 多了窗函数 $w[n]$

窗口长度 $\text{win\_len}$ 最重要:

  • 窗口短
    • 优点: 时间定位准
    • 缺点: 频率分辨率差
  • 窗口长
    • 优点: 时间定位模糊
    • 缺点: 频率分辨率好

$\text{hop length}$: 窗口每次移动多少点

  • 重叠多一点，时频图会更平滑，但计算量更大

$\text{n\_fft}$: 每个窗口做 DFT 时的点数

  • 常见设置: $\text{n\_fft} >= \text{win\_len}$
  • 不会真的提高物理上的频率分辨率，但会让频率轴采样更密，图看起来更细腻

代码里: f_s=200Hz, win_len=64, 所以一个窗口覆盖的真实时间是 $\frac{\text{win\_len}}{\text{f\_s}}=\frac{64}{200}=0.32s$

  • 如果提高win_len=128, 则频谱图的频率分辨率更高
  • 如果提高f_s=1000Hz, 则因为一个窗口覆盖的真实时间变短, 频谱图的频率分辨率更低 (此时需要让win_len=320才能达到之前的效果)

代码 

"""
STFT：短时傅里叶变换
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

教学要点：
  1. DFT 只能分析整段信号包含哪些频率
  2. STFT 把信号切成一小段一小段，每一段分别做 DFT
  3. STFT 可以观察频率随时间的变化
  4. 窗口长度 N 决定时间分辨率和频率分辨率
  5. 窗口越短，时间定位越准，但频率分辨率越差
  6. 窗口越长，频率分辨率越好，但时间定位越模糊
"""

import numpy as np
import matplotlib.pyplot as plt

# ── 信号参数 ──
fs = 200                    # 采样频率 200 Hz
T = 2.0                     # 总时长 2 秒
t = np.arange(0, T, 1 / fs)

# ── 构造一个频率随时间变化的信号 ──
# 0~1s: 5Hz
# 1~2s: 20Hz
x = np.zeros_like(t)
x[t < 1.0] = np.sin(2 * np.pi * 5 * t[t < 1.0])
x[t >= 1.0] = np.sin(2 * np.pi * 20 * t[t >= 1.0])

# ── STFT 参数 ──
win_len = 64                # 窗口长度 N
hop = 8                     # 窗口每次移动 H
n_fft = 256                 # DFT 点数，通常 >= win_len

# Hann窗
window = np.hanning(win_len)

# ── 手写 STFT ──
frames = []
time_centers = []

for start in range(0, len(x) - win_len + 1, hop):
    # 取出一小段信号
    frame = x[start:start + win_len]

    # 乘窗函数
    frame_windowed = frame * window

    # 对这一小段做 DFT
    X = np.fft.rfft(frame_windowed, n=n_fft)

    # 保存幅度谱
    frames.append(np.abs(X))

    # 记录当前窗口中心时间
    center = start + win_len // 2
    time_centers.append(center / fs)

# shape: (num_frames, num_freq_bins)
S = np.array(frames)

# 转成 dB，显示更清楚
S_db = 20 * np.log10(S + 1e-8)

# 频率轴，只保留非负频率
freqs = np.fft.rfftfreq(n_fft, d=1 / fs)

# ── 画图 ──
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))

# 左图：时域信号
ax1.plot(t, x, linewidth=1.2)
ax1.axvline(1.0, linestyle='--', alpha=0.7, label='frequency changes at 1s')
ax1.set_xlabel('t (s)')
ax1.set_ylabel('x(t)')
ax1.set_title('Time-domain signal: 5Hz then 20Hz')
ax1.grid(True, alpha=0.3)
ax1.legend(loc='upper right')

# 右图：STFT 时频图
im = ax2.imshow(
    S_db.T,
    origin='lower',
    aspect='auto',
    extent=[time_centers[0], time_centers[-1], freqs[0], freqs[-1]]
)

ax2.set_xlabel('time (s)')
ax2.set_ylabel('frequency (Hz)')
ax2.set_title('STFT spectrogram')
ax2.set_ylim(0, 40)

fig.colorbar(im, ax=ax2, label='magnitude (dB)')

fig.suptitle('STFT: frequency changes over time',
             fontsize=14, fontweight='bold')
plt.tight_layout(rect=[0, 0, 1, 0.92])
plt.savefig('stft_sin.png', dpi=150, bbox_inches='tight')
