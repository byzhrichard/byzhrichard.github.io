拉普拉斯变换 - Laplace

CTFT 只看虚轴上的频率 -> CTFT 用的是 $e^{-j\Omega t}$ Laplace 看整个复平面 -> Laplace 用的是 $e^{-st}$ 其中 $s=\sigma+j\Omega$

  • $σ$：看信号的增长/衰减特性
  • $Ω$：看信号的振荡频率
  • $s$ = 衰减/增长 + 振荡

Laplace:

$$ X(s)=\int_{0}^{+\infty}x(t)e^{-st}dt \\ x(t)=\mathcal{L}^{-1}\{X(s)\} $$

极点所在处值很大 -> 对应到图上就是很亮

  • 如果极点在左半平面：$\sigma < 0$ -> 一般代表响应会衰减 -> 系统稳定
  • 如果极点在右半平面：$\sigma > 0$ -> 一般代表响应会增长 -> 系统不稳定
  • 如果极点在虚轴上：$\sigma = 0$ -> 一般代表响应会持续振荡
变换 处理对象 频率变量 看哪里 主要用途
CTFT 连续时间信号 $j\Omega$ 虚轴 看频率成分
Laplace 连续时间信号 / 系统 $s=\sigma+j\Omega$ 整个 s 平面 看频率 + 增长衰减 + 稳定性
时域信号 $x(t)$ Laplace 结果 $X(s)$ 收敛条件 ROC 直观理解
$\delta(t)$ $1$ 全 $s$ 平面 冲激包含所有复频率
$u(t)$ $\frac{1}{s}$ $\operatorname{Re}(s)>0$ 单位阶跃
$t u(t)$ $\frac{1}{s^2}$ $\operatorname{Re}(s)>0$ 斜坡信号
$t^n u(t)$ $\frac{n!}{s^{n+1}}$ $\operatorname{Re}(s)>0$ 多项式增长
$e^{-at}u(t)$ $\frac{1}{s+a}$ $\operatorname{Re}(s)>-a$ 指数衰减
$e^{at}u(t)$ $\frac{1}{s-a}$ $\operatorname{Re}(s)>a$ 指数增长
$\sin(\Omega_0 t)u(t)$ $\frac{\Omega_0}{s^2+\Omega_0^2}$ $\operatorname{Re}(s)>0$ 因果正弦
$\cos(\Omega_0 t)u(t)$ $\frac{s}{s^2+\Omega_0^2}$ $\operatorname{Re}(s)>0$ 因果余弦
$e^{-at}\sin(\Omega_0 t)u(t)$ $\frac{\Omega_0}{(s+a)^2+\Omega_0^2}$ $\operatorname{Re}(s)>-a$ 衰减振荡
$e^{-at}\cos(\Omega_0 t)u(t)$ $\frac{s+a}{(s+a)^2+\Omega_0^2}$ $\operatorname{Re}(s)>-a$ 衰减余弦振荡

代码 

"""
Laplace：拉普拉斯变换
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

教学要点：
  1. Laplace 变换用于分析连续时间信号和连续时间系统
  2. Laplace 变量 s = σ + jΩ，是复频率
  3. CTFT 只看虚轴 jΩ，Laplace 看整个 s 平面
  4. 对因果正弦 x(t)=sin(Ω0 t)u(t)，有：
       X(s)=Ω0/(s²+Ω0²)
  5. 极点位于 s=±jΩ0
"""

import numpy as np
import matplotlib.pyplot as plt

# ── 信号参数 ──
f0 = 5                         # 正弦频率 5 Hz
Omega0 = 2 * np.pi * f0        # Ω0 = 2πf0，单位 rad/s

# ── 时域信号：因果正弦 ──
t = np.linspace(0, 1.0, 1000)
x = np.sin(Omega0 * t)

# ── s 平面网格 ──
# s = σ + jΩ
sigma = np.linspace(-20, 20, 500)
Omega = np.linspace(-60, 60, 500)

SIGMA, OMEGA = np.meshgrid(sigma, Omega)
s = SIGMA + 1j * OMEGA

# ── Laplace 变换 ──
# X(s) = Ω0 / (s^2 + Ω0^2)
X = Omega0 / (s**2 + Omega0**2)

# 幅度，为了显示更清楚使用 log
mag = np.abs(X)
mag_log = np.log10(mag + 1e-6)

# 防止极点附近数值过大导致颜色图看不清
mag_log = np.clip(mag_log, -3, 2)

# ── 极点位置 ──
poles_sigma = [0, 0]
poles_omega = [Omega0, -Omega0]

# ── 画图 ──
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))

# 左图：时域因果正弦
ax1.plot(t, x, linewidth=1.2)
ax1.set_xlabel('t (s)')
ax1.set_ylabel('x(t)')
ax1.set_title(f'x(t) = sin(2π·{f0}t)u(t)')
ax1.grid(True, alpha=0.3)

# 右图：s 平面幅度
im = ax2.imshow(
    mag_log,
    origin='lower',
    aspect='auto',
    extent=[sigma[0], sigma[-1], Omega[0], Omega[-1]]
)

# 虚轴 σ=0
ax2.axvline(0, linestyle='--', alpha=0.8, label='jΩ axis')

# 标记极点
ax2.scatter(poles_sigma, poles_omega, marker='x', s=100, linewidths=2, color='red', label='poles')

ax2.set_xlabel('σ')
ax2.set_ylabel('Ω (rad/s)')
ax2.set_title(r'Laplace magnitude on s-plane: $|X(s)|$')
ax2.grid(True, alpha=0.2)
ax2.legend(loc='upper right')

fig.colorbar(im, ax=ax2, label='log10 |X(s)|')

fig.suptitle('Laplace Transform of Causal Sin Signal',
             fontsize=14, fontweight='bold')
plt.tight_layout(rect=[0, 0, 1, 0.92])
plt.savefig('laplace_sin.png', dpi=150, bbox_inches='tight')
