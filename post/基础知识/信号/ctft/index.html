连续时间傅里叶变换 - CTFT

一个连续时间信号里，到底含有哪些频率成分

CTFT 不关心信号“长什么样”，CTFT 关心的是：这个信号由哪些频率组成？

连续时间信号：

$$ x(t)=\sin(\Omega_0 t)=\sin(2\pi f_0 t) $$

CTFT:

$$ X(j\Omega)=\int_{-\infty}^{+\infty}x(t)e^{-j\Omega t}dt \\ x(t)=\frac{1}{2\pi}\int_{-\infty}^{+\infty}X(j\Omega)e^{j\Omega t}d\Omega $$

傅里叶变换里，实数正弦通常会表现成 正负两个频率分量

$e^{−jΩt}$ 可以理解成“频率探测器”:

  • 如果 $x(t)$ 里面真的含有这个频率，那么积分结果就大
  • 如果不含这个频率，正负抵消后结果就接近 0
时域信号 $x(t)$ 频域结果 $X(j\Omega)$ 直观理解
$\delta(t)$ $1$ 冲激包含所有频率
$1$ $2\pi\delta(\Omega)$ 常数只有 0 频率，也就是直流分量
$e^{j\Omega_0 t}$ $2\pi\delta(\Omega-\Omega_0)$ 纯复指数只有一个频率 $\Omega_0$
$e^{-j\Omega_0 t}$ $2\pi\delta(\Omega+\Omega_0)$ 负频率复指数
$\cos(\Omega_0 t)$ $\pi[\delta(\Omega-\Omega_0)+\delta(\Omega+\Omega_0)]$ cos 对应正负两个频率，幅度相同
$\sin(\Omega_0 t)$ $\frac{\pi}{j}[\delta(\Omega-\Omega_0)-\delta(\Omega+\Omega_0)]$ sin 也对应正负两个频率，但符号/相位不同
$e^{-at}u(t),\ a>0$ $\frac{1}{a+j\Omega}$ 衰减指数，频谱是连续的
$\text{rect}\left(\frac{t}{T}\right)$ $T\operatorname{sinc}\left(\frac{\Omega T}{2}\right)$ 矩形窗对应 sinc 形频谱

代码中: T_obs取了2秒, 没法$\int_{-\infty}^{+\infty}$, 所以看起来不像冲激 如果调大T_obs则会越来越接近冲激

代码 

"""
CTFT：连续时间傅里叶变换
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

教学要点：
  1. 连续信号 x(t)=sin(2π·f0·t)
  2. CTFT 用来分析连续时间信号包含哪些频率成分
  3. 理论上，无限长 sin 信号的 CTFT 是 ±f0 处的两个冲激
  4. 代码中只能截取有限时间长度，因此画出来的是 CTFT 的数值近似
  5. 有限时间截断会导致频谱主峰周围出现扩散，也就是频谱泄漏
"""

import numpy as np
import matplotlib.pyplot as plt

# ── 连续信号参数 ──
f0 = 5                       # 正弦信号频率 5 Hz
Omega0 = 2 * np.pi * f0      # 模拟角频率 Ω0 = 2πf0，单位 rad/s

# ── 数值近似参数 ──
fs_plot = 1000               # 用很高的采样率模拟连续时间
T_obs = 2.0                  # 观察 2 秒信号 (T_obs越大, 则频谱越接近理论冲激)
dt = 1 / fs_plot             # 时间间隔

# 连续时间轴
t = np.arange(-T_obs / 2, T_obs / 2, dt)

# 连续时间信号 x(t)
x = np.sin(2 * np.pi * f0 * t)

# ── 用 FFT 近似 CTFT ──
# CTFT: X(f)=∫x(t)e^(-j2πft)dt
# 数值积分近似：积分 ≈ 求和 * dt
X = np.fft.fftshift(np.fft.fft(np.fft.ifftshift(x))) * dt

# 频率轴，单位 Hz
freq = np.fft.fftshift(np.fft.fftfreq(len(t), d=dt))

# 幅度谱
mag = np.abs(X)

# 为了教学展示，把幅度归一化
mag_norm = mag / np.max(mag)

# ── 画图 ──
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))

# 左图：连续时间信号
ax1.plot(t, x, linewidth=1.2)
ax1.set_xlabel('t (s)')
ax1.set_ylabel('x(t)')
ax1.set_title(f'x(t) = sin(2π·{f0}t)')
ax1.grid(True, alpha=0.3)
ax1.set_xlim(-0.5, 0.5)

# 右图：CTFT 数值近似
ax2.plot(freq, mag_norm, linewidth=1.2, label='Numerical CTFT approximation')

# 理论冲激位置
ax2.axvline(f0, linestyle='--', alpha=0.8, label=f'+f0 = {f0} Hz')
ax2.axvline(-f0, linestyle='--', alpha=0.8, label=f'-f0 = {-f0} Hz')

ax2.set_xlabel('f (Hz)')
ax2.set_ylabel('|X(f)| normalized')
ax2.set_title('CTFT of sin: impulses at ±f0')
ax2.grid(True, alpha=0.3)
ax2.set_xlim(-20, 20)
ax2.legend(loc='upper right')

fig.suptitle('CTFT: frequency content of continuous-time sin signal',
             fontsize=14, fontweight='bold')
plt.tight_layout(rect=[0, 0, 1, 0.92])
plt.savefig('ctft_sin.png', dpi=150, bbox_inches='tight')
