连续时间矩形窗

设窗口宽度为 $T$，矩形窗可以写成：

$$ w(t)= \begin{cases} 1, & |t|\le \frac{T}{2}\\ 0, & |t|>\frac{T}{2} \end{cases} $$

也可以写成：

$$ w(t)=\operatorname{rect}\left(\frac{t}{T}\right) $$

y = rect(x): 当 x > 1/2 时, 为 0; 否则为 1

矩形窗的傅里叶变换

矩形窗的傅里叶变换是 sinc 形状

$$ \begin{align} W(f)&=T\operatorname{sinc}(fT) \\ &=T\frac{\sin(\pi fT)}{\pi fT} \\ &=\frac{\sin(\pi fT)}{\pi f} \end{align} $$

$\operatorname{sinc}(x)=\frac{\sin(\pi x)}{\pi x}$

为什么矩形窗频谱是 sinc

因为矩形窗在时域有两个非常突然的边缘：

  • 0 → 1 突然跳变
  • 1 → 0 突然跳变

这种突变需要很多频率成分才能合成出来

-> 频域会出现长长的旁瓣

代码 

import numpy as np
import matplotlib.pyplot as plt

# =========================
# 参数设置
# =========================
T = 1.0          # 矩形窗宽度
f_max = 10       # 频率显示范围

# =========================
# 连续时间矩形窗
# w(t) = 1, |t| <= T/2
# =========================
t_rect = np.array([-1.5*T, -T/2, -T/2, T/2, T/2, 1.5*T])
w_rect = np.array([0,      0,    1,    1,   0,    0])

# =========================
# 连续频谱
# W(f) = T * sinc(fT)
# 注意：np.sinc(x) = sin(pi*x)/(pi*x)
# =========================
f = np.linspace(-f_max, f_max, 5000)
W = T * np.sinc(f * T)
W_mag = np.abs(W)

# 归一化幅度谱
W_mag_norm = W_mag / np.max(W_mag)

# =========================================================
# 图：时域矩形窗 + 频域幅度谱
# =========================================================
fig, axes = plt.subplots(1, 2, figsize=(12, 4))

# --- 时域 ---
axes[0].plot(
    t_rect,
    w_rect,
    linewidth=2,
    label=r"$w(t)=\mathrm{rect}\left(\frac{t}{T}\right)=\mathrm{rect}\left(\frac{t}{%.2f}\right)$" % (T,)
)
axes[0].set_title("Time Domain")
axes[0].set_xlabel("t")
axes[0].set_ylabel("w(t)")
axes[0].set_ylim(-0.4, 1.6)
axes[0].grid(True)
axes[0].legend()

# --- 频域 ---
axes[1].plot(
    f,
    W_mag_norm,
    linewidth=2,
    label=r"$|W(f)|=|T\,\mathrm{sinc}(fT)|=|%.2f\times\mathrm{sinc}(f\times%.2f)|$" % (T, T)
)
axes[1].set_title("Frequency Domain")
axes[1].set_xlabel("Frequency f")
axes[1].set_ylabel(r"$|W(f)|$")
axes[1].set_ylim(-0.2, 1.2)
axes[1].grid(True)
axes[1].legend()

plt.tight_layout()
plt.savefig("awa1.png", dpi=300)
