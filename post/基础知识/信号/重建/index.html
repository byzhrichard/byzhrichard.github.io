重建

采样是：$x(t)\rightarrow x[n]$ 重建就是反过来：$x[n]\rightarrow \hat{x}(t)$

核心思想: 根据离散点，重新生成一条连续曲线

1. 理想重建：sinc 插值

如果满足奈奎斯特条件：

$$ f_s>2f_{\max} $$

理论上可以完美重建：

$$ x(t)=\sum_{n=-\infty}^{+\infty}x[n]\operatorname{sinc}\left(\frac{t-nT}{T}\right) $$

  • 每一个采样点 $x[n]$，都会生成一个以 $nT$ 为中心的 sinc 函数

  • 所有 sinc 函数加起来，就恢复出连续信号

    采样点 1 → 一个 sinc
采样点 2 → 一个 sinc
采样点 3 → 一个 sinc
...
全部叠加 → 连续信号

2. 为什么是 sinc

从频域看，采样后频谱会周期复制 -> 如果没有混叠，这些频谱副本不会重叠

采样后频谱：

      原频谱      复制频谱      复制频谱
        |             |             |
--------|-------------|-------------|------→ f
       0            fs           2fs

重建时，只需要用一个理想低通滤波器，把中间原始频谱留下，把其他复制频谱去掉 -> 而理想低通滤波器在时域里的冲激响应就是 sinc

3. 现实中怎么重建

理论上是 sinc，但现实里不能真正实现无限长的 sinc

近似方法：

方法 形式 特点
零阶保持 ZOH 每个采样值保持一段时间 最简单，输出像阶梯
线性插值 相邻点之间连直线 简单，比阶梯平滑
三次插值 用曲线连接采样点 更平滑
有限长 sinc 插值 截断 sinc 后使用 接近理想重建
DAC + 低通滤波 数字转模拟后滤波 工程常见

代码里采用的是: 有限长度 sinc 插值重建

代码 

"""
采样与 sinc 重建示例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

教学要点：
  1. 连续信号 x(t)=sin(2πf0t)
  2. 采样得到 x[n]=x(nT)
  3. 如果 fs > 2f0，理论上可以用 sinc 插值恢复连续信号
  4. sinc 重建公式：
       x(t)=Σ x[n] sinc((t-nT)/T)
"""

import numpy as np
import matplotlib.pyplot as plt

# ── 信号参数 ──
f0 = 5          # 原始信号频率 5 Hz
fs = 30         # 采样频率 30 Hz，满足 fs > 2*f0
T = 1 / fs      # 采样间隔
duration = 1.0  # 信号时长 1 秒

# ── 高分辨率时间轴：模拟“连续时间” ──
t_cont = np.linspace(0, duration, 3000)
x_cont = np.sin(2 * np.pi * f0 * t_cont)

# ── 采样 ──
n = np.arange(0, int(duration * fs) + 1)
t_sample = n * T
x_sample = np.sin(2 * np.pi * f0 * t_sample)

# ── sinc 重建 ──
# numpy 的 sinc 定义是 sinc(x)=sin(pi*x)/(pi*x)
# 所以刚好对应公式 sinc((t-nT)/T)
x_recon = np.zeros_like(t_cont)

for i in range(len(n)):
    x_recon += x_sample[i] * np.sinc((t_cont - t_sample[i]) / T)

# ── 画图 ──
plt.figure(figsize=(11, 5))

plt.plot(t_cont, x_cont, linewidth=1.5, label='Original continuous signal')
plt.plot(t_cont, x_recon, '--', linewidth=1.5, label='Sinc reconstruction')
plt.stem(
    t_sample,
    x_sample,
    linefmt='C2-',
    markerfmt='C2o',
    basefmt=' ',
    label='Samples'
)

plt.xlabel('t (s)')
plt.ylabel('x(t)')
plt.title(f'Sampling and sinc reconstruction: f0={f0}Hz, fs={fs}Hz')
plt.grid(True, alpha=0.3)
plt.legend()
plt.tight_layout()
plt.savefig('sinc_reconstruction.png', dpi=150, bbox_inches='tight')
# plt.show()
