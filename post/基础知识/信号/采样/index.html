采样

连续信号: $x(t)=\sin(\Omega t)=\sin(2\pi f t)$ -> 模拟角频率: $\Omega$ (rad/s) -> 模拟频率: $f$ (Hz)

离散信号: $x[n]=sin(\omega n)=sin(2\pi f_d n)$ -> 数字角频率: $\omega$ (rad) -> 数字频率: $f_d$ (Hz)

采样频率: $f_s$ 采样间隔: $T$ 有 $T=1/f_s$ -> $t=nT$ -> $n=t/T$

二者关系推导: $x[n]=x(nT)=sin(2π·f·n/f_s)=sin(ω·n)$

所以: $\bold{\omega=\Omega T=2\pi f/f_s}$

得到最常用公式:

  • 连续信号: $x(t)=\sin(2\pi f t)$
  • 离散信号: $x[n]=sin(2\pi \frac{f}{f_s} n)$

1. 混叠

如果采样率太低，不同的连续频率可能采出来一模一样 -> 混叠 aliasing

2. 奈奎斯特采样定理

如果信号最高频率是：

$$ f_{\max} $$

那么采样频率应该满足：

$$ \bold{f_s > 2f_{\max}} $$

因为 DTFT 是以 $2\pi$ 为周期的, 范围划定为 $-\pi \sim \pi$

采样后：

$$ > \omega = 2\pi\frac{f}{f_s} > $$

如果你希望最高频率 $f_{\max}$ 不超过数字频率的边界 $\pi$，就要满足：

$$ > 2\pi\frac{f_{\max}}{f_s} < \pi > $$

所以 $f_s>2f_{\max}$ 才能让 DTFT 包含所有频率

拓: 满足奈奎斯特采样定理后, 才可以通过 理想 sinc 插值 重建原始信号

3. 作用 

连续信号 x(t)
    ↓ 采样
离散信号 x[n]
    ↓ DTFT
离散信号的完整频谱
    ↓ DFT
电脑能算的有限频率点
阶段 对象 工具
连续时间信号 $x(t)$ CTFT
采样后 $x[n]=x(nT)$ DTFT
有限长度序列 $x[0],...,x[N-1]$ DFT/FFT
时间变化频率 分段信号 STFT/CWT

代码 

"""
采样：从连续信号 x(t) 到离散信号 x[n]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

教学要点：
  1. 连续信号 x(t)=sin(2π·f·t)，f 是模拟频率（Hz）
  2. 以采样率 fs 采样，采样间隔 T=1/fs
  3. 离散信号 x[n]=x(nT)=sin(2π·f·n/fs)=sin(ω·n)
     数字角频率 ω = ΩT = 2π·f/fs  (rad/sample)
  4. 左图：连续波形 + 采样点叠加，直观感受"采样的物理意义"
  5. 右图：纯离散的 x[n] stem 图，横轴变成整数 n
"""

import numpy as np
import matplotlib.pyplot as plt

# ── 连续信号参数 ──
f = 5                      # 模拟频率 5 Hz
fs = 20                    # 采样率 20 Hz（每个周期采4个点）
T = 1 / fs                 # 采样间隔 T = 0.05s

# 模拟角频率 Ω 和数字角频率 ω
Omega = 2 * np.pi * f      # Ω = 2πf，单位 rad/s
omega = Omega * T          # ω = ΩT = 2πf/fs，单位 rad/sample

# 连续时间信号 x(t)
t_ct = np.linspace(0, 0.5, 500)
x_ct = np.sin(Omega * t_ct)

# 离散时间信号 x[n]：在 t=nT 时刻的采样值
n = np.arange(0, int(0.5 * fs) + 1)      # n = 0,1,...,10
x_dt = np.sin(omega * n)                 # x[n] = sin(ωn)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))

# 左图：连续波形 + 采样点，展示"从曲线上取点"
ax1.plot(t_ct, x_ct, linewidth=1.2, alpha=0.5)
ax1.stem(n * T, x_dt, linefmt='C1-', markerfmt='C1o', basefmt=' ')
ax1.set_xlabel('t (s)')
ax1.set_ylabel('x(t)')
ax1.set_title(f'x(t) = sin(2π·{f}t)  +  sampling at fs={fs}Hz')
ax1.grid(True, alpha=0.3)

# 右图：纯离散序列，横轴为整数 n
ax2.stem(n, x_dt, linefmt='C1-', markerfmt='C1o', basefmt=' ')
ax2.set_xlabel('n (sample)')
ax2.set_ylabel('x[n]')
ax2.set_title(
    f'x[n] = x(nT) = sin(ωn),  ω = 2π·{f}/{fs} = {2*f/fs:.2f}π rad/sample'
)
ax2.grid(True, alpha=0.3)
ax2.set_xlim(-0.5, len(x_dt) - 0.5)

fig.suptitle('Sampling: x(t) to x[n]', fontsize=14, fontweight='bold')
plt.tight_layout(rect=[0, 0, 1, 0.92])
plt.savefig('sampling.png', dpi=150, bbox_inches='tight')
