离散时间傅里叶变换 - DTFT

CTFT 是看连续信号有哪些频率 DTFT 是看离散信号有哪些频率

离散时间信号：

$$ x[n]=\sin(\omega_0 n)=\sin(2\pi f_d n) $$
  • 采 样频率: $f_s$
  • $\omega_0=2\pi\frac{f_0}{f_s}$

DTFT:

$$ X(e^{j\omega})=\sum_{n=-\infty}^{+\infty}x[n]e^{-j\omega n} \\ x[n]=\frac{1}{2\pi}\int_{-\pi}^{\pi}X(e^{j\omega})e^{j\omega n}d\omega $$

DTFT 的频率变量 $\omega$ 仍然是连续变化的 -> 离散时间，不等于离散频率

DTFT: 离散时间 → 连续频率

变换 时域 频域
CTFT 连续时间 $x(t)$ 连续频率 $X(j\Omega)$
DTFT 离散时间 $x[n]$ 连续频率 $X(e^{j\omega})$

DTFT 的频率只看 $[-\pi,\pi]$:

  • DTFT 的频谱以 $2\pi$ 为周期 ($X(e^{j(\omega+2\pi)})=X(e^{j\omega})$)
  • DTFT 通常只看 $-\pi \le \omega \le \pi$ 或者 $0 \le \omega < 2\pi$
名称 输入信号 频率结果
DTFT 无限长离散序列 $x[n]$ 连续频率函数 $X(e^{j\omega})$
DFT 有限长离散序列 $x[0],...,x[N-1]$ 有限个频率点 $X[0],...,X[N-1]$

DTFT 是理论版频谱 DFT 是电脑能算的有限点频谱

离散时域信号 $x[n]$ DTFT $X(e^{j\omega})$ 直观理解
$\delta[n]$ $1$ 离散冲激包含所有数字频率
$1$ $2\pi\sum_k\delta(\omega-2\pi k)$ 常数只有 0 频率，但周期重复
$e^{j\omega_0 n}$ $2\pi\sum_k\delta(\omega-\omega_0-2\pi k)$ 单个数字频率，周期重复
$\cos(\omega_0 n)$ $\pi\sum_k[\delta(\omega-\omega_0-2\pi k)+\delta(\omega+\omega_0-2\pi k)]$ 正负频率，周期重复
$\sin(\omega_0 n)$ $\frac{\pi}{j}\sum_k[\delta(\omega-\omega_0-2\pi k)-\delta(\omega+\omega_0-2\pi k)]$ 正负频率，相位不同，周期重复
$a^n u[n],\ a <1$

改变 $f_s$, 则频谱图的位置也会发生变化

本质上是看$\omega_0$, 而$\omega_0=2\pi\frac{f_0}{f_s}$

代码 

"""
DTFT：离散时间傅里叶变换
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

教学要点：
  1. 采样后得到离散信号 x[n]=sin(ω0 n)
  2. 数字角频率 ω0 = 2πf0/fs，单位 rad/sample
  3. DTFT 用来分析离散时间信号 x[n] 的频率组成
  4. 理论上，无限长 sin 的 DTFT 在 ±ω0 处有冲激
  5. DTFT 的频谱以 2π 为周期
  6. 代码中用有限长度序列近似无限长序列，所以峰值附近会有扩散
"""

import numpy as np
import matplotlib.pyplot as plt

# ── 信号参数 ──
f0 = 5                     # 原连续信号频率 5 Hz
fs = 40                    # 采样频率 40 Hz
omega0 = 2 * np.pi * f0 / fs   # 数字角频率 ω0 = 2πf0/fs

# ── 离散时间序列 ──
N = 64                     # 截取 64 个采样点，用来近似无限长序列
n = np.arange(N)
x = np.sin(omega0 * n)

# ── DTFT 数值计算 ──
# DTFT 的频率变量 ω 是连续的
# 为了画图，在 [-2π, 2π] 上取很多个频率点
omega = np.linspace(-2 * np.pi, 2 * np.pi, 2000)

# 根据 DTFT 定义：
# X(e^{jω}) = Σ x[n] e^{-jωn}
X = np.zeros_like(omega, dtype=complex)

for i, w in enumerate(omega):
    X[i] = np.sum(x * np.exp(-1j * w * n))

# 幅度谱
mag = np.abs(X)

# 为了教学展示，把幅度归一化
mag_norm = mag / np.max(mag)

# ── 画图 ──
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))

# 左图：离散时间信号 x[n]
ax1.stem(n, x, linefmt='C1-', markerfmt='C1o', basefmt=' ')
ax1.set_xlabel('n (sample)')
ax1.set_ylabel('x[n]')
ax1.set_title(
    f'x[n] = sin(ω0 n),  ω0 = 2π·{f0}/{fs} = {2*f0/fs:.2f}π rad/sample'
)
ax1.grid(True, alpha=0.3)
ax1.set_xlim(-1, 32)

# 右图：DTFT 幅度谱
ax2.plot(omega / np.pi, mag_norm, linewidth=1.2)

# 理论冲激位置：±ω0，以及周期重复的位置 ±ω0 ± 2π
peak_positions = [
    -omega0 - 2 * np.pi,
    omega0 - 2 * np.pi,
    -omega0,
    omega0,
    -omega0 + 2 * np.pi,
    omega0 + 2 * np.pi,
]

for pos in peak_positions:
    if -2 * np.pi <= pos <= 2 * np.pi:
        ax2.axvline(pos / np.pi, linestyle='--', alpha=0.7)

ax2.set_xlabel('ω / π')
ax2.set_ylabel('|X(e^{jω})| normalized')
ax2.set_title('DTFT magnitude: peaks at ±ω0, repeated every 2π')
ax2.grid(True, alpha=0.3)
ax2.set_xlim(-2, 2)

fig.suptitle('DTFT: frequency content of discrete-time sin signal',
             fontsize=14, fontweight='bold')
plt.tight_layout(rect=[0, 0, 1, 0.92])
plt.savefig('dtft_sin.png', dpi=150, bbox_inches='tight')
