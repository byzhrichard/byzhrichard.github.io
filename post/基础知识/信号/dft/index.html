离散傅里叶变换 - DFT

DTFT 是理论上的连续频率谱 -> $\omega$ 可以是任意值 DFT 是电脑真正能算的有限个频率点 -> 只看 $N$ 个固定频率点 $\omega_0, \omega_1,..., \omega_{N-1}$ -> DTFT 是离散信号的完整频谱，而 DFT 是把 DTFT 采样成有限个点 -> 而 $X(e^{j\omega})$ 以 $2\pi$ 为周期, 所以 DFT 只需要在一个周期 $0\sim 2\pi$ 里取点

$$ \omega_k=\frac{2\pi}{N}k $$

DFT 的基函数是 $e^{j\frac{2\pi}{N}kn}$, 其中 $n=0,1,\cdots,N-1$

当 $n$ 从 $0$ 走到 $N$ 时，相位变化是：$\frac{2\pi}{N}k \cdot N = 2\pi k$ 也就是: 刚好转了 $k$ 圈

-> 第 $k$ 个 DFT 频率，就是“每个采样点前进 $\frac{2\pi k}{N}$ 弧度”，这样 $N$ 个点刚好转 $k$ 圈

DFT 的第 $k$ 个标准频率波形是：$e^{j\frac{2\pi}{N}kn}$ (表示: 在 $N$ 个采样点里，刚好转 $k$ 圈的复指数波形)

$k$ 标准波形 标准频率波形的含义
$k=0$ $1$ $N$ 个点里转 0 圈，也就是常数(用来检测直流分量)
$k=1$ $e^{j\frac{2\pi}{N}n}$ $N$ 个点里转 1 圈
$k=2$ $e^{j\frac{2\pi}{N}2n}$ $N$ 个点里转 2 圈
$k=5$ $e^{j\frac{2\pi}{N}5n}$ $N$ 个点里转 5 圈
$k=N-1$ $e^{j\frac{2\pi}{N}(N-1)n}$ 等价于转 -1 圈

以 $N=8$ 为例:

比如 $k=0$ 时, $e^{j\frac{2\pi}{8}\cdot 0\cdot n}=1$, 所以它是不变的常数：

1, 1, 1, 1, 1, 1, 1, 1

-> 用来检测直流分量

有限长离散序列:

$$ x[0],x[1],...,x[N-1] $$

DFT:

$$ X[k]=\sum_{n=0}^{N-1}x[n]e^{-j\frac{2\pi}{N}kn} \\ x[n]=\frac{1}{N}\sum_{k=0}^{N-1}X[k]e^{j\frac{2\pi}{N}kn} $$

DFT 的第 $k$ 个频率点对应：

$$ \omega_k=\frac{2\pi}{N}k $$

如果有采样频率 $f_s$，则第 $k$ 个频率点对应的实际频率 $f_k$ 是：

$$ f_k=\frac{k}{N}f_s $$

推导:

$$ > f_k=\frac{\omega_k}{2\pi}f_s=\frac{k}{N}f_s > $$

定义频率分辨率为 $\Delta f$：每个 DFT 频点的间隔 (Hz)

$$ \Delta f=\frac{f_s}{N} $$

对于信号: $x[n]=\sin(2\pi\cdot f_0n/f_s)=\sin(2\pi\cdot 5n/64)$, 其 $\omega_0 = 2\pi\frac{5}{64}$

如果把 $2\pi$ 分为 $N=64$ 份, 因为 $\omega_k=\frac{2\pi}{N}k$, 所以 $\omega_5=w_0$

-> 对该信号求DFT得的频谱图中, $k=5$ 的 $X[5]$ 就是该信号在第 5 个频率点上的频谱系数

-> DFT 主要峰值位置: $k=5$ 和 $k=64-5=59$ (因为 sin 是实信号，所以还会在负频率位置 $k=64-5=59$ 出现对应峰值)

如果 $N=64$ 改成 $N=128$, 则 $\Delta f = 0.5Hz$, 所以峰值还在 ±5Hz 的位置

如果 $N=64$ 改成 $N=32$, 则 $\Delta f = 2Hz$, 所以原先峰值的能量被周边几个峰瓜分 -> 频谱泄漏

快速傅里叶算法 - FFT

FFT 是快速计算 DFT 的方法

  • DFT：我要算什么
  • FFT：我怎么快速算

代码 

"""
DFT：离散傅里叶变换
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

教学要点：
  1. DFT 处理的是有限长离散序列 x[0], x[1], ..., x[N-1]
  2. DFT 得到 N 个离散频率点 X[0], X[1], ..., X[N-1]
  3. DFT 可以看作：对 DTFT 的频率轴取 N 个采样点
  4. 对 x[n]=sin(2πf0 n/fs)，若 f0 正好落在 DFT 频点上，频谱会在 ±f0 处出现峰值
  5. DFT 中负频率通常出现在频谱后半部分
"""

import numpy as np
import matplotlib.pyplot as plt

# ── 信号参数 ──
fs = 64                    # 采样频率 64 Hz
N = 64                     # DFT 点数，也是截取的样本点数
f0 = 5                     # 正弦信号频率 5 Hz

# ── 离散时间信号 ──
n = np.arange(N)
x = np.sin(2 * np.pi * f0 * n / fs)

# ── DFT 计算 ──
X = np.fft.fft(x)          # 计算 N 点 DFT
mag = np.abs(X)            # 幅度谱

# 频率轴：0 到 fs
freq = np.fft.fftfreq(N, d=1/fs)

# 为了画得更直观，把负频率移到左边
X_shift = np.fft.fftshift(X)
mag_shift = np.abs(X_shift)
freq_shift = np.fft.fftshift(freq)

# 幅度归一化，方便教学展示
mag_shift_norm = mag_shift / np.max(mag_shift)

# ── 画图 ──
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))

# 左图：有限长离散信号
ax1.stem(n, x, linefmt='C1-', markerfmt='C1o', basefmt=' ')
ax1.set_xlabel('n (sample)')
ax1.set_ylabel('x[n]')
ax1.set_title(f'x[n] = sin(2π·{f0}·n/{fs}),  N={N}')
ax1.grid(True, alpha=0.3)
ax1.set_xlim(-1, 32)

# 右图：DFT 幅度谱，使用 fftshift 后的双边频谱
ax2.stem(freq_shift, mag_shift_norm, linefmt='C1-', markerfmt='C1o', basefmt=' ')
ax2.axvline(f0, linestyle='--', alpha=0.7, label=f'+f0 = {f0} Hz')
ax2.axvline(-f0, linestyle='--', alpha=0.7, label=f'-f0 = {-f0} Hz')

ax2.set_xlabel('frequency (Hz)')
ax2.set_ylabel('|X[k]| normalized')
ax2.set_title('DFT magnitude spectrum')
ax2.grid(True, alpha=0.3)
ax2.set_xlim(-20, 20)
ax2.legend()

fig.suptitle('DFT: frequency samples of finite discrete-time signal',
             fontsize=14, fontweight='bold')
plt.tight_layout(rect=[0, 0, 1, 0.92])
plt.savefig('dft_sin.png', dpi=150, bbox_inches='tight')
