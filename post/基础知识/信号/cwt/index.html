连续小波变换 - CWT

STFT：用固定长度的窗口去扫信号 CWT：用可以缩放的小波去扫信号

信号里的频率特点不一样：

  • 低频变化慢，需要更长时间才能看清楚
  • 高频变化快，用短窗口就够了，还能更准地定位时间

CWT 的想法是：

  • 高频：用短小波，时间定位更准
  • 低频：用长小波，频率分辨率更好

CWT: Continuous Wavelet Transform: 连续小波变换

$$ W_x(a,b)=\frac{1}{\sqrt{a}}\int_{-\infty}^{+\infty}x(t)\psi^*\left(\frac{t-b}{a}\right)dt $$
  • $x(t)$: 原始信号
  • $\psi(t)$: 母小波
  • $\psi^*(t)$: 小波的复共轭
  • $a$: 尺度 scale
    • 当 $a$ 小时，小波被压短 -> 适合看高频、突变、瞬态
    • 当 $a$ 大时，小波被拉长 -> 适合看低频、慢变化
  • $b$: 平移位置 time shift
  • $W_x(a,b)$: 在尺度 $a$、时间 $b$ 附近的小波系数

每移动到一个位置，就问 这个位置附近，信号像不像这个小波？

  • 像，系数大
  • 不像，系数小
小波位置 b：
|---ψ---|                       

信号：
--------------------------------

移动后：
      |---ψ---|                 

再移动：
             |---ψ---|

CWT 画出来一般叫：scalogram，小波尺度图 / 小波时频图

代码 

"""
CWT：连续小波变换
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

教学要点：
  1. CWT 用于时间-频率分析，和 STFT 类似
  2. STFT 使用固定长度窗口
  3. CWT 使用可以缩放的小波
  4. 小尺度 scale 小：对应高频，时间定位更准
  5. 大尺度 scale 大：对应低频，频率分辨率更好
  6. 本例使用 Morlet 小波分析一个频率随时间变化的 sin 信号
"""

import numpy as np
import matplotlib.pyplot as plt

# ── 信号参数 ──
fs = 200                    # 采样频率 200 Hz
T = 2.0                     # 总时长 2 秒
t = np.arange(0, T, 1 / fs)

# ── 构造频率随时间变化的信号 ──
# 0~1s: 5Hz
# 1~2s: 20Hz
x = np.zeros_like(t)
x[t < 1.0] = np.sin(2 * np.pi * 5 * t[t < 1.0])
x[t >= 1.0] = np.sin(2 * np.pi * 20 * t[t >= 1.0])

# ── Morlet 小波参数 ──
w0 = 6.0                    # Morlet 小波中心角频率，常用 6

def morlet_wavelet(tau, w0=6.0):
    """
    Morlet 母小波：
        ψ(t) = π^(-1/4) exp(j w0 t) exp(-t^2/2)
    """
    return (np.pi ** (-0.25)) * np.exp(1j * w0 * tau) * np.exp(-(tau ** 2) / 2)


def cwt_morlet(x, fs, freqs, w0=6.0):
    """
    用 Morlet 小波手写 CWT。

    参数:
        x: 输入离散信号
        fs: 采样频率
        freqs: 希望分析的频率数组，单位 Hz
        w0: Morlet 小波中心角频率

    返回:
        cwt_matrix: shape = (len(freqs), len(x))
    """
    x = np.asarray(x)
    N = len(x)

    cwt_matrix = []

    for f in freqs:
        # Morlet 小波中，尺度 scale 和频率近似关系：
        # f ≈ w0 * fs / (2π * scale)
        scale = w0 * fs / (2 * np.pi * f)

        # 小波长度取 ±4 个 scale
        half_width = int(4 * scale)

        # 避免小波长度过长，超过原始信号太多
        half_width = min(half_width, N // 2)

        tau = np.arange(-half_width, half_width + 1)

        # 缩放后的小波
        wavelet = (1 / np.sqrt(scale)) * morlet_wavelet(tau / scale, w0=w0)

        # full 卷积，然后手动截取中间 N 个点
        full = np.convolve(x, np.conj(wavelet[::-1]), mode='full')

        start = (len(full) - N) // 2
        coef = full[start:start + N]

        cwt_matrix.append(coef)

    return np.vstack(cwt_matrix)


# ── CWT 频率范围 ──
freqs = np.linspace(1, 40, 100)

# ── 计算 CWT ──
CWT = cwt_morlet(x, fs, freqs, w0=w0)
power = np.abs(CWT)

# 归一化，方便显示
power = power / np.max(power)

# ── 画图 ──
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4))

# 左图：时域信号
ax1.plot(t, x, linewidth=1.2)
ax1.axvline(1.0, linestyle='--', alpha=0.7, label='frequency changes at 1s')
ax1.set_xlabel('t (s)')
ax1.set_ylabel('x(t)')
ax1.set_title('Time-domain signal: 5Hz then 20Hz')
ax1.grid(True, alpha=0.3)
ax1.legend(loc='upper right')

# 右图：CWT 时频图
im = ax2.imshow(
    power,
    origin='lower',
    aspect='auto',
    extent=[t[0], t[-1], freqs[0], freqs[-1]]
)

ax2.set_xlabel('time (s)')
ax2.set_ylabel('frequency (Hz)')
ax2.set_title('CWT scalogram using Morlet wavelet')
ax2.set_ylim(0, 40)

fig.colorbar(im, ax=ax2, label='normalized |CWT|')

fig.suptitle('CWT: multi-scale time-frequency analysis',
             fontsize=14, fontweight='bold')
plt.tight_layout(rect=[0, 0, 1, 0.92])
plt.savefig('cwt_sin.png', dpi=150, bbox_inches='tight')
# plt.show()

Morlet 小波

Morlet 小波可以理解成：一个被“高斯包络”包起来的复指数波 (中间在振荡，两边逐渐衰减到 0 的一小段波)

$$ \psi(t)=\pi^{-1/4}e^{j\omega_0t}e^{-t^2/2} $$
  • $e^{j\omega_0t}$: 负责振荡，像正弦/复指数，能检测频率
  • $e^{-t^2/2}$: 负责局部化，让波形两边衰减到 0，能定位时间
小波 特点 更适合
Morlet 像一小段正弦，频率解释强 时频分析、振荡信号
Haar 像阶跃，特别简单 突变、边缘、分段常数
Mexican Hat 像二阶导数，高低起伏明显 峰值检测、脉冲结构
Daubechies 紧支撑、正交性好 压缩、去噪、多分辨率分析

1. 用积分表示该段信号与小波“相似”

如果两个函数形状很像，比如都是正的地方一起正，负的地方一起负：

$$ x(t)\psi(t) $$

大部分地方都是正数，积分后就会很大。

信号:    + + + - - -
小波:    + + + - - -
相乘:    + + + + + +    → 累加很大

如果两个函数不像，比如一个正的时候另一个负：

信号:    + + + - - -
小波:    - - - + + +
相乘:    - - - - - -    → 累加很负

如果完全对不上，有的地方正、有的地方负，最后会互相抵消：

信号:    + + - + - -
小波:    + - + - + -
相乘:    + - - - - +    → 正负抵消，接近 0

2. 为什么要做成复数

复数 Morlet 不仅能看“有没有这个频率”，还能保留相位信息

let 小波：

$$ \begin{align} \psi(t)&=\pi^{-1/4}e^{j\omega_0t}e^{-t^2/2} \\ &=\pi^{-1/4}e^{-t^2/2} \left[ \cos(\omega_0t)+j\sin(\omega_0t) \right] \end{align} $$

所以它自然有：

$$ \text{real part}=\pi^{-1/4}e^{-t^2/2}\cos(\omega_0t) \\ \text{imaginary part}=\pi^{-1/4}e^{-t^2/2}\sin(\omega_0t) $$

如果只用实部 $\cos(\omega_0t)e^{-t^2/2}$, 就只能像一个局部 cos 模板 但信号可能刚好更像 sin, 而不是 cos -> 会因为相位差导致匹配结果不稳定

而如果有相位信息, 就可以用两个相差 $90^\circ$ 的模板去匹配信号 -> 就可以不太受相位影响

代码 

"""
画 Morlet 小波
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Morlet 小波：
    ψ(t) = π^(-1/4) exp(j w0 t) exp(-t²/2)

含义：
    exp(j w0 t) 负责振荡
    exp(-t²/2) 负责让两边衰减
"""

import numpy as np
import matplotlib.pyplot as plt

# ── 参数 ──
w0 = 6.0                      # Morlet 小波中心角频率
t = np.linspace(-5, 5, 2000)  # 时间范围

# ── Morlet 小波 ──
morlet = (np.pi ** (-0.25)) * np.exp(1j * w0 * t) * np.exp(-(t ** 2) / 2)

# 实部、虚部、幅度
morlet_real = np.real(morlet)
morlet_imag = np.imag(morlet)
morlet_abs = np.abs(morlet)

# 高斯包络
envelope = (np.pi ** (-0.25)) * np.exp(-(t ** 2) / 2)

# ── 画图 ──
fig, ax = plt.subplots(figsize=(10, 4))

ax.plot(t, morlet_real, label='Real part')
ax.plot(t, morlet_imag, linestyle='--', label='Imaginary part')
ax.plot(t, envelope, linestyle=':', label='Gaussian envelope')
ax.plot(t, -envelope, linestyle=':')

ax.set_xlabel('t')
ax.set_ylabel('amplitude')
ax.set_title(r'Morlet wavelet: $\psi(t)=\pi^{-1/4}e^{j\omega_0t}e^{-t^2/2}$')
ax.grid(True, alpha=0.3)
ax.legend()

plt.tight_layout()
plt.savefig('morlet_wavelet.png', dpi=150, bbox_inches='tight')
# plt.show()
