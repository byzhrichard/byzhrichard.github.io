推荐系统

推荐系统 = 在**“用户 + 物品”**的巨大空间里，预测你最可能喜欢的东西

给定用户 $u$，在所有物品集合 $I$ 中，找一个排序：

$$ i_1, i_2, ..., i_k = \arg\max_{i \in I} P(i \mid u) $$

-> 输出 Top-K

输入：你最近看了
- AI视频
- 机器学习课程
- PyTorch教程

输出推荐：
- “Transformer讲解视频”
- “SNN神经网络论文解读”
- “深度学习课程”

1. 难点

  1. 信息爆炸 (用户:几亿, 物品:几千万～几十亿)
  2. 用户行为是稀疏的 (看过 100 个视频, 但平台有 1,000,000 个视频) [空的 ≠ 不喜欢，只是“没见过”]
  3. 反馈是隐含的 (从正样本 + 未知中推断偏好)

2. 主流方法

  • 协同过滤(CF): 相似用户喜欢相似物品
  • 矩阵压缩技术 CSR
  • 矩阵分解(MF/SVD):
    • 用户 = 一个向量
    • 物品 = 一个向量
    • -> 点积 = 匹配程度
  • 深度学习方法:
    • VAE
    • GCN

3. 核心指标

Recall@K: 召回率 (我喜欢的东西，有多少被推荐出来了？)

$$ \text{Recall@K} = \frac{\#\text{Hit items in Top-K}} {\#\text{Relevant items}} = \frac{\text{Top-K 推荐中命中的数量}} {\text{用户实际喜欢的数量}} $$
真实喜欢：  A B C D
推荐Top10： B E F C G H I

命中：B, C → 2个

NDCG@K: 排序质量指标 (推荐得好不好？好的是不是排在前面？)

$$ \text{折损累积增益:}\quad DCG@K = \sum_{i=1}^K \frac{rel_i}{\log_2(i+1)} \\ \text{理想折损累积增益:}\quad IDCG@K = \sum_{i=1}^K \frac{rel_i^*}{\log_2(i+1)} \\ \text{排序质量指标:}\quad NDCG = \frac{DCG}{IDCG} $$
真实喜欢：A B C D [不管先后]
推荐Top5：A C B E F [有顺序]

→ DCG@3 ≈ 1 + 0.63 + 0.5 = 2.13 (命中3个)
→ IDCG@3 = 1 + 0.63 + 0.5 = 2.13 (该真实喜欢情况下, 理想命中3个: A,B,C)
→ NDCG@3 = 1.0

真实喜欢：A B C D [不管先后]
推荐Top5：B A E F G [有顺序]

→ DCG@5 ≈ 1 + 0.63 = 1.63 (命中2个)
→ IDCG@5 = 1 + 0.63 + 0.5 + 0.43 = 2.56 (该真实喜欢情况下, 理想命中4个: A,)
→ NDCG@5 = 0.64

HitRate@K: 命中率 (有没有至少推荐对一个？)

$$ HitRate@K = \begin{cases} 1, & \text{Top-K 有至少一个正样本}\\ 0, & \text{否则} \end{cases} $$
真实喜欢：A B C
推荐：     X Y A Z

→ Hit = 1
指标 关注点 特点
Recall@K 找回多少 全局覆盖
NDCG@K 排序质量 最严格
HitRate@K 是否命中 最宽松

4. 弱泛化划分

弱泛化划分 = 用“用户过去的行为”预测“这个用户未来的行为”

错误做法: 随机划分（强泛化）

现实系统是：

  • 你今天看过什么
  • 明天推荐什么

不是：

  • 随机猜你喜欢什么

-> 先按timestamp来排序, 再分出 80% train, 20% test

推荐系统数据集

MovieLens 1M

1. 数据集介绍

  • 官网: https://grouplens.org/datasets/movielens/1m/

  • 下载链接: https://files.grouplens.org/datasets/movielens/ml-1m.zip

  • 数据集全称: MovieLens 1M Dataset

  • 数据集中文全称: MovieLens 百万级电影评分数据集

  • 样本规模（train / test 数量）

    • 用户数：6,040
    • 电影数：3,952
    • 评分数：1,000,209（约 100 万条）

  • 类别数量: 电影类型（Genres）约 18 类（如 Action / Comedy / Drama 等）

2. 任务定义

  • 任务类型：推荐系统（Top-K Recommendation）
  • 输入：(用户ID, 电影ID, 1~5评分, 时间戳)
  • 输出：Top-K推荐列表
  • 难点：
    • 数据极度稀疏（<5%）
    • 无显式负样本（未评分 ≠ 不喜欢）
    • 用户行为存在时间偏移

3. 主要领域用途

  • 协同过滤（CF）
  • 矩阵分解（MF / SVD）
  • 隐变量模型（VAE / Autoencoder）
  • 图神经网络推荐（GCN / GraphVAE）
  • 排序学习（Ranking / BPR）

4. 文件组成

users.dat:

UserID::Gender::Age::Occupation::Zip-code

movies.dat:

MovieID::Title::Genres

ratings.dat: 核心

UserID::MovieID::Rating::Timestamp

create_dataset.py 代码:

import os
import numpy as np
import pandas as pd
import torch
from scipy.sparse import csr_matrix


class ML1MDataset(torch.utils.data.Dataset):
    def __init__(self, root_path="ml-1m", implicit_threshold=4):
        self.root_path = root_path
        self.raw_path = os.path.join(root_path, "raw")

        self.ratings = self._load_ratings()
        self.ratings = self._preprocess(implicit_threshold)

        self.user2idx, self.item2idx = self._build_mapping()

        self.matrix = self._build_csr()

    # -------------------------
    # 1. load raw ratings
    # -------------------------
    def _load_ratings(self):
        file_path = os.path.join(self.raw_path, "ratings.dat")

        ratings = pd.read_csv(
            file_path,
            sep="::",
            engine="python",
            names=["user", "item", "rating", "timestamp"]
        )

        return ratings

    # -------------------------
    # 2. implicit feedback
    # -------------------------
    def _preprocess(self, threshold):
        ratings = self.ratings[self.ratings["rating"] >= threshold].copy()
        ratings = ratings.sort_values("timestamp")
        return ratings

    # -------------------------
    # 3. mapping
    # -------------------------
    def _build_mapping(self):
        user_ids = self.ratings["user"].unique()
        item_ids = self.ratings["item"].unique()

        user2idx = {int(u): i for i, u in enumerate(user_ids)}
        item2idx = {int(i): j for j, i in enumerate(item_ids)}

        return user2idx, item2idx

    # -------------------------
    # 4. build CSR matrix
    # -------------------------
    def _build_csr(self):
        df = self.ratings.copy()

        df["user"] = df["user"].map(self.user2idx)
        df["item"] = df["item"].map(self.item2idx)

        row = df["user"].values
        col = df["item"].values
        data = np.ones(len(df), dtype=np.float32)

        num_users = len(self.user2idx)
        num_items = len(self.item2idx)

        matrix = csr_matrix(
            (data, (row, col)),
            shape=(num_users, num_items)
        )

        return matrix

    # -------------------------
    # PyTorch API
    # -------------------------
    def __len__(self):
        return self.matrix.shape[0]

    def __getitem__(self, idx):
        """
        返回一个 user 的 interaction vector
        shape: [num_items]
        """
        vec = self.matrix[idx].toarray().squeeze()
        return torch.FloatTensor(vec)

    # -------------------------
    # optional helper
    # -------------------------
    def get_matrix(self):
        return self.matrix

    def get_mappings(self):
        return self.user2idx, self.item2idx

if __name__ == '__main__':
    dataset = ML1MDataset()
    print("Number of users:", len(dataset.user2idx))
    print("Number of items:", len(dataset.item2idx))
    print("Sample interaction vector shape:", dataset[0].shape)

MovieLens 100K

  • 数据集全称: MovieLens 100K Dataset

  • 数据集中文全称: MovieLens 十万级电影评分数据集

  • 样本规模（train / test 数量）

    • 用户数：943
    • 电影数：1,682
    • 评分数：100,000（约 10 万条）

  • 类别数量: 电影类型（Genres）19 类（如 Action / Comedy / Drama 等）

2. 任务定义

  • 任务类型：推荐系统（Top-K Recommendation）
  • 输入：(用户ID, 电影ID, 1~5评分, 时间戳)
  • 输出：Top-K推荐列表
  • 难点：
    • 数据极度稀疏（<6.3%）
    • 无显式负样本（未评分 ≠ 不喜欢）
    • 用户行为存在时间偏移

  • 协同过滤（CF）

  • 隐变量模型（VAE / Autoencoder）
  • 图神经网络推荐（GCN / GraphVAE）
  • 排序学习（Ranking / BPR）

4. 文件组成

u.data:

user_id \t item_id \t rating \t timestamp

u.item:

movie_id \t title \t release_date \t ... \t genre_0 \t ... \t genre_18

u.user:

user_id \t age \t gender \t occupation \t zip_code

import os
import numpy as np
import pandas as pd
import torch
from scipy.sparse import csr_matrix


class ML100KDataset(torch.utils.data.Dataset):
    def __init__(self, root_path="ml-100k", implicit_threshold=4):
        self.root_path = root_path
        self.raw_path = os.path.join(root_path, "raw")

        self.ratings = self._load_ratings()
        self.ratings = self._preprocess(implicit_threshold)

        self.user2idx, self.item2idx = self._build_mapping()

        self.matrix = self._build_csr()

    # -------------------------
    # 1. load raw ratings
    # -------------------------
    def _load_ratings(self):
        file_path = os.path.join(self.raw_path, "u.data")

        ratings = pd.read_csv(
            file_path,
            sep="\t",
            names=["user", "item", "rating", "timestamp"]
        )

        return ratings

    # -------------------------
    # 2. implicit feedback
    # -------------------------
    def _preprocess(self, threshold):
        ratings = self.ratings[self.ratings["rating"] >= threshold].copy()
        ratings = ratings.sort_values("timestamp")
        return ratings

    # -------------------------
    # 3. mapping
    # -------------------------
    def _build_mapping(self):
        user_ids = self.ratings["user"].unique()
        item_ids = self.ratings["item"].unique()

        user2idx = {int(u): i for i, u in enumerate(user_ids)}
        item2idx = {int(i): j for j, i in enumerate(item_ids)}

        return user2idx, item2idx

    # -------------------------
    # 4. build CSR matrix
    # -------------------------
    def _build_csr(self):

        df["user"] = df["user"].map(self.user2idx)
        df["item"] = df["item"].map(self.item2idx)

        row = df["user"].values
        col = df["item"].values
        data = np.ones(len(df), dtype=np.float32)

        num_users = len(self.user2idx)
        num_items = len(self.item2idx)

        matrix = csr_matrix(
            (data, (row, col)),
            shape=(num_users, num_items)
        )

        return matrix

    # -------------------------
    # PyTorch API
    # -------------------------
    def __len__(self):
        return self.matrix.shape[0]

    def __getitem__(self, idx):
        vec = self.matrix[idx].toarray().squeeze()
        return torch.FloatTensor(vec)

    # -------------------------
    # helper
    # -------------------------
    def get_matrix(self):
        return self.matrix

    def get_mappings(self):
        return self.user2idx, self.item2idx


if __name__ == "__main__":
    dataset = ML100KDataset(root_path="ml-100k")

    print("Users:", len(dataset.user2idx))
    print("Items:", len(dataset.item2idx))
    print("Sample shape:", dataset[0].shape)
