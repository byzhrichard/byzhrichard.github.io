工业诊断数据集 (industrial diagnosis)

CWRU数据集

1. 数据集介绍

  • 官网:

  • 下载链接: https://engineering.case.edu/bearingdatacenter/download-data-file

  • 数据集全称: Case Western Reserve University Bearing Fault Dataset

  • 数据集中文全称: 凯斯西储大学轴承故障数据集

  • 样本规模（train / test 数量）

    • 官方发布的是若干条连续采集的 MATLAB 振动信号记录，并未提供固定的训练集和测试集
    • 使用时通常通过固定长度滑动窗口，将连续信号切分为多个样本
    • 最终样本数量取决于所选原始文件、采样频率、信号通道、窗口长度、滑动步长和数据划分比例

  • 类别数量

    • 官方没有规定统一的机器学习分类标签
    • 从轴承基本状态来看，可分为正常、内圈故障、滚动体故障和外圈故障四种
    • 若进一步区分故障尺寸、轴承位置、外圈故障方位和负载工况，则可以构造更多类别

CWRU 数据集是机械故障诊断领域常用的公开轴承振动数据集。实验平台主要由一台 2 hp 电机、转矩传感器、测功机和控制设备组成。加速度传感器被安装在电机驱动端和风扇端附近，用于采集不同轴承状态下的振动信号。

数据集中的轴承故障通过电火花加工，即 Electrical Discharge Machining，EDM，人为设置。故障位置包括：

  • 内圈：Inner Race，IR；
  • 滚动体或滚珠：Ball，B；
  • 外圈：Outer Race，OR。

官方设置的单点故障直径包括 0.007、0.014、0.021、0.028 和 0.040 inch，但不同轴承端、故障位置和采样设置下可获得的数据并不完全相同。

实验在不同电机负载下进行。常见负载包括 0、1、2 和 3 hp，对应的电机转速大约从 1797 rpm 下降至 1730 rpm 左右。不同负载和转速条件使该数据集能够用于研究跨工况故障诊断问题。

2. 任务定义

  • 任务类型：多分类（Multi-class Classification）

  • 输入：

    • 从轴承加速度传感器采集的一维振动时间序列
    • 常用输入为驱动端振动信号 DE_time
    • 也可以使用风扇端信号 FE_time、底座信号 BA_time，或者将多个测点组成多通道输入
    • 原始连续信号通常需要经过滑动窗口切分，形成固定长度样本

  • 输出：自行处理

  • 难点：

    • 不同负载和转速会改变振动信号的时域幅值与频谱分布
    • 相同故障位置在不同故障尺寸下可能具有不同的振动响应
    • 不同故障类型之间可能存在相似的周期性冲击特征
    • 轻微故障与正常状态之间的差异较弱
    • 外圈故障相对于负载区的位置会显著影响振动响应
    • 12 kHz 和 48 kHz 数据具有不同的采样率，不能直接作为相同格式的数据混合使用
    • 不同原始记录的长度可能不同，滑动窗口后容易产生类别不平衡
    • 实验室人工制造的单点故障与真实工业场景中的自然退化、复合故障仍存在差异
    • 若先对原始信号进行高度重叠的滑窗，再随机划分训练集和测试集，相邻窗口可能同时出现在训练集与测试集中，导致数据泄漏
    • 跨负载、跨转速或跨传感器位置测试通常比同工况随机划分更困难，也更能反映模型的泛
    • 能力

3. 主要领域用途

CWRU 数据集主要用于以下研究方向：

  • 滚动轴承故障检测
  • 轴承故障类型识别
  • 故障严重程度识别
  • 旋转机械状态监测
  • 电机健康状态评估
  • 工业设备预测性维护
  • 一维振动信号分类
  • 时域、频域和时频域特征提取
  • 传统机器学习故障诊断
  • 卷积神经网络故障诊断
  • 循环神经网络和 Transformer 时序建模
  • 脉冲神经网络振动信号分类
  • 小样本与少样本故障诊断
  • 跨负载、跨转速和跨设备迁移学习
  • 领域自适应与领域泛化
  • 噪声环境下的鲁棒故障诊断
  • 模型压缩、轻量化和边缘设备部署
  • 可解释机械故障诊断
  • 故障诊断算法的性能验证与对比

4. 文件组成

CWRU 官方数据主要以 MATLAB 的 .mat 格式提供，可分为以下几个部分：

CWRU/
├── Normal Baseline Data/
│   └── 正常轴承在不同负载下的振动数据
│
├── 12k Drive End Bearing Fault Data/
│   └── 驱动端轴承故障的12 kHz采样数据
│
├── 48k Drive End Bearing Fault Data/
│   └── 驱动端轴承故障的48 kHz采样数据
│
└── 12k Fan End Bearing Fault Data/
    └── 风扇端轴承故障的12 kHz采样数据
