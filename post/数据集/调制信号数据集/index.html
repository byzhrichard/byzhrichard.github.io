调制信号数据集

RML2016.10a数据集

1. 数据集介绍

  • 官网: https://www.deepsig.ai/datasets/

  • 下载链接:

  • 数据集全称: Radio Machine Learning 2016.10a

  • 数据集中文全称: RadioML 2016.10a 无线电调制识别数据集

  • 样本规模（train / test 数量）

    • 总样本数为 220,000
    • 官方没有固定划分训练集和测试集
    • 每种调制方式在每个信噪比下包含 1,000 个样本

  • 类别数量: 11 类

RML2016.10a 是使用 GNU Radio 生成的合成无线通信信号数据集，包含 8 种数字调制和 3 种模拟调制。每个样本由连续 128 个复数基带采样点组成，实部和虚部分别保存为 I、Q 两个通道。数据生成过程考虑了高斯噪声、频率偏移、采样率偏移、多径传播和衰落等信道影响。

11 种调制方式如下：

类型 调制方式
数字调制 BPSK、QPSK、8PSK、PAM4、16QAM、64QAM、CPFSK、GFSK
模拟调制 AM-DSB、AM-SSB、WBFM

2. 任务定义

  • 任务类型：多分类（Multi-class Classification）
  • 输入x：一段长度为 128 的复数基带 I/Q 信号
  • 输出y：输入信号所属的调制类别
  • 难点：
    • 低信噪比下信号特征容易被噪声淹没
    • QPSK 与 8PSK、16QAM 与 64QAM 等调制方式较为相似
    • AM-DSB 与 WBFM 等模拟信号的内容变化会影响识别
    • 频率偏移、多径和采样率偏差会造成信号畸变
    • 单个样本只有 128 个采样点，可利用的时间信息有限

3. 主要领域用途

  • 自动调制识别
  • 无线电信号分类
  • 频谱感知与认知无线电
  • 通信信号侦察
  • 复杂电磁环境下的信号识别
  • CNN、RNN、Transformer 和 SNN 的一维信号分类
  • 低信噪比鲁棒识别研究

4. 文件组成

解压后通常得到：

RML2016.10a/
└── RML2016.10a_dict.pkl

数据以 Python Pickle 字典形式保存，字典的键为：

(modulation, snr)

例如：

("BPSK", -20)
("QPSK", 0)
("WBFM", 18)

每个键对应一个 NumPy 数组：

[1000, 2, 128]

含义分别为：

1000  → 当前调制方式和SNR组合下的样本数
2     → I、Q两个通道
128   → 每个样本的时间采样点数

整体数据结构可以理解为：

RML2016.10a_dict.pkl
├── ('BPSK', -20) → [1000, 2, 128]
├── ('BPSK', -18) → [1000, 2, 128]
├── ...
├── ('WBFM', 16)  → [1000, 2, 128]
└── ('WBFM', 18)  → [1000, 2, 128]

RML2016.10b数据集

1. 数据集介绍

  • 官网: https://www.deepsig.ai/datasets/

  • 下载链接:

  • 数据集全称: Radio Machine Learning 2016.10b

  • 数据集中文全称: RadioML 2016.10b 无线电调制识别数据集

  • 样本规模（train / test 数量）

    • 总样本数约为 1,200,000
    • 官方没有固定的训练集和测试集
    • 每种调制方式在每个信噪比下通常包含 6,000 个样本

  • 类别数量: 10 类

RML2016.10b 可以看作规模更大的 RadioML 2016 数据版本。它与 RML2016.10a 使用相似的 I/Q 信号表示和信噪比设置，但去除了 AM-SSB 类别，并显著增加了每种调制方式和信噪比组合下的样本数量。

10 种调制方式如下：

类型 调制方式
数字调制 BPSK、QPSK、8PSK、PAM4、16QAM、64QAM、CPFSK、GFSK
模拟调制 AM-DSB、WBFM

2. 任务定义

  • 任务类型：多分类（Multi-class Classification）
  • 输入x：长度为 128 的复数基带 I/Q 信号
  • 输出y：输入无线电信号所属的 10 种调制类别之一
  • 难点：
    • 低信噪比条件下不同调制信号的可分性较差
    • 高阶调制之间容易发生混淆
    • 信道衰落、噪声和频率偏移会破坏星座结构
    • 数据量较大，对内存、数据加载速度和训练效率要求较高
    • 若随机按样本划分数据，需要保证不同调制类别和 SNR 等级分布均衡

3. 主要领域用途

  • 大规模自动调制识别
  • 无线通信信号分类
  • 认知无线电和动态频谱感知
  • 通信侦察与信号监测
  • 深度神经网络性能对比
  • 不同信噪比条件下的鲁棒性分析
  • 轻量化网络与实时信号识别研究

4. 文件组成

数据文件通常为：

RML2016.10b/
└── RML2016.10b.dat

虽然文件后缀为 .dat，其内部一般仍采用 Python Pickle 格式保存，可以使用 pickle.load() 读取。

字典结构与 RML2016.10a 类似：

(modulation, snr) -> samples

每个键对应的数据形状通常为：

[6000, 2, 128]

整体结构可以表示为：

RML2016.10b.dat
├── ('BPSK', -20) → [6000, 2, 128]
├── ('BPSK', -18) → [6000, 2, 128]
├── ...
├── ('WBFM', 16)  → [6000, 2, 128]
└── ('WBFM', 18)  → [6000, 2, 128]
