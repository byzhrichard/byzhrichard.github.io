📖参考资料

Surrogate Gradient Learning in Spiking Neural Networks.

🌱背景

在 CNN 或 RNN 中，神经元之间传递的是连续的激活值（例如 0.85 或 -0.2） 而在 SNN 中，神经元之间只传递脉冲Spike:

  • 用 0 或 1 表示
  • 信息不再由数值的大小决定，而是由 脉冲发放的时间Timing 或 脉冲发放的频率Rate 决定

ANN： 像是在打长途电话，你对着麦克风喊出的音量大小代表信息 SNN： 像是在发莫尔斯电码，只有“点”和“划”，信息隐藏在点击的节奏中

⚙️编码

SNN 不能直接处理像 0.75 这样的像素值，必须先“翻译”成脉冲:

  1. 频率编码 (Rate Coding)： 数值越大，单位时间内发的脉冲越多（最常用，最简单）
  2. 时间编码 (Temporal Coding)： 数值越大，脉冲发出的时间越早
  3. 直接驱动： 现在很多主流框架直接把第一层作为编码层，让 CNN 的输出直接进入 LIF 神经元

⚙️神经元模型 - 从 ReLU 到 LIF

LIF: Leaky Integrate-and-Fire: 漏电积分发放模型

  1. 积分 (Integrate)： 输入脉冲进入木桶，水面 (膜电位 $V$) 上升
  2. 漏电 (Leaky)： 如果一段时间没动静，水会从小孔慢慢流走（电位随时间衰减）
  3. 发放 (Fire)： 当水面超过阈值 $V_{th}$，神经元瞬间喷出一个脉冲，同时水面立刻清空（重置电位）

-> 达到阈值就发射并归零”的离散跳变机制

$$ V_t = \tau V_{t-1} + I_t - V_{th} S_{t-1} $$
  • $I_t$: 当前时间步输入到神经元的“刺激” (连续值 - 信息输入) (来自 Conv / Linear)
  • $S_{t-1}$: 上一时间步时, 神经元是否发放脉冲 (0或1) (离散值 - 信息输出) (来自 LIFNode 的输出)

⚙️如何训练 - Surrogate Gradient

SNN 的脉冲函数（Step Function）在阈值处是不可导的（导数为无穷大，其余地方为0），这意味着 Backprop 炸了

在传统的神经网络里，我们通过链式法则计算梯度：$\frac{\partial Loss}{\partial W} = \frac{\partial Loss}{\partial Output} \cdot \frac{\partial Output}{\partial Input} \cdot \frac{\partial Input}{\partial W}$

在 SNN 的神经元里，$Output$ 是脉冲 $S$，它与电压 $V$ 的关系是一个 阶跃函数:

  • 在绝大多数地方，导数是 0 -> 导致“梯度消失”，权重根本没法更新。
  • 在 $V=V_{th}$ 的那一点，导数是 无穷大 -> 导致“梯度爆炸”

-> 传统的微积分在这里失效了，你没法直接告诉 $W$ 应该往左还是往右挪

代理梯度(Surrogate Gradient):

  • 在前向传播时用离散脉冲
  • 在反向传播时，假装那个阶跃函数是一个平滑的 Sigmoid 或 ArcTan

-> 就可以直接用 PyTorch/TensorFlow 直接跑 Backprop 了

前向传播 (Forward)： 依然严格遵守阶跃函数。电压够了就发 1，不够就发 0。这样可以保持 SNN 的生物特性（稀疏性、低功耗）

反向传播 (Backward)： 当我们要计算 $\frac{\partial S}{\partial V}$ 时，我们假装前向传播用的不是阶跃函数，而是一个平滑的函数（比如 Sigmoid 或 ArcTan）

-> 虽然反向传播用的是假的导数，但只要梯度下降的方向是对的，就能训好 （如果 $V$ 接近阈值但没超过，代理梯度会给出一个正值，告诉网络：“嘿，再增加一点点电压，这里就能出脉冲了！”）

建议代码实现:

class SurrogateSpike(torch.autograd.Function):
    @staticmethod
    def forward(ctx, input):
        # 前向传播：标准的阶跃函数（离散脉冲）
        ctx.save_for_backward(input) # 保存电压用于反向计算
        return (input >= 1.0).float() 

    @staticmethod
    def backward(ctx, grad_output):
        # 反向传播：编一个平滑的导数（比如 Sigmoid 的导数）
        input, = ctx.saved_tensors
        # 这里的计算完全不看阶跃函数，而是用一个平滑函数代替
        grad_input = grad_output * (某个平滑函数的导数公式) 
        return grad_input

👍优点

  1. 极低功耗: SNN 是**“事件驱动”**的。如果没有脉冲，神经元就不计算 (在专门的类脑芯片 [如 Intel Loihi 或华为天机] 上，功耗比 GPU 低几个数量级)
  2. 时间动力学: SNN 天然擅长处理语音、视频等时序强相关信号（比 RNN 更省电）
  3. 高动态范围: 适合处理来自 **DVS（事件相机）**的数据

👎缺点

  1. 高推理延迟:
    • 设置 $T=100$ 就意味着为了得到一个分类结果，网络必须重复计算 100 次
    • 在传统 GPU 上，这会导致推理速度比普通 CNN 慢得多，无法体现出“快”的优势 (如果没有专门的类脑芯片，SNN 反而更费电、更慢)
  2. 训练难度大且耗时:
    • SNN 的训练本质上是在训练一个沿着时间轴展开的 RNN
    • SNN 对超参数（如阈值 $V_{th}$、衰减系数 $\tau$）非常敏感，调参比 ANN 困难得多
  3. SNN 的精度通常仍略低于同规模的顶级 CNN 或 Transformer (信息在“连续值”转“0/1 脉冲”的过程中存在量化损失，且深层 SNN 的梯度流非常不稳定)

补充: detach_reset

detach_reset默认为False

SpikingJelly源码里神经元都会把 detach_reset 传给训练时的 autograd 函数，用来控制 reset 过程的梯度行为

  • detach_reset=False：reset 过程也参与反向传播，梯度会考虑“发放脉冲导致膜电位重置”这件事
  • detach_reset=True：重置电压时，把 spike 从计算图里断开，不让 reset 对 spike 的梯度反传产生影响
$$ v_{next} = v_{reset} * spike + v * (1 - spike) $$
  • detach_reset=False -> 反传路径与公式一样
  • detach_reset=True -> $v_{reset}$ 这条路径不再通过 spike 反传梯度

1. detach_reset=False

优点：

  • 梯度更完整
  • 理论上更符合 BPTT 中的时序依赖
  • 小模型、短时间步时可能效果更细腻

缺点：

  • reset 带来的梯度路径会更复杂；
  • 时间步长较长时，梯度更容易不稳定；
  • 有时训练会更敏感。

2. detach_reset=True

优点：

  • 训练更稳
  • 减少 reset 对梯度的干扰
  • 在较深 SNN、较长时间步、DVS 这类时序任务中经常更好调

缺点：

  • 梯度近似更粗
  • 理论上忽略了 reset 对后续膜电位的部分影响
  • 不一定总是提升精度

1. LeNet 

import torch
import torch.nn as nn
from spikingjelly.activation_based import neuron, functional, surrogate, layer

# ==========================================
# ANN 的 LeNet
# ==========================================
class ClassicLeNet(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()
        self.feature_extractor = nn.Sequential(
            nn.Conv2d(1, 6, kernel_size=5, padding=2),  # 32x32 -> 32x32
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 32x32 -> 16x16

            nn.Conv2d(6, 16, kernel_size=5),  # 16x16 -> 12x12
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 12x12 -> 6x6
        )
        self.classifier = nn.Sequential(
            nn.Flatten(),
            nn.Linear(16 * 6 * 6, 120),
            nn.ReLU(),
            nn.Linear(120, 84),
            nn.ReLU(),
            nn.Linear(84, num_classes)
        )

    def forward(self, x):
        x = self.feature_extractor(x)
        logits = self.classifier(x)
        return logits


# ==========================================
# SNN 的 LeNet
# ==========================================
class SpikingLeNet(nn.Module):
    def __init__(self, num_classes=10, T=8):
        super().__init__()
        self.T = T  # 时间步长

        # 使用 spikingjelly.activation_based.layer 自动处理 (T, N, C, H, W)
        self.feature_extractor = nn.Sequential(
            layer.Conv2d(1, 6, kernel_size=5, padding=2),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),  # 替代 ReLU
            layer.MaxPool2d(2, 2),

            layer.Conv2d(6, 16, kernel_size=5),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),
        )

        self.classifier = nn.Sequential(
            layer.Flatten(),
            layer.Linear(16 * 6 * 6, 120),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.Linear(120, 84),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.Linear(84, num_classes),
            # 注意：最后一层通常输出膜电位或脉冲率，这里接 Linear 得到 Logits
        )

        functional.set_step_mode(self, step_mode='m')

    def forward(self, x):
        # 1. 将输入图片在时间轴上复制 T 份: (N, C, H, W) -> (T, N, C, H, W)
        x_seq = x.unsqueeze(0).repeat(self.T, 1, 1, 1, 1)

        # 2. 脉冲神经元是有状态的，计算前需重置膜电位
        functional.reset_net(self)

        # 3. 运行网络
        # SpikingJelly 的层会自动循环 T 个时间步
        x_seq = self.feature_extractor(x_seq)
        x_seq = self.classifier(x_seq)

        # 4. 输出层处理：计算 T 个时间步的脉冲平均值作为最终 Logits
        # x_seq 形状是 (T, N, num_classes)
        logits = x_seq.mean(0)
        return logits


if __name__ == '__main__':
    N, C, H, W = 4, 1, 32, 32  # 随机生成 Batch=4, 通道=1, 32x32 的数据
    test_input = torch.randn(N, C, H, W)

    print(f"--- 测试开始 ---")
    print(f"输入形状: {test_input.shape}")

    # 测试 ANN
    ann_model = ClassicLeNet(num_classes=10)
    ann_output = ann_model(test_input)
    print(f"\n[ANN 版本]")
    print(f"输出形状: {ann_output.shape}")  # 预期 (4, 10)
    print(f"输出 Logits 示例 (前2个):\n{ann_output[:2, :5]}")

    # 测试 SNN
    snn_model = SpikingLeNet(num_classes=10, T=8)
    snn_output = snn_model(test_input)
    print(f"\n[SNN 版本]")
    print(f"输出形状: {snn_output.shape}")  # 预期 (4, 10)
    print(f"输出 Logits 示例 (前2个):\n{snn_output[:2, :5]}")

2. AlexNet 

import torch
import torch.nn as nn
from spikingjelly.activation_based import neuron, functional, surrogate, layer


# ==========================================
# ANN 的 AlexNet（适配 32x32）
# ==========================================
class ClassicAlexNet(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()

        self.feature_extractor = nn.Sequential(
            nn.Conv2d(1, 64, kernel_size=3, stride=1, padding=1),  # 32x32
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 16x16

            nn.Conv2d(64, 192, kernel_size=3, padding=1),  # 16x16
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 8x8

            nn.Conv2d(192, 384, kernel_size=3, padding=1),  # 8x8
            nn.ReLU(),

            nn.Conv2d(384, 256, kernel_size=3, padding=1),
            nn.ReLU(),

            nn.Conv2d(256, 256, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 4x4
        )

        self.classifier = nn.Sequential(
            nn.Flatten(),
            nn.Linear(256 * 4 * 4, 1024),
            nn.ReLU(),
            nn.Dropout(0.5),

            nn.Linear(1024, 512),
            nn.ReLU(),
            nn.Dropout(0.5),

            nn.Linear(512, num_classes)
        )

    def forward(self, x):
        x = self.feature_extractor(x)
        logits = self.classifier(x)
        return logits


# ==========================================
# SNN 的 AlexNet
# ==========================================
class SpikingAlexNet(nn.Module):
    def __init__(self, num_classes=10, T=8):
        super().__init__()
        self.T = T

        self.feature_extractor = nn.Sequential(
            layer.Conv2d(1, 64, kernel_size=3, stride=1, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),

            layer.Conv2d(64, 192, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),

            layer.Conv2d(192, 384, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),

            layer.Conv2d(384, 256, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),

            layer.Conv2d(256, 256, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),
        )

        self.classifier = nn.Sequential(
            layer.Flatten(),

            layer.Linear(256 * 4 * 4, 1024),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),

            layer.Linear(1024, 512),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),

            layer.Linear(512, num_classes),
        )

        # 多步模式
        functional.set_step_mode(self, step_mode='m')

    def forward(self, x):
        # (N, C, H, W) -> (T, N, C, H, W)
        x_seq = x.unsqueeze(0).repeat(self.T, 1, 1, 1, 1)

        # reset membrane
        functional.reset_net(self)

        # forward
        x_seq = self.feature_extractor(x_seq)
        x_seq = self.classifier(x_seq)

        # 时间平均
        logits = x_seq.mean(0)
        return logits


# ==========================================
# 测试
# ==========================================
if __name__ == '__main__':
    N, C, H, W = 4, 1, 32, 32
    test_input = torch.randn(N, C, H, W)

    print(f"--- AlexNet 测试开始 ---")
    print(f"输入形状: {test_input.shape}")

    # ANN
    ann_model = ClassicAlexNet(num_classes=10)
    ann_output = ann_model(test_input)
    print(f"\n[ANN 版本]")
    print(f"输出形状: {ann_output.shape}")
    print(f"输出示例:\n{ann_output[:2, :5]}")

    # SNN
    snn_model = SpikingAlexNet(num_classes=10, T=8)
    snn_output = snn_model(test_input)
    print(f"\n[SNN 版本]")
    print(f"输出形状: {snn_output.shape}")
    print(f"输出示例:\n{snn_output[:2, :5]}")

3. VGG11 

import torch
import torch.nn as nn
from spikingjelly.activation_based import neuron, functional, surrogate, layer


# ==========================================
# ANN 的 VGG11（适配 32x32）
# ==========================================
class ClassicVGG11(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()

        self.features = nn.Sequential(
            # Block 1
            nn.Conv2d(1, 64, kernel_size=3, padding=1),  # 32x32
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 16x16

            # Block 2
            nn.Conv2d(64, 128, kernel_size=3, padding=1),  # 16x16
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 8x8

            # Block 3
            nn.Conv2d(128, 256, kernel_size=3, padding=1),  # 8x8
            nn.ReLU(),
            nn.Conv2d(256, 256, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 4x4

            # Block 4
            nn.Conv2d(256, 512, kernel_size=3, padding=1),  # 4x4
            nn.ReLU(),
            nn.Conv2d(512, 512, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 2x2

            # Block 5
            nn.Conv2d(512, 512, kernel_size=3, padding=1),  # 2x2
            nn.ReLU(),
            nn.Conv2d(512, 512, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 1x1
        )

        self.classifier = nn.Sequential(
            nn.Flatten(),
            nn.Linear(512 * 1 * 1, 512),
            nn.ReLU(),
            nn.Dropout(0.5),

            nn.Linear(512, 256),
            nn.ReLU(),
            nn.Dropout(0.5),

            nn.Linear(256, num_classes)
        )

    def forward(self, x):
        x = self.features(x)
        logits = self.classifier(x)
        return logits


# ==========================================
# SNN 的 VGG11
# ==========================================
class SpikingVGG11(nn.Module):
    def __init__(self, num_classes=10, T=8):
        super().__init__()
        self.T = T

        self.features = nn.Sequential(
            # Block 1
            layer.Conv2d(1, 64, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),

            # Block 2
            layer.Conv2d(64, 128, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),

            # Block 3
            layer.Conv2d(128, 256, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.Conv2d(256, 256, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),

            # Block 4
            layer.Conv2d(256, 512, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.Conv2d(512, 512, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),

            # Block 5
            layer.Conv2d(512, 512, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.Conv2d(512, 512, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),
        )

        self.classifier = nn.Sequential(
            layer.Flatten(),

            layer.Linear(512, 512),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),

            layer.Linear(512, 256),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),

            layer.Linear(256, num_classes),
        )

        functional.set_step_mode(self, step_mode='m')

    def forward(self, x):
        # (N, C, H, W) -> (T, N, C, H, W)
        x_seq = x.unsqueeze(0).repeat(self.T, 1, 1, 1, 1)

        # reset 神经元状态
        functional.reset_net(self)

        # forward
        x_seq = self.features(x_seq)
        x_seq = self.classifier(x_seq)

        # 时间平均
        logits = x_seq.mean(0)
        return logits

import torch
import torch.nn as nn
from spikingjelly.activation_based import neuron, functional, surrogate, layer


# ==========================================
# ANN Inception Block
# ==========================================
class Inception(nn.Module):
    def __init__(self, in_ch, c1, c3r, c3, c5r, c5, pool):
        super().__init__()

        self.branch1 = nn.Sequential(
            nn.Conv2d(in_ch, c1, kernel_size=1),
            nn.ReLU()
        )

        self.branch2 = nn.Sequential(
            nn.Conv2d(in_ch, c3r, kernel_size=1),
            nn.ReLU(),
            nn.Conv2d(c3r, c3, kernel_size=3, padding=1),
            nn.ReLU()
        )

        self.branch3 = nn.Sequential(
            nn.Conv2d(in_ch, c5r, kernel_size=1),
            nn.ReLU(),
            nn.Conv2d(c5r, c5, kernel_size=3, padding=1),  # 用3x3代替5x5
            nn.ReLU()
        )

        self.branch4 = nn.Sequential(
            nn.MaxPool2d(3, stride=1, padding=1),
            nn.Conv2d(in_ch, pool, kernel_size=1),
            nn.ReLU()
        )

    def forward(self, x):
        return torch.cat([
            self.branch1(x),
            self.branch2(x),
            self.branch3(x),
            self.branch4(x)
        ], dim=1)


# ==========================================
# ANN GoogLeNet（适配32x32）
# ==========================================
class ClassicGoogLeNet(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()

        self.stem = nn.Sequential(
            nn.Conv2d(1, 64, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 16x16

            nn.Conv2d(64, 192, kernel_size=3, padding=1),
            nn.ReLU(),
            nn.MaxPool2d(2, 2),  # 8x8
        )

        self.inception3 = nn.Sequential(
            Inception(192, 64, 96, 128, 16, 32, 32),
            Inception(256, 128, 128, 192, 32, 96, 64),
            nn.MaxPool2d(2, 2),  # 4x4
        )

        self.inception4 = nn.Sequential(
            Inception(480, 192, 96, 208, 16, 48, 64),
            Inception(512, 160, 112, 224, 24, 64, 64),
            nn.MaxPool2d(2, 2),  # 2x2
        )

        self.inception5 = nn.Sequential(
            Inception(512, 256, 160, 320, 32, 128, 128),
            nn.AdaptiveAvgPool2d((1, 1)),
        )

        self.fc = nn.Linear(832, num_classes)

    def forward(self, x):
        x = self.stem(x)
        x = self.inception3(x)
        x = self.inception4(x)
        x = self.inception5(x)

        x = torch.flatten(x, 1)
        return self.fc(x)


# ==========================================
# SNN Inception Block
# ==========================================
class SpikingInception(nn.Module):
    def __init__(self, in_ch, c1, c3r, c3, c5r, c5, pool):
        super().__init__()

        self.branch1 = nn.Sequential(
            layer.Conv2d(in_ch, c1, kernel_size=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid())
        )

        self.branch2 = nn.Sequential(
            layer.Conv2d(in_ch, c3r, kernel_size=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.Conv2d(c3r, c3, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid())
        )

        self.branch3 = nn.Sequential(
            layer.Conv2d(in_ch, c5r, kernel_size=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.Conv2d(c5r, c5, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid())
        )

        self.branch4 = nn.Sequential(
            layer.MaxPool2d(3, stride=1, padding=1),
            layer.Conv2d(in_ch, pool, kernel_size=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid())
        )

    def forward(self, x):
        return torch.cat([
            self.branch1(x),
            self.branch2(x),
            self.branch3(x),
            self.branch4(x)
        ], dim=2)  # 注意：SNN是 (T,N,C,H,W)，拼接通道是 dim=2


# ==========================================
# SNN GoogLeNet
# ==========================================
class SpikingGoogLeNet(nn.Module):
    def __init__(self, num_classes=10, T=8):
        super().__init__()
        self.T = T

        self.stem = nn.Sequential(
            layer.Conv2d(1, 64, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),

            layer.Conv2d(64, 192, kernel_size=3, padding=1),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid()),
            layer.MaxPool2d(2, 2),
        )

        self.inception3 = nn.Sequential(
            SpikingInception(192, 64, 96, 128, 16, 32, 32),
            SpikingInception(256, 128, 128, 192, 32, 96, 64),
            layer.MaxPool2d(2, 2),
        )

        self.inception4 = nn.Sequential(
            SpikingInception(480, 192, 96, 208, 16, 48, 64),
            SpikingInception(512, 160, 112, 224, 24, 64, 64),
            layer.MaxPool2d(2, 2),
        )

        self.inception5 = nn.Sequential(
            SpikingInception(512, 256, 160, 320, 32, 128, 128),
            layer.AdaptiveAvgPool2d((1, 1)),
        )

        self.fc = layer.Linear(832, num_classes)

        functional.set_step_mode(self, step_mode='m')

    def forward(self, x):
        x_seq = x.unsqueeze(0).repeat(self.T, 1, 1, 1, 1)
        functional.reset_net(self)

        x_seq = self.stem(x_seq)
        x_seq = self.inception3(x_seq)
        x_seq = self.inception4(x_seq)
        x_seq = self.inception5(x_seq)

        x_seq = x_seq.flatten(2)  # (T,N,C,1,1) -> (T,N,C)
        x_seq = self.fc(x_seq)

        return x_seq.mean(0)

5. ResNet18 

import torch
import torch.nn as nn
from spikingjelly.activation_based import neuron, functional, surrogate, layer


# ==========================================
# ANN BasicBlock
# ==========================================
class BasicBlock(nn.Module):
    expansion = 1

    def __init__(self, in_planes, planes, stride=1):
        super().__init__()

        self.conv1 = nn.Conv2d(in_planes, planes, kernel_size=3,
                               stride=stride, padding=1, bias=False)
        self.bn1 = nn.BatchNorm2d(planes)
        self.relu = nn.ReLU()

        self.conv2 = nn.Conv2d(planes, planes, kernel_size=3,
                               stride=1, padding=1, bias=False)
        self.bn2 = nn.BatchNorm2d(planes)

        self.shortcut = nn.Sequential()
        if stride != 1 or in_planes != planes:
            self.shortcut = nn.Sequential(
                nn.Conv2d(in_planes, planes, kernel_size=1,
                          stride=stride, bias=False),
                nn.BatchNorm2d(planes)
            )

    def forward(self, x):
        out = self.relu(self.bn1(self.conv1(x)))
        out = self.bn2(self.conv2(out))
        out += self.shortcut(x)
        out = self.relu(out)
        return out


# ==========================================
# ANN ResNet18
# ==========================================
class ClassicResNet18(nn.Module):
    def __init__(self, num_classes=10):
        super().__init__()

        self.in_planes = 64

        self.conv1 = nn.Sequential(
            nn.Conv2d(1, 64, kernel_size=3, stride=1, padding=1, bias=False),
            nn.BatchNorm2d(64),
            nn.ReLU()
        )

        self.layer1 = self._make_layer(64, 2, stride=1)
        self.layer2 = self._make_layer(128, 2, stride=2)
        self.layer3 = self._make_layer(256, 2, stride=2)
        self.layer4 = self._make_layer(512, 2, stride=2)

        self.pool = nn.AdaptiveAvgPool2d((1, 1))
        self.fc = nn.Linear(512, num_classes)

    def _make_layer(self, planes, num_blocks, stride):
        strides = [stride] + [1]*(num_blocks-1)
        layers = []

        for s in strides:
            layers.append(BasicBlock(self.in_planes, planes, s))
            self.in_planes = planes

        return nn.Sequential(*layers)

    def forward(self, x):
        x = self.conv1(x)
        x = self.layer1(x)  # 32x32
        x = self.layer2(x)  # 16x16
        x = self.layer3(x)  # 8x8
        x = self.layer4(x)  # 4x4
        x = self.pool(x)
        x = torch.flatten(x, 1)
        return self.fc(x)


# ==========================================
# SNN BasicBlock
# ==========================================
class SpikingBasicBlock(nn.Module):
    expansion = 1

    def __init__(self, in_planes, planes, stride=1):
        super().__init__()

        self.conv1 = layer.Conv2d(in_planes, planes, kernel_size=3,
                                 stride=stride, padding=1, bias=False)
        self.bn1 = layer.BatchNorm2d(planes)
        self.lif1 = neuron.LIFNode(surrogate_function=surrogate.Sigmoid())

        self.conv2 = layer.Conv2d(planes, planes, kernel_size=3,
                                 stride=1, padding=1, bias=False)
        self.bn2 = layer.BatchNorm2d(planes)
        self.lif2 = neuron.LIFNode(surrogate_function=surrogate.Sigmoid())

        self.shortcut = None
        if stride != 1 or in_planes != planes:
            self.shortcut = nn.Sequential(
                layer.Conv2d(in_planes, planes, kernel_size=1,
                             stride=stride, bias=False),
                layer.BatchNorm2d(planes)
            )

    def forward(self, x):
        out = self.conv1(x)
        out = self.bn1(out)
        out = self.lif1(out)

        out = self.conv2(out)
        out = self.bn2(out)

        shortcut = x if self.shortcut is None else self.shortcut(x)

        out = out + shortcut
        out = self.lif2(out)
        return out


# ==========================================
# SNN ResNet18
# ==========================================
class SpikingResNet18(nn.Module):
    def __init__(self, num_classes=10, T=8):
        super().__init__()
        self.T = T

        self.in_planes = 64

        self.conv1 = nn.Sequential(
            layer.Conv2d(1, 64, kernel_size=3, stride=1, padding=1, bias=False),
            layer.BatchNorm2d(64),
            neuron.LIFNode(surrogate_function=surrogate.Sigmoid())
        )

        self.layer1 = self._make_layer(64, 2, stride=1)
        self.layer2 = self._make_layer(128, 2, stride=2)
        self.layer3 = self._make_layer(256, 2, stride=2)
        self.layer4 = self._make_layer(512, 2, stride=2)

        self.pool = layer.AdaptiveAvgPool2d((1, 1))
        self.fc = layer.Linear(512, num_classes)

        functional.set_step_mode(self, step_mode='m')

    def _make_layer(self, planes, num_blocks, stride):
        strides = [stride] + [1]*(num_blocks-1)
        layers = []

        for s in strides:
            layers.append(SpikingBasicBlock(self.in_planes, planes, s))
            self.in_planes = planes

        return nn.Sequential(*layers)

    def forward(self, x):
        x_seq = x.unsqueeze(0).repeat(self.T, 1, 1, 1, 1)
        functional.reset_net(self)

        x_seq = self.conv1(x_seq)
        x_seq = self.layer1(x_seq)
        x_seq = self.layer2(x_seq)
        x_seq = self.layer3(x_seq)
        x_seq = self.layer4(x_seq)

        x_seq = self.pool(x_seq)
        x_seq = x_seq.flatten(2)
        x_seq = self.fc(x_seq)

        return x_seq.mean(0)
