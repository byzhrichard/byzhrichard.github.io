📖参考资料

U-Net: Convolutional Networks for Biomedical Image Segmentation.

🌱背景

语义分割的目标：输入一张图像，输出一张像素级类别图

  • $Input: (H, W, C)$
  • $Output: (H, W, K)$

FCN 的核心思想是：把分类 CNN 改造成全卷积网络，让它可以输出 dense prediction map

但是 FCN 也有一个比较明显的问题：

  • 深层特征语义强，但是空间分辨率低
  • 浅层特征细节多，但是语义弱

FCN 虽然使用了 skip connection，但是融合方式相对比较粗

能不能在恢复分辨率的时候，更充分地利用浅层特征中的空间细节? -> U-Net

U-Net 的核心思想：编码器负责提取语义，解码器负责恢复分辨率，并通过 skip connection 把浅层细节直接拼接回来

它的整体结构长得像一个字母 U：

  graph LR
    A[Input Image] --> B[Encoder
Downsampling]
    B --> C[Bottleneck]
    C --> D[Decoder
Upsampling]
    D --> E[Segmentation Map]

    B -. skip connection .-> D

更加直观地说：

左边：不断下采样，提取高级语义 右边：不断上采样，恢复空间分辨率 中间：最深层 bottleneck 横向：skip connection，把浅层细节传给解码器

⚙️架构(公式)

1. Encoder（下采样）

每一层通常包含：

$$ Conv(3\times3) \rightarrow ReLU \rightarrow Conv(3\times3) \rightarrow ReLU \rightarrow Pool $$

特点：

  • 分辨率 ↓
  • 通道数 ↑
  • 语义增强

2. Decoder（上采样）

每一层：

$$ UpConv \rightarrow Concat(skip) \rightarrow Conv \rightarrow Conv $$
  • $[...,...]$ 表示 channel concat
  • skip feature 来自 encoder 对应层

3. Concat

U-Net 的关键变化：

$$ F_{fusion} = concat(F_{encoder}, F_{decoder}) $$
方法 融合方式 - Skip Connection
FCN element-wise sum
U-Net channel-wise concat
  • add: y=x1+x2 -> 信息混合不可逆
  • concat: y=[x1,x2] -> 信息保留更完整 -> decoder 可以“自己学怎么用浅层信息”

FCN：把深层语义“粗暴加回去”

U-Net：把浅层细节“完整带回来再重新融合”

👍优点/创新点

1. 强保真空间恢复

skip connection 直接传递高分辨率特征

2. 小数据友好

  • 参数效率高
  • inductive bias 强 (归纳偏置)

👎缺点

1. 显存消耗大

concat 会导致 channel doubling

2. 语义融合方式简单

只是 concat + conv，没有显式 attention

💻模型代码 

import torch
import torch.nn as nn


# =========================
# 基本卷积模块（Conv + ReLU ×2）
# =========================
class DoubleConv(nn.Module):
    """
    两次 3x3 Conv + ReLU
    U-Net 的基本构建单元
    """

    def __init__(self, in_channels, out_channels):
        super(DoubleConv, self).__init__()

        self.conv = nn.Sequential(
            nn.Conv2d(in_channels, out_channels, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),

            nn.Conv2d(out_channels, out_channels, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),
        )

    def forward(self, x):
        return self.conv(x)


# =========================
# U-Net 主体结构
# =========================
class UNet(nn.Module):
    def __init__(self, num_classes=2):
        super(UNet, self).__init__()

        # -------- Encoder（下采样）--------
        self.enc1 = DoubleConv(3, 64)
        self.enc2 = DoubleConv(64, 128)
        self.enc3 = DoubleConv(128, 256)
        self.enc4 = DoubleConv(256, 512)

        # 下采样（2x2 maxpool）
        self.pool = nn.MaxPool2d(kernel_size=2, stride=2)

        # -------- Bottleneck（最深层）--------
        self.bottleneck = DoubleConv(512, 1024)

        # -------- Decoder（上采样）--------
        self.up4 = nn.ConvTranspose2d(1024, 512, kernel_size=2, stride=2)
        self.dec4 = DoubleConv(1024, 512)

        self.up3 = nn.ConvTranspose2d(512, 256, kernel_size=2, stride=2)
        self.dec3 = DoubleConv(512, 256)

        self.up2 = nn.ConvTranspose2d(256, 128, kernel_size=2, stride=2)
        self.dec2 = DoubleConv(256, 128)

        self.up1 = nn.ConvTranspose2d(128, 64, kernel_size=2, stride=2)
        self.dec1 = DoubleConv(128, 64)

        # -------- 输出层 --------
        self.out_conv = nn.Conv2d(64, num_classes, kernel_size=1)

    def forward(self, x):
        # =========================
        # Encoder path
        # =========================
        x1 = self.enc1(x)          # 64 channels
        x2 = self.enc2(self.pool(x1))  # 128 channels
        x3 = self.enc3(self.pool(x2))  # 256 channels
        x4 = self.enc4(self.pool(x3))  # 512 channels

        # =========================
        # Bottleneck
        # =========================
        x5 = self.bottleneck(self.pool(x4))  # 1024 channels

        # =========================
        # Decoder path + Skip Connection
        # =========================

        # ---- 8s level ----
        d4 = self.up4(x5)
        d4 = torch.cat([d4, x4], dim=1)  # skip connection (concat)
        d4 = self.dec4(d4)

        # ---- 16s level ----
        d3 = self.up3(d4)
        d3 = torch.cat([d3, x3], dim=1)
        d3 = self.dec3(d3)

        # ---- 32s level ----
        d2 = self.up2(d3)
        d2 = torch.cat([d2, x2], dim=1)
        d2 = self.dec2(d2)

        # ---- final level ----
        d1 = self.up1(d2)
        d1 = torch.cat([d1, x1], dim=1)
        d1 = self.dec1(d1)

        # 输出 segmentation map
        out = self.out_conv(d1)

        return out


# =========================
# 简单 random 测试 main
# =========================
if __name__ == "__main__":

    # 随机输入：
    # batch_size=1, RGB image 256x256
    x = torch.randn(1, 3, 256, 256)

    # 初始化模型
    model = UNet(num_classes=3)

    # forward
    y = model(x)

    # 输出 shape
    print("Input shape :", x.shape)
    print("Output shape:", y.shape)
