📖参考资料

Fully Convolutional Networks for Semantic Segmentation.

🌱背景

语义分割（Semantic Segmentation）：给图像里的每一个像素都分一个类别

输入图像 输出结果
一张街景 每个像素标成：道路 / 人 / 车 / 建筑

普通分类任务：

  • 输入：一张图片
  • 输出：一个标签 👉 “这是一只猫”

目标检测：

  • 输出：多个框 + 类别 👉 “猫在左边，狗在右边”

语义分割：

  • 输出：一张“同尺寸或近似尺寸”的图
  • 每个像素都有类别

在 FCN 出现之前，语义分割任务基本是“patch-based classification”的天下

  • 把一张图像切成很多小块（patch）
  • 每个 patch 丢进 CNN 做分类
  • 再把结果拼回去形成分割图

问题很明显：

  1. 计算重复严重 (相邻 patch 大量重叠 → 重复卷积)
  2. 空间信息利用差 (patch 级别分类 → 缺少全局结构)
  3. 输出分辨率低 (CNN 后面接 FC 层 → 固定尺寸输出)

于是就有一个很直接的问题：能不能让 CNN 直接输出“像素级预测图”？

FCN: Fully Convolutional Networks: 把全连接层全部换成卷积层，让网络从“分类器”变成“密集预测器”

标准 CNN：$Conv \rightarrow Conv \rightarrow FC \rightarrow FC$

FCN：$Conv \rightarrow Conv \rightarrow Conv \rightarrow 1 \times 1 Conv$

⚙️架构(公式)

1. FC → Conv 的等价变换

假设输入是：$x \in \mathbb{R}^{H \times W \times C}$

则全连接层: (展平后, 全相连)

$$ y = Wx + b,\quad x \in \mathbb{R}^{H W C} $$

则1x1Conv: (在每一个空间位置上做一次“通道上的全连接”)

$$ y_{i,j} = W x_{i,j} + b,\quad x_{i,j} \in \mathbb{R}^{C} $$
操作 看什么 是否保留空间
FC 看整张图（flatten） ❌ 不保留
1×1 Conv 每个像素独立做 FC ✔ 保留

FCN 是怎么“等价”的 -> trick: 把 FC layer reshape 成 Conv kernel

举例: 假设 CNN 最后 feature map 为 $7 \times 7 \times 512$

原来: (展平后全连接)

flatten: 7×7×512 → 25088
FC: 25088 → 4096

现在: (把 FC 改写成：7×7 卷积)

conv(7x7): 7×7×512 → 1×1×4096

-> 注意: 二者本质相同, 且参数量是一样的 $7×7×512×4096$

1x1 Conv: 每个像素做 FC（只在 channel 上）

所以:

  • 7×7 Conv: 负责“全局感受野（替代 FC）”
  • 1x1 Conv: 负责“类别映射（classifier head）”

-> 输入不再被限制为固定尺寸

2. Dense Prediction

FCN 的输出变成：

$$ H \times W \times C $$

每个位置预测一个类别：

$$ \hat{y}_{i,j} = softmax(f(x_{i,j})) $$

3. 上采样（Upsampling）

由于 下采样 pooling/downsampling：

$$ H_{out} \ll H_{in} $$

需要恢复分辨率 -> 反卷积（Transposed Convolution）:

$$ Y = W^{T} * X $$
  • learnable upsampling
  • 可以恢复空间结构

4. skip connection

FCN 的经典版本：

  • FCN-32s（粗）
  • FCN-16s（中）
  • FCN-8s（细）

融合方式：

$$ F = F_{deep} + F_{shallow} $$
输入图像:        H × W
pool3:          H/8  × W/8
pool4:          H/16 × W/16
pool5:          H/32 × W/32

越往深层：

  • 分辨率越来越低
  • 语义越来越强

-> 深层负责判断“是什么”, 浅层负责帮助判断“在哪里”

特征层 尺寸 特点
pool5 H/32 语义最强，但最粗
pool4 H/16 语义较强，细节稍好
pool3 H/8 细节更好，但语义较弱

FCN-32s 只用最深层特征：语义强，但是结果粗

pool5 → score → 上采样 32 倍 → 输出

FCN-16s 不仅用 pool5，还融合 pool4

pool5 score:      H/32
      ↓ upsample ×2
                  H/16
pool4 score:      H/16

相加 → H/16 → upsample ×16 → H

FCN-8s 进一步融合 pool3

pool5 score:      H/32
      ↓ upsample ×2
                  H/16
pool4 score:      H/16
      ↓ 相加
                  H/16
      ↓ upsample ×2
                  H/8
pool3 score:      H/8
      ↓ 相加
                  H/8
      ↓ upsample ×8
                  H

👍优点/创新点

1. 端到端语义分割

不再需要 patch cropping, 而可以输入整图-输出整图

2. 任意尺寸输入

因为没有 FC：

$$ H, W \rightarrow arbitrary $$

3. 全卷积结构

把 CNN 从 classification network 变成 dense prediction network

👎缺点

1. 输出仍然不够精细

pooling 丢失空间信息

upsampling 过于粗糙

-> 边界模糊（classic FCN problem）

2. spatial consistency 弱

像素之间独立预测：

$$ p(y_{i,j}) \neq p(y_{i+1,j}) $$

导致：

  • 噪点
  • 边界锯齿

💻模型代码 

import torch
import torch.nn as nn
import torch.nn.functional as F

from byzh_ai.Butils import b_get_params

# =========================
# VGG16 Backbone (FCN version)
# =========================
class VGG16_FCN(nn.Module):
    """
    输入:  (N, 3, 224, 224)

    输出:
    pool3: (N, 256, H/8,  W/8)
    pool4: (N, 512, H/16, W/16)
    pool5: (N, 512, H/32, W/32)
    """

    def __init__(self):
        super().__init__()

        # block1: 提取边缘、纹理等低级特征
        self.b1 = nn.Sequential(
            nn.Conv2d(3, 64, 3, padding=1),
            nn.ReLU(True),
            nn.Conv2d(64, 64, 3, padding=1),
            nn.ReLU(True),
            nn.MaxPool2d(2, 2),   # /2
        )

        # block2: 提取中级纹理特征
        self.b2 = nn.Sequential(
            nn.Conv2d(64, 128, 3, padding=1),
            nn.ReLU(True),
            nn.Conv2d(128, 128, 3, padding=1),
            nn.ReLU(True),
            nn.MaxPool2d(2, 2),   # /4
        )

        # block3: 提取初步语义结构，输出 pool3
        self.b3 = nn.Sequential(
            nn.Conv2d(128, 256, 3, padding=1),
            nn.ReLU(True),
            nn.Conv2d(256, 256, 3, padding=1),
            nn.ReLU(True),
            nn.Conv2d(256, 256, 3, padding=1),
            nn.ReLU(True),
            nn.MaxPool2d(2, 2),   # /8
        )

        # block4: 提取高层语义，输出 pool4
        self.b4 = nn.Sequential(
            nn.Conv2d(256, 512, 3, padding=1),
            nn.ReLU(True),
            nn.Conv2d(512, 512, 3, padding=1),
            nn.ReLU(True),
            nn.Conv2d(512, 512, 3, padding=1),
            nn.ReLU(True),
            nn.MaxPool2d(2, 2),   # /16
        )

        # block5: 提取最深层的全局语义，输出 pool5
        self.b5 = nn.Sequential(
            nn.Conv2d(512, 512, 3, padding=1),
            nn.ReLU(True),
            nn.Conv2d(512, 512, 3, padding=1),
            nn.ReLU(True),
            nn.Conv2d(512, 512, 3, padding=1),
            nn.ReLU(True),
            nn.MaxPool2d(2, 2),   # /32
        )

    def forward(self, x):
        x = self.b1(x)      # (N, 64, H/2, W/2)
        x = self.b2(x)      # (N, 128, H/4, W/4)

        pool3 = self.b3(x)  # (N, 256, H/8, W/8)
        pool4 = self.b4(pool3)  # (N, 512, H/16, W/16)
        pool5 = self.b5(pool4)  # (N, 512, H/32, W/32)

        return pool3, pool4, pool5


# =========================
# FCN Model
# =========================
class FCN(nn.Module):
    """
    mode:
        - fcn32s
        - fcn16s
        - fcn8s
    """

    def __init__(self, num_classes=21, mode="fcn32s"):
        super().__init__()

        self.mode = mode
        self.backbone = VGG16_FCN()

        # FC6: 7×7 卷积（padding=3 保持分辨率），模拟全连接层进行全局语义聚合
        self.fc6 = nn.Conv2d(512, 4096, 7, padding=3)
        self.relu6 = nn.ReLU(True)
        self.drop6 = nn.Dropout2d()          # 按通道随机失活，防止过拟合

        # FC7: 1×1 卷积，对全局特征做非线性变换
        self.fc7 = nn.Conv2d(4096, 4096, 1)
        self.relu7 = nn.ReLU(True)
        self.drop7 = nn.Dropout2d()          # 按通道随机失活

        # 将深层特征映射为各类别的分数图
        self.score_fr = nn.Conv2d(4096, num_classes, 1)

        # 跳跃连接：将 pool4 特征映射为类别分数（用于 fcn16s / fcn8s）
        self.score_pool4 = nn.Conv2d(512, num_classes, 1)

        # 跳跃连接：将 pool3 特征映射为类别分数（用于 fcn8s）
        self.score_pool3 = nn.Conv2d(256, num_classes, 1)

    def forward(self, x):
        # 保存原始输入尺寸，用于最后上采样到全分辨率
        input_size = x.shape[-2:]  # (H, W)

        pool3, pool4, pool5 = self.backbone(x)
        # pool3: (N, 256, H/8, W/8)
        # pool4: (N, 512, H/16, W/16)
        # pool5: (N, 512, H/32, W/32)

        # ---- 深层分支 ----
        x = self.fc6(pool5)         # (N, 4096, H/32, W/32)
        x = self.relu6(x)
        x = self.drop6(x)

        x = self.fc7(x)             # (N, 4096, H/32, W/32)
        x = self.relu7(x)
        x = self.drop7(x)

        score = self.score_fr(x)    # (N, num_classes, H/32, W/32)

        # ---- FCN-32s：直接将 /32 分数图上采样回原图 ----
        if self.mode == "fcn32s":
            return F.interpolate(score,
                                 size=input_size,
                                 mode="bilinear",
                                 align_corners=False)   # (N, num_classes, H, W)

        # ---- FCN-16s / FCN-8s 共用：融合 pool4 分数 ----
        if self.mode in ["fcn16s", "fcn8s"]:
            score_pool4 = self.score_pool4(pool4)       # (N, num_classes, H/16, W/16)

            # 将 /32 分数图上采样 2 倍，与 pool4 对齐
            score = F.interpolate(score,
                                  size=score_pool4.shape[2:],
                                  mode="bilinear",
                                  align_corners=False)  # (N, num_classes, H/16, W/16)
            score = score + score_pool4                 # 融合

            if self.mode == "fcn16s":
                # 融合后直接上采样 16 倍回原图
                return F.interpolate(score,
                                     size=input_size,
                                     mode="bilinear",
                                     align_corners=False)  # (N, num_classes, H, W)

        # ---- FCN-8s：进一步融合 pool3 分数 ----
        if self.mode == "fcn8s":
            score_pool3 = self.score_pool3(pool3)       # (N, num_classes, H/8, W/8)

            # 将当前融合后的 /16 分数图上采样 2 倍，与 pool3 对齐
            score = F.interpolate(score,
                                  size=score_pool3.shape[2:],
                                  mode="bilinear",
                                  align_corners=False)  # (N, num_classes, H/8, W/8)
            score = score + score_pool3                 # 再次融合

            # 最终上采样 8 倍回原图
            return F.interpolate(score,
                                 size=input_size,
                                 mode="bilinear",
                                 align_corners=False)  # (N, num_classes, H, W)


# =========================
# test
# =========================
if __name__ == "__main__":
    x = torch.randn(2, 3, 224, 224)

    model32 = FCN(21, "fcn32s")
    model16 = FCN(21, "fcn16s")
    model8  = FCN(21, "fcn8s")

    print("FCN-32s:", model32(x).shape)
    print(f"参数量: {b_get_params(model32)}") # 134_362_751
    print("FCN-16s:", model16(x).shape)
    print(f"参数量: {b_get_params(model16)}") # 134_362_751
    print("FCN-8s :", model8(x).shape)
    print(f"参数量: {b_get_params(model8)}") # 134_362_751
