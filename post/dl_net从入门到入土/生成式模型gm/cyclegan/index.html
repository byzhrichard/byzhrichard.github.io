📖参考资料

Unpaired Image-to-Image Translation using Cycle-Consistent Adversarial Networks.

🌱背景

普通 GAN 更偏向于：从随机噪声 z 生成一张图片

例如：随机噪声 z → Generator → 生成图片

而在实际任务中，我们更希望做的是把一种风格的图片转换成另一种风格的图片

  • 马 → 斑马
  • 苹果 → 橙子
  • 白天 → 黑夜
  • 普通照片 → 梵高风格
  • 夏天场景 → 冬天场景

这类任务叫做：Image-to-Image Translation：图像到图像转换

Pix2Pix 可以完成, 但需要成对数据 paired data 但是现实中, 成对数据很难收集, 故而采用CycleGAN

CycleGAN 的核心优势：Unpaired Image-to-Image Translation：非成对图像转换

CycleGAN 的核心思想：如果一张图片从 A 域转换到 B 域，再从 B 域转换回 A 域，应该还能变回原来的图片 -> 称为 循环一致性(Cycle Consistency)

⚙️架构(公式) 

  graph LR
X[真实图片 x ∈ X] --> G[生成器 G: X -> Y]
G --> FY["生成图片 fake_y = G(x)"]
FY --> DY[判别器 D_Y]
DY --> OY[判断是否像真实 Y 域图片]

FY --> F[生成器 F: Y -> X]
F --> RX["重建图片 rec_x = F(G(x))"]
RX --> CX["希望 rec_x ≈ x"]

Y[真实图片 y ∈ Y] --> F2[生成器 F: Y -> X]
F2 --> FX["生成图片 fake_x = F(y)"]
FX --> DX[判别器 D_X]
DX --> OX[判断是否像真实 X 域图片]

FX --> G2[生成器 G: X -> Y]
G2 --> RY["重建图片 rec_y = G(F(y))"]
RY --> CY["希望 rec_y ≈ y"]

CycleGAN 主要包含：

  • 两个生成器
  • 两个判别器

假设有两个图像域：

  • $X$：源域，例如马
  • $Y$：目标域，例如斑马

1. 两个生成器

有两个生成器：

  • $G: X \rightarrow Y$ (把 X 域图片转换成 Y 域风格)
  • $F: Y \rightarrow X$ (把 Y 域图片转换成 X 域风格)

2. 两个判别器

有两个判别器：

  • $D_Y$：判断图片是不是真实的 Y 域图片
  • $D_X$：判断图片是不是真实的 X 域图片

3. 对抗损失 Adversarial Loss

对于映射：

$$ G: X \rightarrow Y $$

其损失可以写成：

$$ L_{GAN}(G, D_Y, X, Y)= \mathbb{E}_{y \sim p_{data}(y)}[\log D_Y(y)] + \mathbb{E}_{x \sim p_{data}(x)}[\log(1 - D_Y(G(x)))] $$
  • $D_Y$ 希望真实的 $y$ 被判断为真
  • $D_Y$ 希望生成的 $G(x)$ 被判断为假
  • $G$ 希望生成的 $G(x)$ 能骗过 $D_Y$

同理，对于反方向映射：

$$ > F: Y \rightarrow X > $$

也有：

$$ > L_{GAN}(F, D_X, Y, X)=\mathbb{E}_{x \sim p_{data}(x)}[\log D_X(x)]+\mathbb{E}_{y \sim p_{data}(y)}[\log(1 - D_X(F(y)))] > $$

4. 循环一致性损失 Cycle Consistency Loss

只使用对抗损失是不够的 因为模型只需要生成“像目标域”的图片即可，不一定保留原图内容

例如：

输入一张马, 生成器只要输出一张看起来像斑马的图片 但这张斑马可能和原来的马没有任何关系

所以 CycleGAN 加入了循环一致性约束：

$$ x \rightarrow G(x) \rightarrow F(G(x)) \approx x $$

对应损失为：

$$ L_{cyc}(G, F)= \mathbb{E}_{x \sim p_{data}(x)} [ ||F(G(x)) - x||_1 ] + \mathbb{E}_{y \sim p_{data}(y)} [ ||G(F(y)) - y||_1 ] $$

这里通常使用 L1 损失，而不是 L2 损失

因为 L1 损失相对更不容易让图像变得过度平滑

5. 身份损失 Identity Loss

有时 CycleGAN 还会加入 Identity Loss 作用是：如果输入图片已经属于目标域，那么生成器最好不要乱改它

例如，生成器 $G$ 的作用是：

$$ G: X \rightarrow Y $$

如果输入本来就是 Y 域图片，那么：

$$ G(y) \approx y $$

例如：

如果给 马→斑马 的生成器输入一张斑马 它最好不要再乱改

Identity Loss 可以写成：

$$ L_{identity}(G, F)= \mathbb{E}_{y \sim p_{data}(y)} [ ||G(y) - y||_1 ] + \mathbb{E}_{x \sim p_{data}(x)} [ ||F(x) - x||_1 ] $$

这个损失在颜色迁移、风格迁移中比较有用，可以帮助模型保留颜色、亮度等信息

6. 总目标函数

CycleGAN 的总损失通常由三部分组成：

$$ L(G, F, D_X, D_Y)= L_{GAN}(G, D_Y, X, Y) + L_{GAN}(F, D_X, Y, X) + \lambda L_{cyc}(G, F) $$

如果加入 Identity Loss，则可以写成：

$$ L_{total}= L_{GAN} + \lambda_{cyc} L_{cyc} + \lambda_{id} L_{identity} $$
  • $L_{GAN}$：让生成图片看起来像目标域
  • $L_{cyc}$：保证转换前后内容一致
  • $L_{identity}$：减少不必要的颜色和内容改变
  • $\lambda_{cyc}$：循环一致性损失权重
  • $\lambda_{id}$：身份损失权重

常见设置：$\lambda_{cyc} = 10$

👍优点/创新点

1. 不需要成对数据

Pix2Pix 需要成对数据：

输入图 A ↔ 输出图 B

但是 CycleGAN 只需要：

一堆 X 域图片 + 一堆 Y 域图片

2. 循环一致性约束

普通 GAN 只关心生成结果像不像真实样本 但是 CycleGAN 还要求：转换过去之后，还能转换回来

也就是：$x → G(x) → F(G(x)) ≈ x$

3. 适合风格迁移类任务

👎缺点

1. 不适合几何结构变化太大的任务

CycleGAN 更擅长风格转换，而不是复杂形状变化

因为循环一致性会强迫模型尽量保留原图结构

2. 训练仍然不稳定, 计算成本更高

💻模型代码 

import torch
import torch.nn as nn


class ResnetBlock(nn.Module):
    """
    ResNet 残差块

    输入 shape:
        [batch_size, channels, height, width]

    输出 shape:
        [batch_size, channels, height, width]
    """

    def __init__(self, channels):
        super().__init__()

        self.block = nn.Sequential(
            nn.ReflectionPad2d(1),
            nn.Conv2d(channels, channels, kernel_size=3, stride=1, padding=0),
            nn.InstanceNorm2d(channels),
            nn.ReLU(inplace=True),

            nn.ReflectionPad2d(1),
            nn.Conv2d(channels, channels, kernel_size=3, stride=1, padding=0),
            nn.InstanceNorm2d(channels)
        )

    def forward(self, x):
        return x + self.block(x)


class Generator(nn.Module):
    """
    CycleGAN 生成器 Generator

    作用:
        将一个域的图片转换到另一个域

    输入:
        img: 图片
        shape = [batch_size, img_channels, height, width]

    输出:
        fake_img: 转换后的图片
        shape = [batch_size, img_channels, height, width]
    """

    def __init__(self, img_channels=3, num_residual_blocks=9):
        super().__init__()

        model = []

        # 1. 初始卷积层
        model += [
            nn.ReflectionPad2d(3),
            nn.Conv2d(img_channels, 64, kernel_size=7, stride=1, padding=0),
            nn.InstanceNorm2d(64),
            nn.ReLU(inplace=True)
        ]

        # 2. 下采样
        in_channels = 64
        out_channels = 128

        for _ in range(2):
            model += [
                nn.Conv2d(
                    in_channels,
                    out_channels,
                    kernel_size=3,
                    stride=2,
                    padding=1
                ),
                nn.InstanceNorm2d(out_channels),
                nn.ReLU(inplace=True)
            ]

            in_channels = out_channels
            out_channels *= 2

        # 3. 残差块
        for _ in range(num_residual_blocks):
            model += [ResnetBlock(in_channels)]

        # 4. 上采样
        out_channels = in_channels // 2

        for _ in range(2):
            model += [
                nn.ConvTranspose2d(
                    in_channels,
                    out_channels,
                    kernel_size=3,
                    stride=2,
                    padding=1,
                    output_padding=1
                ),
                nn.InstanceNorm2d(out_channels),
                nn.ReLU(inplace=True)
            ]

            in_channels = out_channels
            out_channels = in_channels // 2

        # 5. 输出层
        model += [
            nn.ReflectionPad2d(3),
            nn.Conv2d(64, img_channels, kernel_size=7, stride=1, padding=0),
            nn.Tanh()
        ]

        self.model = nn.Sequential(*model)

    def forward(self, img):
        return self.model(img)


class Discriminator(nn.Module):
    """
    CycleGAN 判别器 Discriminator

    PatchGAN 判别器:
        不直接输出一个单独的真假概率
        而是输出一个二维特征图
        每个位置表示对应 patch 的真假判断

    输入:
        img: 图片
        shape = [batch_size, img_channels, height, width]

    输出:
        validity: Patch 真假判断结果
        shape 约为 [batch_size, 1, h, w]
    """

    def __init__(self, img_channels=3):
        super().__init__()

        def discriminator_block(in_channels, out_channels, normalize=True):
            layers = [
                nn.Conv2d(
                    in_channels,
                    out_channels,
                    kernel_size=4,
                    stride=2,
                    padding=1
                )
            ]

            if normalize:
                layers.append(nn.InstanceNorm2d(out_channels))

            layers.append(nn.LeakyReLU(0.2, inplace=True))

            return layers

        self.model = nn.Sequential(
            *discriminator_block(img_channels, 64, normalize=False),
            *discriminator_block(64, 128),
            *discriminator_block(128, 256),
            *discriminator_block(256, 512),

            nn.ZeroPad2d((1, 0, 1, 0)),
            nn.Conv2d(512, 1, kernel_size=4, padding=1)
        )

    def forward(self, img):
        return self.model(img)


if __name__ == "__main__":
    batch_size = 2
    img_channels = 3
    img_size = 256

    # 两个生成器
    G_XtoY = Generator(img_channels=img_channels)
    G_YtoX = Generator(img_channels=img_channels)

    # 两个判别器
    D_X = Discriminator(img_channels=img_channels)
    D_Y = Discriminator(img_channels=img_channels)

    # 假设 X 域和 Y 域各有一批图片
    real_x = torch.randn(batch_size, img_channels, img_size, img_size)
    real_y = torch.randn(batch_size, img_channels, img_size, img_size)

    # X -> Y
    fake_y = G_XtoY(real_x)

    # Y -> X
    fake_x = G_YtoX(real_y)

    # 循环重建
    rec_x = G_YtoX(fake_y)
    rec_y = G_XtoY(fake_x)

    # 判别器判断
    pred_fake_y = D_Y(fake_y)
    pred_fake_x = D_X(fake_x)

    print("real_x 的形状:", real_x.shape)
    print("real_y 的形状:", real_y.shape)

    print("fake_y = G_XtoY(real_x) 的形状:", fake_y.shape)
    print("fake_x = G_YtoX(real_y) 的形状:", fake_x.shape)

    print("rec_x = G_YtoX(fake_y) 的形状:", rec_x.shape)
    print("rec_y = G_XtoY(fake_x) 的形状:", rec_y.shape)

    print("D_Y(fake_y) 的输出形状:", pred_fake_y.shape)
    print("D_X(fake_x) 的输出形状:", pred_fake_x.shape)
