📖参考资料

Semi-Supervised Classification with Graph Convolutional Networks.

🌱背景

在传统深度学习中（CNN / RNN），数据通常满足欧式结构：

  • 图像 → 网格结构
  • 序列 → 时间结构

但现实中大量数据是非欧式结构：

  • 社交网络（用户-用户关系）
  • 知识图谱（实体-关系）
  • 分子结构（原子-键）
  • 推荐系统（用户-物品图）

这些数据可以统一表示为：Graph = (V, E) (其中 V 是节点，E 是边)

CNN 无法直接处理“非规则邻接关系” -> 图神经网络（GNN） / 图卷积网络（GCN）

⚙️架构(公式)

1. 图结构定义

给定图：

  • 邻接矩阵：$A \in \mathbb{R}^{N \times N}$
  • 特征矩阵：$X \in \mathbb{R}^{N \times F}$

自环(self-loop)：

$$ \tilde{A} = A + I $$

-> 让“自己也参与消息传递”

度矩阵(degree matrix)：只在对角线上有值 第 i 个节点“连了多少条边”（包括自己）

$$ \tilde{D}_{ii} = \sum_j \tilde{A}_{ij} $$

-> 统计每个节点“连接强度”

2. 传播公式

GCN 的一层可以写为：

$$ H^{(l+1)} = \sigma \left( \tilde{D}^{-\frac{1}{2}} \tilde{A} \tilde{D}^{-\frac{1}{2}} H^{(l)} W^{(l)} \right) $$
  • $H^{(0)} = X$
  • $W^{(l)}$：可学习参数
  • $\sigma$：激活函数（ReLU）
  • 归一化邻接矩阵: $\tilde{D}^{-1/2}\,\tilde{A}\,\tilde{D}^{-1/2}$ -> 让图上的信息传播变得公平、不偏向大度节点的一种加权平均规则

比如：

  • 一个节点连了 100 个邻居
  • 一个节点只连 2 个邻居

如果不归一化邻接矩阵, 而是直接相加, 则高度节点信息会压制其他节点

3. 两层GCN结构

$$ Z = \text{softmax}(\hat{A} \cdot \text{ReLU}(\hat{A} X W_0) W_1) $$

👍优点/创新点

1. 能处理非欧式数据

首次把 CNN 的“卷积思想”推广到图结构

2. 参数共享

相比全连接网络：

  • 参数量小
  • 泛化能力强

3. 局部信息传播机制

本质是：“邻居平均 + 特征变换”

适合社交网络 / 知识图谱

4. 半监督学习能力强

只需要少量标注节点即可训练：

  • 节点分类（Node Classification）
  • 图分类（Graph Classification）

👎缺点

1. 过平滑

多层之后：所有节点特征趋于一致

本质：随着层数加深，每个节点的特征越来越像，最后几乎一样 (所有节点“失去个性”，变成一团平均值)

$$ H^{(l)} \rightarrow \text{constant} $$

-> 只适合浅层模型 (深层会退化)

2. 依赖图结构质量

如果图有噪声：

  • 错误边会传播错误信息
  • 性能明显下降

3. 计算效率问题

  • 对大图不友好
  • 内存开销较大

💻模型代码 

import torch
import torch.nn as nn
import torch.nn.functional as F


def normalize_adj(A):
    """
    A: [N, N] adjacency matrix
    return: D^{-1/2} A D^{-1/2}
    """
    I = torch.eye(A.size(0))
    A_hat = A + I  # add self-loop

    D = torch.sum(A_hat, dim=1)  # degree
    D_inv_sqrt = torch.diag(torch.pow(D, -0.5))

    A_norm = D_inv_sqrt @ A_hat @ D_inv_sqrt
    return A_norm


class GCNLayer(nn.Module):
    def __init__(self, in_dim, out_dim):
        super().__init__()
        self.linear = nn.Linear(in_dim, out_dim)

    def forward(self, X, A_norm):
        # X: [N, F]
        # A_norm: [N, N]
        return F.relu(A_norm @ self.linear(X))


class SimpleGCN(nn.Module):
    def __init__(self, in_dim, hidden_dim, out_dim):
        super().__init__()
        self.gcn1 = GCNLayer(in_dim, hidden_dim)
        self.gcn2 = GCNLayer(hidden_dim, out_dim)

    def forward(self, X, A_norm):
        x = self.gcn1(X, A_norm)
        x = self.gcn2(x, A_norm)
        return x

if __name__ == "__main__":
    torch.manual_seed(0)

    # -----------------------
    # 1. 随机图 (4个节点)
    # -----------------------
    N = 4
    F_in = 3

    # 随机邻接矩阵（无向图）
    A = torch.randint(0, 2, (N, N)).float()
    A = (A + A.t()) / 2  # make symmetric
    A.fill_diagonal_(0)

    print("Adjacency A:\n", A)

    # -----------------------
    # 2. 随机节点特征
    # -----------------------
    X = torch.randn(N, F_in)

    print("\nNode features X:\n", X)

    # -----------------------
    # 3. 归一化邻接矩阵
    # -----------------------
    A_norm = normalize_adj(A)

    print("\nNormalized adjacency:\n", A_norm)

    # -----------------------
    # 4. GCN forward
    # -----------------------
    model = SimpleGCN(in_dim=3, hidden_dim=4, out_dim=2)

    out = model(X, A_norm)

    print("\nGCN output:\n", out)
