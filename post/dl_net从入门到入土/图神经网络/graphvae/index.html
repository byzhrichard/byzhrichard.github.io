📖参考资料

GraphVAE: Towards Generation of Small Graphs Using Variational Autoencoders.

🌱背景

在传统 VAE 中，我们处理的是：

  • 图像（像素）
  • 向量（连续特征）

但现实中很多数据是：图结构（Graph）

  • 社交网络（用户-用户）
  • 分子结构（原子-键）
  • 推荐系统（用户-物品图）
  • 知识图谱（实体-关系）

GraphVAE要解决的问题: **学习一个“图的生成模型” $p(G)$ **

本质是: VAE + 图结构建模

  • 图编码（Graph Encoding）
  • 图生成（Graph Generation）
  • -> 学习“图的分布”

⚙️架构(公式) 

输入图 G = (A, X)
        ↓
   GCN Encoder
        ↓
      μ, σ
        ↓
   sampling z
        ↓
Inner Product Decoder
        ↓
  重构邻接矩阵 Â

1. Encoder（图编码器）

GraphVAE 不再用 MLP，而是 GNN / GCN 编码图结构

$$ q_\phi(z|G) $$

常见实现：

$$ Z = GCN(X, A) $$

然后得到：

$$ \mu, \sigma = f(Z) $$

2. 重参数化

$$ z = \mu + \sigma \cdot \epsilon,\quad \epsilon \sim \mathcal{N}(0, I) $$

3. Decoder（图重构）

GraphVAE的关键变化：不再重建 x，而是重建图结构 A

边概率建模：

$$ p(A_{ij} = 1 | z_i, z_j) $$

常见形式：

$$ \hat{A}_{ij} = \sigma(z_i^T z_j) $$

4. Loss函数（ELBO）

$$ \mathcal{L} = \mathbb{E}_{q(z|G)}[\log p(A|z)]

  • KL(q(z|G)|p(z)) $$

👍优点/创新点

1. 可以生成图结构

2. 首次把 GCN 和 VAE 结合

3. 能学习结构分布

相比普通 GNN:

  • 不只是表示学习，而是分布学习

👎缺点

1. 规模受限

GraphVAE 通常只能处理小图

因为 adjacency matrix 太大（$O(N^2)$）

2. edge independence 假设

通常假设：每条边独立生成

现实中往往是 强依赖性

3. 生成质量有限

相比：

  • diffusion model
  • autoregressive graph model

GraphVAE较弱

💻模型代码 

import torch
import torch.nn as nn
import torch.nn.functional as F


class GraphVAE(nn.Module):
    def __init__(self, in_dim, hidden_dim, z_dim):
        super().__init__()

        # --------------------
        # Encoder (GCN-style)
        # --------------------
        self.fc1 = nn.Linear(in_dim, hidden_dim)
        self.fc_mu = nn.Linear(hidden_dim, z_dim)
        self.fc_logvar = nn.Linear(hidden_dim, z_dim)

    # ------------------------
    # adjacency normalization
    # ------------------------
    def normalize_adj(self, A):
        I = torch.eye(A.size(0), device=A.device)
        A_hat = A + I

        D = torch.sum(A_hat, dim=1)
        D_inv_sqrt = torch.diag(torch.pow(D, -0.5))

        return D_inv_sqrt @ A_hat @ D_inv_sqrt

    # ------------------------
    # encoder forward
    # ------------------------
    def encode(self, X, A_norm):
        h = torch.relu(A_norm @ self.fc1(X))
        mu = self.fc_mu(h)
        logvar = self.fc_logvar(h)
        return mu, logvar

    # ------------------------
    # reparameterization
    # ------------------------
    def reparameterize(self, mu, logvar):
        std = torch.exp(0.5 * logvar)
        eps = torch.randn_like(std)
        return mu + eps * std

    # ------------------------
    # decoder
    # ------------------------
    def decode(self, z):
        return torch.sigmoid(z @ z.t())

    # ------------------------
    # forward
    # ------------------------
    def forward(self, X, A):
        A_norm = self.normalize_adj(A)

        mu, logvar = self.encode(X, A_norm)
        z = self.reparameterize(mu, logvar)

        A_hat = self.decode(z)

        return A_hat, mu, logvar

    # ------------------------
    # loss (ELBO)
    # ------------------------
    def loss(self, A_hat, A, mu, logvar):
        recon_loss = F.binary_cross_entropy(A_hat, A)

        kl_loss = -0.5 * torch.mean(
            1 + logvar - mu.pow(2) - logvar.exp()
        )

        return recon_loss + kl_loss
   
if __name__ == "__main__":
    torch.manual_seed(0)

    # -----------------------
    # 1. toy graph
    # -----------------------
    N = 6        # nodes
    F_in = 3     # feature dim

    # adjacency matrix
    A = torch.randint(0, 2, (N, N)).float()
    A = (A + A.t()) / 2
    A.fill_diagonal_(0)

    # node features
    X = torch.randn(N, F_in)

    # -----------------------
    # 2. model
    # -----------------------
    model = GraphVAE(in_dim=3, hidden_dim=4, z_dim=2)

    optimizer = torch.optim.Adam(model.parameters(), lr=0.01)

    # -----------------------
    # 3. training loop
    # -----------------------
    for epoch in range(100):

        optimizer.zero_grad()

        # forward
        A_hat, mu, logvar = model(X, A)

        # loss
        loss = model.loss(A_hat, A, mu, logvar)

        # backward
        loss.backward()
        optimizer.step()

        if epoch % 10 == 0:
            print(f"Epoch {epoch:03d} | Loss: {loss.item():.4f}")

    # -----------------------
    # 4. inference result
    # -----------------------
    A_hat, _, _ = model(X, A)

    print("\nOriginal A:\n", A)
    print("\nReconstructed A_hat:\n", A_hat)
