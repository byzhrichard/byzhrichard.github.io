参考资料

The perceptron: A probabilistic model for information storage and organization in the brain.

Learning representations by back-propagating errors.

背景

单层感知机本质是线性分类器，无法解决 XOR 等线性不可分问题

把多个==线性变换==与==非线性激活==函数层叠起来，从而形成能够表示复杂非线性映射的模型 -> 多层感知机: Multi-layer perceptron

  • 通过 非线性激活 + 层叠组合，把输入空间映射到更可分的表示空间；
  • 在分类、回归、表征学习中都作为基础模块存在

MLP和FFN在大部分语境下是一个意思

架构(公式)

令第 $l$ 层的权重、偏置为 $W_l, b_l$，激活函数为 $\phi_l(\cdot)$ 记 $h_0=x$，则对 $l=1,2,\dots,L$：

$$ z_l = W_l h_{l-1} + b_l \\ h_l = \phi_l(z_l) $$

$z_l$：第 $l$ 层的 pre-activation（线性输出）

$h_l$：第 $l$ 层的激活输出

1.输出层 - 多分类(softmax)

最后一层输出logits:

$$ logits:z_L\in\mathbb{R}^K $$

经过softmax: (属于第k个类别的概率)

$$ p(y=k\mid x)=\text{softmax}(z_L)_k=\frac{e^{z_{L,k}}}{\sum_{j=1}^{K}e^{z_{L,j}}} $$

各概率之和为1

损失函数: 交叉熵(CE: cross-entropy)

若标签用 one-hot: $Y\in\mathbb{R}^{B\times K}$

$$ \mathcal{L} = -\frac{1}{B}\sum_{i=1}^{B}\sum_{k=1}^{K} Y_{i,k}\log P_{i,k} $$

若标签用整数类别 $y_i\in\{1,\dots,K\}$：

$$ \mathcal{L} = -\frac{1}{B}\sum_{i=1}^{B}\log P_{i,y_i} $$

工程里常用 “Softmax + CE” 一体化实现（更稳定） (pytorch的cross-entropy-loss自带softmax)

2.输出层 - 二分类(sigmoid)

最后一层输出logits:

$$ logits:z_L\in\mathbb{R}^K $$

概率：

$$ p=\sigma(z)=\frac{1}{1+e^{-z}} $$

损失函数：二元交叉熵（BCE）

$$ \mathcal{L}=-\frac{1}{B}\sum_{i=1}^{B}\Big[y_i\log p_i + (1-y_i)\log(1-p_i)\Big] $$

工程里常用 BCE With Logits（logits 直接进损失，更稳定）

3.输出层 - 回归

最后一层输出 $\hat{y}\in\mathbb{R}^{B\times m}$（最后层一般不加激活）

损失函数：均方误差（MSE）

$$ \mathcal{L}=\frac{1}{B}\sum_{i=1}^{B}\|\hat{y}_i-y_i\|_2^2 $$

也常用 MAE / Huber（更抗异常值）

创新点

  1. 用非线性层叠突破线性可分限制 单层只能做超平面划分，MLP 通过非线性激活把决策边界变成复杂曲线/曲面
  2. 强表达能力（通用逼近） 在足够宽的隐藏层与适当激活下，MLP 能逼近任意连续函数（理论层面支撑其通用性）
  3. 成为深度网络的基础积木 CNN 的 $1\times1$ 卷积/全连接头、Transformer 的 FFN，本质上都能视作 MLP 结构在不同数据形态上的扩展

代码实现 - pytorch 

import torch
import torch.nn as nn
import torch.optim as optim


class MLP(nn.Module):
    def __init__(self, input_dim:int, hidden_dims:list, output_dim:int):
        super().__init__()

        layers = []
        prev = input_dim
        for h in hidden_dims:
            layers.append(nn.Linear(prev, h))
            layers.append(nn.ReLU())
            prev = h
        layers.append(nn.Linear(prev, output_dim))

        self.net = nn.Sequential(
            *layers
        )

    def forward(self, x):
        return self.net(x)


def demo_torch():
    # XOR
    X = torch.tensor([[0.,0.], [0.,1.], [1.,0.], [1.,1.]])
    y = torch.tensor([[0.], [1.], [1.], [0.]])  # BCE target

    # 超参数
    lr = 0.05
    epochs = 2000

    model = MLP(input_dim=2, hidden_dims=[8, 8], output_dim=1)
    criterion = nn.BCEWithLogitsLoss() # BCEWithLogitsLoss = Sigmoid + BCE
    optimizer = optim.Adam(model.parameters(), lr=lr)

    for epoch in range(1, epochs+1):
        logits = model(X)
        loss = criterion(logits, y)

        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

        if epoch % 200 == 0 or epoch == 1:
            probs = torch.sigmoid(logits)
            pred = (probs >= 0.5).int().view(-1)
            print(f"epoch={epoch:4d} loss={loss.item():.4f} pred={pred.tolist()}")

    print("Done.")


if __name__ == "__main__":
    demo_torch()

代码实现 - numpy 

import numpy as np

# =========================
# 1) 基础函数：sigmoid 与导数
# =========================
def sigmoid(x):
    return 1.0 / (1.0 + np.exp(-x))


def sigmoid_grad(a):
    # 因为 a = sigmoid(z), 所以 δa/δz = a(1-a)
    return a * (1.0 - a)

# =========================
# 2) 两层 MLP：2 -> 2 -> 1，用于 XOR
# =========================
class MLP_XOR:
    """
    用一个最小的 MLP 来解决 XOR（线性不可分）：

        输入 x(2维)
            ↓ 线性层：Z1 = X @ W1 + b1
            ↓ 非线性：A1 = sigmoid(Z1)
            ↓ 线性层：Z2 = A1 @ W2 + b2
            ↓ 非线性：A2 = sigmoid(Z2)  -> 作为概率输出

    损失函数：BCE（二元交叉熵）
    优化方法：梯度下降（这里用全量/小批都可以，XOR 太小就全量）
    """

    def __init__(self, seed=0):
        rng = np.random.default_rng(seed)
        self.W1 = rng.normal(0, 1.0, size=(2, 2)) * 0.5
        self.b1 = np.zeros((1, 2))
        self.W2 = rng.normal(0, 1.0, size=(2, 1)) * 0.5
        self.b2 = np.zeros((1, 1))

    def forward(self, X):
        """
        输入:
            X: [B,2]，B 是 batch size
        输出:
            A2: [B,1]，表示属于正类(1)的概率
            cache: 保存中间变量，用于反向传播
        """
        Z1 = X @ self.W1 + self.b1
        A1 = sigmoid(Z1)
        Z2 = A1 @ self.W2 + self.b2
        A2 = sigmoid(Z2)  # 概率
        cache = (X, Z1, A1, Z2, A2)
        return A2, cache

    @staticmethod
    def bce_loss(p, y, eps=1e-12):
        """
        二元交叉熵 BCE:
            L = - mean( y*log(p) + (1-y)*log(1-p) )

        p: [B,1] 预测概率
        y: [B,1] 真实标签（0或1）
        """
        p = np.clip(p, eps, 1 - eps)
        return -np.mean(y * np.log(p) + (1 - y) * np.log(1 - p))

    def backward(self, cache, y):
        """
        根据缓存的中间结果，计算各参数的梯度

        记号说明：
            Z1 = XW1 + b1
            A1 = sigmoid(Z1)
            Z2 = A1W2 + b2
            A2 = sigmoid(Z2)

        特别重要的简化（sigmoid + BCE）：
            dL/dZ2 = (A2 - y) / B

        返回：
            grads: dict 包含 dW1, db1, dW2, db2
        """
        X, Z1, A1, Z2, A2 = cache
        B = X.shape[0]

        # δL/δA2 for BCE
        # L = -[y log a + (1-y) log(1-a)]
        dA2 = (A2 - y) / (A2 * (1 - A2) + 1e-12) / B
        dZ2 = dA2 * sigmoid_grad(A2)

        dW2 = A1.T @ dZ2
        db2 = np.sum(dZ2, axis=0, keepdims=True)

        dA1 = dZ2 @ self.W2.T
        dZ1 = dA1 * sigmoid_grad(A1)

        dW1 = X.T @ dZ1
        db1 = np.sum(dZ1, axis=0, keepdims=True)

        return dW1, db1, dW2, db2

    def step(self, grads, lr=0.5):
        """
        用梯度下降更新参数：
            W <- W - lr * dW
            b <- b - lr * db
        """
        dW1, db1, dW2, db2 = grads
        self.W1 -= lr * dW1
        self.b1 -= lr * db1
        self.W2 -= lr * dW2
        self.b2 -= lr * db2

    def predict(self, X):
        """
        返回 0/1 预测类别：
            p>=0.5 -> 1
            p<0.5  -> 0
        """
        p, _ = self.forward(X)
        return (p >= 0.5).astype(int)


def demo_xor():
    # XOR data
    X = np.array([
        [0, 0],
        [0, 1],
        [1, 0],
        [1, 1],
    ], dtype=float)
    y = np.array([
        [0],
        [1],
        [1],
        [0],
    ], dtype=float)

    # 初始化模型
    model = MLP_XOR(seed=0)
    # 训练超参数
    lr = 0.8
    epochs = 5000

    for epoch in range(1, epochs+1):
        # 1) 前向：得到概率
        p, cache = model.forward(X)
        # 2) 计算损失
        loss = model.bce_loss(p, y)
        # 3) 反向：得到梯度
        grads = model.backward(cache, y)
        # 4) 更新参数
        model.step(grads, lr=lr)

        if epoch % 500 == 0 or epoch == 1:
            pred = model.predict(X).reshape(-1)
            print(f"epoch={epoch:4d} loss={loss:.4f} pred={pred}")

    print("Final preds:", model.predict(X).reshape(-1))


if __name__ == "__main__":
    demo_xor()
