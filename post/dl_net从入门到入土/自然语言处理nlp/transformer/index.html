📖参考资料

Attention Is All You Need.

🌱背景

核心思想: 不再依赖 RNN、LSTM、CNN，而是主要依赖 Attention 机制完成序列建模

在 Transformer 出现之前，处理序列数据时常见做法是：

  • RNN
  • LSTM
  • GRU
  • Seq2Seq
  • Seq2Seq + Attention

⚙️架构(公式) 

Source Sentence
      ↓
Embedding + Positional Encoding
      ↓
Encoder Layer × N
      ↓
Memory
      ↓
Decoder Layer × N
      ↓
Linear + Softmax
      ↓
Target Sentence

Encoder 负责理解输入句子

  • Multi-Head Self-Attention
  • Feed Forward Network

Decoder 负责根据 Encoder 的表示一步一步生成目标句子

  • Masked Multi-Head Self-Attention
  • Encoder-Decoder Attention
  • Feed Forward Network

Transformer 里面最核心的部分有：

  • Embedding
  • Positional Encoding
  • Scaled Dot-Product Attention
  • Multi-Head Attention
  • Feed Forward Network
  • Residual Connection
  • Layer Normalization
  • Mask

1. Embedding

输入文本一般不能直接送进神经网络，需要先把 token 转成向量

  • 假设一句话经过分词后为：我 爱 你
  • 然后映射成 id：[3, 4, 5]
  • 再经过 Embedding 层：[3, 4, 5] -> [x1, x2, x3]

其中每个 token 会变成一个 d_model 维向量 $x_i$

如果 batch size 为 B，序列长度为 T，词向量维度为 d_model，那么输入形状为：(B, T)

经过 Embedding 后变成：(B, T, d_model)
