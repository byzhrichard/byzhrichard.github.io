📖参考资料

Neural Machine Translation by Jointly Learning to Align and Translate.

🌱背景

Temporal Attention: 时间注意力机制 (temporal: 时间步, 序列位置, token位置)

例如一句话：I love deep learning, 可以看成一个具有四个时间步的序列：

  • x1 = I
  • x2 = love
  • x3 = deep
  • x4 = learning

因为不是每个词都一样重要, Temporal Attention就来解决这个问题

Temporal Attention 的核心思想: 能够自动判断哪些时间步更重要

RNN/LSTM 负责提取每个时间步的隐藏状态 Temporal Attention 负责给不同时间步分配不同权重

h1, h2, h3, h4
 ↓   ↓   ↓   ↓
α1, α2, α3, α4

最后对所有隐藏状态加权求和：$context = α_1h_1 + α_2h_2 + α_3h_3 + α_4h_4$

⚙️架构(公式)

假设输入序列为：

$$ X = (x_1, x_2, ..., x_T) $$

经过 RNN/LSTM/GRU 后，可以得到每个时间步的隐藏状态：

$$ H = (h_1, h_2, ..., h_T) $$
  • $h_t \in R^d$: 表示第 $t$ 个时间步的隐藏状态

Temporal Attention 的目标是为每个 $h_t$ 计算一个权重 $\alpha_t$ -> 权重越大，说明这个时间步越重要

1. 计算注意力分数

首先，对每个隐藏状态 $h_t$ 计算一个分数：

$$ e_t = score(h_t) $$

常见写法是：

$$ e_t = v^T tanh(W h_t + b) $$
  • $W$ 是可学习参数
  • $b$ 是偏置
  • $v$ 是可学习向量
  • $e_t$ 表示第 $t$ 个时间步的重要性分数

2. softmax 归一化

为了让所有时间步的权重加起来等于 1，需要使用 softmax：

$$ \alpha_t = \frac{exp(e_t)}{\sum_{i=1}^{T} exp(e_i)} $$

这样就得到了每个时间步的注意力权重

3. 加权求和得到上下文向量

得到注意力权重后，对所有隐藏状态加权求和：

$$ c = \sum_{t=1}^{T} \alpha_t h_t $$
  • $c$ 是 context vector，也就是上下文向量
  • $\alpha_t$ 是第 $t$ 个时间步的注意力权重
  • $h_t$ 是第 $t$ 个时间步的隐藏状态

这个 $c$ 就是模型根据注意力机制得到的序列表示

👍优点/创新点

1. 不再只依赖最后一个隐藏状态

传统 RNN/LSTM 常常使用最后一个隐藏状态表示整个序列 但是最后一个隐藏状态可能无法完整保留前面所有信息

Temporal Attention 会综合所有时间步的信息：然后自动选择重要位置，这比只看 $h_T$ 更合理

2. 一定程度上缓解长序列信息丢失

即使某个关键信息在较早位置出现，模型也可以通过注意力权重重新关注它

3. 可解释性

Temporal Attention 的权重可以可视化

例如： I 0.03 really 0.10 love 0.35 this 0.08 movie 0.44

这可以帮助我们观察模型更关注哪些位置

👎缺点

1. 仍然依赖 RNN/LSTM

只是增强了 RNN/LSTM 的表示能力，但没有完全解决 RNN/LSTM 的串行计算问题

2. 注意力计算基于隐藏状态质量

Attention 不是凭空创造信息, 而是从已有隐藏状态中重新分配重要性

Temporal Attention 的输入是 RNN/LSTM 的隐藏状态 如果隐藏状态本身提取不好，注意力机制也很难发挥作用

3. 对长序列仍有压力

长序列建模问题只是被缓解，并没有被彻底解决

4. 表达能力不如后来的 Self-Attention 灵活

Temporal Attention 通常是：

一个 query 对所有时间步做加权

而 Self-Attention 是：

每个 token 都可以和其他 token 互相计算注意力

因此 Self-Attention 可以建模更复杂的 token-token 关系

💻模型代码 

import torch
import torch.nn as nn
import torch.nn.functional as F

from byzh.ai.Butils import b_get_params


# T: 序列长度
# H: hidden_size
class B_TemporalAttention(nn.Module):
    """
    Temporal Attention:
        对每一个时间步的 hidden state 计算注意力权重
        然后加权求和得到 context vector

    输入:
        hidden_states: (B, T, H)
        mask: (B, T)

    输出:
        context: (B, H)
        attn_weights: (B, T)
    """

    def __init__(
        self,
        hidden_size: int,
        attn_size: int = None
    ):
        super().__init__()

        if attn_size is None:
            attn_size = hidden_size

        # 对应公式: tanh(W h_t + b)
        self.W = nn.Linear(hidden_size, attn_size)

        # 对应公式: v^T tanh(W h_t + b)
        # 输出一个分数 e_t
        self.v = nn.Linear(attn_size, 1, bias=False)

    def forward(self, hidden_states, mask=None):
        """
        输入:
            hidden_states: (B, T, H)  RNN/GRU/LSTM 每个时间步的隐藏状态
            mask: (B, T)              非 pad 的位置为 True，pad 的位置为 False

        输出:
            context: (B, H)           加权求和后的上下文向量
            attn_weights: (B, T)      每个时间步的注意力权重
        """

        # 1) 计算注意力分数
        # hidden_states: (B, T, H)
        score = torch.tanh(self.W(hidden_states))   # (B, T, A)
        score = self.v(score).squeeze(-1)           # (B, T)

        # 2) 如果有 padding，需要把 pad 位置的分数变成极小值
        # 这样 softmax 后，pad 位置的权重就接近 0
        if mask is not None:
            score = score.masked_fill(mask == 0, -1e9)

        # 3) softmax 得到注意力权重
        attn_weights = F.softmax(score, dim=1)      # (B, T)

        # 4) 加权求和得到 context vector
        # attn_weights.unsqueeze(1): (B, 1, T)
        # hidden_states: (B, T, H)
        context = torch.bmm(
            attn_weights.unsqueeze(1),
            hidden_states
        ).squeeze(1)                                # (B, H)

        return context, attn_weights


class B_TemporalAttention_Model(nn.Module):
    """
    GRU + Temporal Attention:
        token 序列 -> embedding -> GRU -> Temporal Attention -> 分类结果

    适合任务:
        文本分类
        情感分类
        序列分类
        时间序列分类

    输入:
        x: (B, T)

    输出:
        logits: (B, num_classes)
        attn_weights: (B, T)
    """

    def __init__(
        self,
        vocab_size: int,
        emb_dim: int,
        hidden_size: int,
        num_classes: int,
        num_layers: int = 1,
        pad_idx: int = 0,        # 规定词表里 index=0 代表 <pad>
        dropout: float = 0.1,
        bidirectional: bool = False
    ):
        super().__init__()

        self.pad_idx = pad_idx
        self.bidirectional = bidirectional

        # 把 “离散的 token id” 映射成 “连续的向量表示”
        self.embedding = nn.Embedding(
            vocab_size,
            emb_dim,
            padding_idx=pad_idx
        )

        # GRU 负责提取每个时间步的隐藏状态
        self.gru = nn.GRU(
            input_size=emb_dim,
            hidden_size=hidden_size,
            num_layers=num_layers,
            batch_first=True,
            dropout=dropout if num_layers > 1 else 0.0,
            bidirectional=bidirectional
        )

        # 如果是双向 GRU，那么每个时间步的 hidden_size 会变成 hidden_size * 2
        rnn_out_size = hidden_size * 2 if bidirectional else hidden_size

        # Temporal Attention 负责对所有时间步分配权重
        self.attention = B_TemporalAttention(
            hidden_size=rnn_out_size,
            attn_size=rnn_out_size
        )

        self.dropout = nn.Dropout(dropout)

        # 最后用 context vector 做分类
        self.fc = nn.Linear(rnn_out_size, num_classes)

    def forward(self, x):
        """
        输入:
            x: (B, T)

        输出:
            logits: (B, num_classes)       每个类别的预测分数
            attn_weights: (B, T)           每个时间步的注意力权重
        """

        # mask 用来标记哪些位置不是 <pad>
        # 非 pad 位置: True
        # pad 位置: False
        mask = x != self.pad_idx                  # (B, T)

        emb = self.embedding(x)                   # (B, T, E)

        # outputs 是每一个时间步的隐藏状态
        # hT 是最后一个时间步的隐藏状态
        outputs, hT = self.gru(emb)               # outputs: (B, T, H)

        # Temporal Attention:
        # 从所有时间步的 hidden state 中，自动挑重点
        context, attn_weights = self.attention(
            outputs,
            mask=mask
        )                                         # context: (B, H), attn_weights: (B, T)

        context = self.dropout(context)

        logits = self.fc(context)                 # (B, num_classes)

        return logits, attn_weights


# vocab = {
#     "<pad>": 0,
#     "<bos>": 1,
#     "<eos>": 2,
#     "I": 3,
#     "really": 4,
#     "love": 5,
#     "this": 6,
#     "movie": 7,
#     "hate": 8,
#     "bad": 9,
# }
#
# x = torch.tensor([
#     [3, 4, 5, 6, 7, 2],    # I really love this movie <eos>
#     [3, 8, 6, 9, 2, 0],    # I hate this bad <eos> <pad>
# ])
#
# y = torch.tensor([
#     1,    # positive
#     0,    # negative
# ])

if __name__ == "__main__":
    # ===== 超参数 =====
    B = 4
    T = 7                     # 输入序列长度
    vocab_size = 100          # 词表大小（包含特殊 token，如 <pad>, <bos>, <eos>）
    emb_dim = 32
    hidden_size = 64
    num_classes = 3           # 分类类别数

    model = B_TemporalAttention_Model(
        vocab_size=vocab_size,
        emb_dim=emb_dim,
        hidden_size=hidden_size,
        num_classes=num_classes,
        num_layers=1,
        pad_idx=0,
        dropout=0.1,
        bidirectional=False
    )

    # 随机生成输入序列
    x = torch.randint(3, vocab_size, (B, T))

    # 手动造几个 pad，模拟真实 batch 中的变长序列
    x[0, -2:] = 0
    x[1, -1:] = 0

    logits, attn_weights = model(x)

    print(logits.shape)        # (B, num_classes)
    print(attn_weights.shape)  # (B, T)

    # 每一行的注意力权重加起来应该约等于 1
    print(attn_weights.sum(dim=1))  # (B,)

    print(f"参数量: {b_get_params(model)}")
