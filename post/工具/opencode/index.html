规则

1. 项目级规则

在项目根目录放置一个 AGENTS.md 文件

3. 全局规则

在 ~/.config/opencode/AGENTS.md 文件中设置全局规则

你始终用中文回复。你要开始进行操作时，需要先询问我是否开始。

常用命令

读取项目

/init

压缩上下文

/compact

选择model

/models

添加model

/connect

添加的model供应商, 在%USERPROFILE%\.local\share\opencode

1. 如果不小心退出了

直接恢复上一次会话:

opencode -c

查看所有历史会话:

opencode session list

恢复置指定的会话:

opencode -s <sessionID>

需要完整的sessionID

多对话框

创建新的section

/new

查看section

/section

分享

创建链接(公网)

/share

取消链接(公网)

/unshare

导出为文件

/export

skills

1. 项目级

根目录/.opencode/skills

2. 全局

%USERPROFILE%/.config/opencode/skills

https://github.com/anthropics/skills

Skill 功能
algorithmic-art 用 p5.js 创建算法艺术（种子随机、交互参数）
brand-guidelines 应用官方品牌颜色和排版到工件
canvas-design 设计海报/艺术品等静态视觉（PNG/PDF）
claude-api 构建/调试/优化 Claude API / Anthropic SDK 应用
doc-coauthoring 结构化协作撰写文档/提案/技术规格
docx 创建/读取/编辑 Word 文档（.docx）
frontend-design 创建高质量生产级前端界面
internal-comms 编写内部通讯（状态报告、简报、FAQ 等）
mcp-builder 创建 MCP 服务器，支持 Python FastMCP 和 Node/TypeScript
pdf 处理 PDF（读取、合并、拆分、OCR、加密等）
pptx 创建/编辑/操作 PowerPoint 演示文稿
skill-creator 创建/修改/评估新的 skill
slack-gif-creator 创建 Slack 优化的动画 GIF
theme-factory 为幻灯片/文档/HTML 等应用主题样式（10 套预设）
web-artifacts-builder 用 React+Tailwind+shadcn/ui 构建复杂 HTML 工件
webapp-testing 用 Playwright 测试本地 Web 应用
xlsx 处理电子表格（创建、编辑、图表、格式转换等）
