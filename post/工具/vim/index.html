安装vim

https://github.com/vim/vim-win32-installer/releases

在vim中输入:echo $MYVIMRC可以找到文件位置 打开, 在最下面添加一行set number就可以永久显示行号 打开, 在最下面添加一行set relativenumber就可以永久显示相对行号

模式

正常模式 编辑模式 命令行模式
Normal Mode Insert Mode Command-line mode
浏览 修改 保存,退出,搜索…

正常模式->编辑模式: i 编辑模式->正常模式: esc

:q -> 退出 :q! -> 不管保存与否, 都要退出 :wq -> 保存并退出

↑: k ↓: j ←: h →: l

快捷

o: 在下面插入新的一行 O: 在上面插入新的一行

i: 在光标当前位置输入 a: 在光标后一个字符位置输入 I: 在行首输入 A: 在行末输入

G: 光标移动到最下面 gg: 光标移动到最上面

数字k: 移动到上面的相对数字行 数字j: 移动到下面的相对数字行

yw: 复制一个词 yy: 复制行 p: 粘贴行

dd: 删除行

.: 重复前次操作 u: 撤销前次操作 ctrl+r: 撤销前次撤销

修改字符

dw: 删除一个单词 cw: 修改一个单词

w: 移动到下一个单词的开头 e: 移动到下一个单词的最后

b: 移动到上一个单词的开头

功能

/awa: 找到awa

:%s/awa/qwq/g: 把awa全局替换为qwq

在{}内输入ci{可以直接删除里面的内容

shift+v: 可视化行 shift+q: 可视化块
